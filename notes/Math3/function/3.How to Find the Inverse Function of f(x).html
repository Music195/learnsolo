How to Find the Inverse Function of $f(x)$
1.  Solve $y = f(x)$ for $x$, and rewrite it in the form $x = g(y)$.
2.  Swap $x$ and $y$, and write $y = g(x)$.
The domain of the inverse function $g(x)$ is the same as the range of the original function $f(x)$.
For a function and its inverse, the domain and range are swapped.
