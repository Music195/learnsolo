Magnitude of Vector

Mathematical Foundation

The magnitude (or length) of a vector represents its size or distance from the origin in vector space. For a vector \(\vec{v} = (v_1, v_2)\), the magnitude is calculated using the Euclidean distance formula.

Vector Magnitude Formula
For a vector \(\vec{v} = (v_1, v_2)\):
\(|\vec{v}| = \sqrt{v_1^2 + v_2^2}\)
For a vector \(\vec{v} = (v_1, v_2, v_3)\) in 3D:
\(|\vec{v}| = \sqrt{v_1^2 + v_2^2 + v_3^2}\)
Key Properties of Vector Magnitude
\(|\vec{v}| \geq 0\) (non-negative)
\(|\vec{v}| = 0\) if and only if \(\vec{v} = \vec{0}\)
\(|\vec{v}| = |- \vec{v}|\) (magnitude of negative vector)

Position Vectors and Distance

The position vector of a point P(a,b) is the vector from the origin to that point: \(\overrightarrow{OP} = (a,b)\). The magnitude of this position vector gives the distance from the origin to the point.

Distance from Origin: \(|\overrightarrow{OP}| = \sqrt{a^2 + b^2}\)
Distance Between Two Points
\(|\overrightarrow{AB}| = \sqrt{(b_1 - a_1)^2 + (b_2 - a_2)^2}\)
Where A(a₁,a₂) and B(b₁,b₂) are points in the plane

Worked Examples

1
Find the magnitude of vector \(\vec{v} = (3, 4)\)
\(|\vec{v}| = \sqrt{3^2 + 4^2} = \sqrt{9 + 16} = \sqrt{25} = 5\)
The magnitude is 5 units.
2
Find the distance between points A(1,2) and B(4,6)
\(|\overrightarrow{AB}| = \sqrt{(4-1)^2 + (6-2)^2} = \sqrt{3^2 + 4^2} = \sqrt{9 + 16} = \sqrt{25} = 5\)
The distance between A and B is 5 units.
3
Find the magnitude of vector \(\vec{u} = (-2, 3)\)
\(|\vec{u}| = \sqrt{(-2)^2 + 3^2} = \sqrt{4 + 9} = \sqrt{13} \approx 3.606\)
The magnitude is \(\sqrt{13}\) units.

Applications

Vector magnitude has numerous applications in physics, engineering, and computer graphics:

  • Physics: Speed, force magnitude, displacement distance
  • Engineering: Structural analysis, stress calculations
  • Computer Graphics: Distance calculations, collision detection
  • Navigation: GPS distance calculations, path finding
  • Statistics: Euclidean distance in clustering algorithms

    • Practice Problems

    1
    Calculate \(|\vec{v}|\) for \(\vec{v} = (6, 8)\)
    \(|\vec{v}| = \sqrt{6^2 + 8^2} = \sqrt{36 + 64} = \sqrt{100} = 10\)
    2
    Find the distance between P(-1,3) and Q(2,7)
    \(|\overrightarrow{PQ}| = \sqrt{(2-(-1))^2 + (7-3)^2} = \sqrt{3^2 + 4^2} = \sqrt{9 + 16} = \sqrt{25} = 5\)
    3
    Which vector has greater magnitude: \(\vec{a} = (5, 12)\) or \(\vec{b} = (8, 15)\)?
    \(|\vec{a}| = \sqrt{5^2 + 12^2} = \sqrt{25 + 144} = \sqrt{169} = 13\)
    \(|\vec{b}| = \sqrt{8^2 + 15^2} = \sqrt{64 + 225} = \sqrt{289} = 17\)
    Vector \(\vec{b}\) has greater magnitude (17 > 13).

    Key Takeaways

    • The magnitude of a vector is always non-negative and represents its length
    • Use the formula \(|\vec{v}| = \sqrt{v_1^2 + v_2^2}\) for 2D vectors
    • The distance between two points is the magnitude of the vector connecting them
    • Vector magnitude is essential for measuring distances in coordinate geometry
    • The magnitude of a zero vector is zero, and zero magnitude implies a zero vector
