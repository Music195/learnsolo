Angle Between Two Vectors

Mathematical Foundation

The angle between two vectors is defined using the dot product. This angle represents the smallest angle between the directions of the two vectors, measured from 0° to 180°.

Angle Between Vectors Formula
For non-zero vectors \(\vec{a}\) and \(\vec{b}\) with angle \(\theta\):
\(\cos\theta = \frac{\vec{a} \cdot \vec{b}}{|\vec{a}| |\vec{b}|}\)
In component form: \(\cos\theta = \frac{a_1b_1 + a_2b_2}{\sqrt{a_1^2 + a_2^2} \sqrt{b_1^2 + b_2^2}}\)
Important Notes
The angle θ satisfies: \(0^\circ \leq \theta \leq 180^\circ\)
The formula gives the acute or obtuse angle between the vectors
For zero vectors, the angle is undefined

Angle Interpretation

Acute Angle (0° < θ < 90°)
cosθ > 0
Vectors point in similar directions
Right Angle (θ = 90°)
cosθ = 0
Vectors are perpendicular (orthogonal)
Obtuse Angle (90° < θ < 180°)
cosθ < 0
Vectors point in opposite directions
Parallel Vectors (θ = 0° or 180°)
cosθ = ±1
Vectors are in the same or opposite direction

Derivation of the Formula

The angle formula is derived from the geometric definition of the dot product:

Starting with: \(\vec{a} \cdot \vec{b} = |\vec{a}| |\vec{b}| \cos\theta\)
  • Step 1: Solve for cosθ: \(\cos\theta = \frac{\vec{a} \cdot \vec{b}}{|\vec{a}| |\vec{b}|}\)
  • Step 2: Substitute component form: \(\cos\theta = \frac{a_1b_1 + a_2b_2}{\sqrt{a_1^2 + a_2^2} \sqrt{b_1^2 + b_2^2}}\)
  • Step 3: Take inverse cosine to find θ: \(\theta = \cos^{-1}\left(\frac{\vec{a} \cdot \vec{b}}{|\vec{a}| |\vec{b}|}\right)\)

    • Worked Examples

    1
    Find the angle between \(\vec{a} = (2, 3)\) and \(\vec{b} = (4, 1)\)
    Dot product: \(\vec{a} \cdot \vec{b} = 2 \times 4 + 3 \times 1 = 8 + 3 = 11\)
    Magnitudes: \(|\vec{a}| = \sqrt{2^2 + 3^2} = \sqrt{13}\), \(|\vec{b}| = \sqrt{4^2 + 1^2} = \sqrt{17}\)
    \(\cos\theta = \frac{11}{\sqrt{13} \times \sqrt{17}} = \frac{11}{\sqrt{221}}\)
    \(\theta = \cos^{-1}\left(\frac{11}{\sqrt{221}}\right) \approx 25.5^\circ\)
    2
    Determine if vectors \(\vec{p} = (1, 1)\) and \(\vec{q} = (-1, 1)\) are perpendicular
    Dot product: \(\vec{p} \cdot \vec{q} = 1 \times (-1) + 1 \times 1 = -1 + 1 = 0\)
    \(\cos\theta = \frac{0}{|\vec{p}| |\vec{q}|} = 0\), so \(\theta = 90^\circ\)
    The vectors are perpendicular.
    3
    Find the angle between \(\vec{u} = (3, 0)\) and \(\vec{v} = (0, 4)\)
    Dot product: \(\vec{u} \cdot \vec{v} = 3 \times 0 + 0 \times 4 = 0\)
    Magnitudes: \(|\vec{u}| = 3\), \(|\vec{v}| = 4\)
    \(\cos\theta = \frac{0}{3 \times 4} = 0\), so \(\theta = 90^\circ\)
    The vectors are perpendicular (along x and y axes).

    Applications

    Vector angles are crucial in many mathematical and practical applications:

  • Physics: Force analysis, projectile motion, work calculations
  • Computer Graphics: Lighting angles, surface normals, collision detection
  • Navigation: Direction finding, GPS calculations
  • Engineering: Structural analysis, stress calculations
  • Machine Learning: Cosine similarity, document comparison

    • Practice Problems

    1
    Find the angle between \(\vec{a} = (1, 2)\) and \(\vec{b} = (3, 4)\)
    \(\vec{a} \cdot \vec{b} = 1 \times 3 + 2 \times 4 = 3 + 8 = 11\)
    \(|\vec{a}| = \sqrt{1 + 4} = \sqrt{5}\), \(|\vec{b}| = \sqrt{9 + 16} = 5\)
    \(\cos\theta = \frac{11}{5\sqrt{5}}\), \(\theta = \cos^{-1}\left(\frac{11}{5\sqrt{5}}\right) \approx 12.1^\circ\)
    2
    Are vectors \(\vec{m} = (2, 1)\) and \(\vec{n} = (1, -2)\) perpendicular?
    \(\vec{m} \cdot \vec{n} = 2 \times 1 + 1 \times (-2) = 2 - 2 = 0\)
    Dot product is zero, so vectors are perpendicular.
    3
    Find the angle between \(\vec{p} = (4, 3)\) and \(\vec{q} = (-4, 3)\)
    \(\vec{p} \cdot \vec{q} = 4 \times (-4) + 3 \times 3 = -16 + 9 = -7\)
    \(|\vec{p}| = \sqrt{16 + 9} = 5\), \(|\vec{q}| = \sqrt{16 + 9} = 5\)
    \(\cos\theta = \frac{-7}{25} = -0.28\), \(\theta = \cos^{-1}(-0.28) \approx 107^\circ\)

    Key Takeaways

    • The angle between vectors is found using: \(\cos\theta = \frac{\vec{a} \cdot \vec{b}}{|\vec{a}| |\vec{b}|}\)
    • The angle ranges from 0° to 180°
    • Perpendicular vectors have a 90° angle (dot product = 0)
    • Parallel vectors have 0° or 180° angles (cosθ = ±1)
    • The formula works for both 2D and 3D vectors
    • Cosθ > 0 for acute angles, cosθ < 0 for obtuse angles
