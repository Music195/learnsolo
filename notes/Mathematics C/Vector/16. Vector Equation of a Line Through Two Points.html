Vector Equation of a Line Through Two Points
The vector equation of a line passing through two distinct points A and B can be expressed in two equivalent forms, both using position vectors and parameters.
Form 1: $$\vec{r} = (1-t)\vec{a} + t\vec{b}$$
Form 2: $$\vec{r} = s\vec{a} + t\vec{b} \quad (s+t=1)$$
Parameter Interpretation
Form 1: t parameter
t ∈ ℝ (all real numbers)
• t = 0: Point A
• t = 1: Point B
• 0 < t < 1: Between A and B
Form 2: s,t parameters
s + t = 1
• s ≥ 0, t ≥ 0: Line segment AB
• s > 1, t < 0: Extended beyond A
• s < 0, t > 1: Extended beyond B

Mathematical Foundation

For two distinct points A(\(\vec{a}\)) and B(\(\vec{b}\)), the line can be expressed in two equivalent parametric forms:

$$\vec{r} = (1-t)\vec{a} + t\vec{b} = \vec{a} + t(\vec{b} - \vec{a})$$
$$\vec{r} = s\vec{a} + t\vec{b} \quad \text{where } s + t = 1 \quad (t = 1-s)$$

The first form uses a single parameter t, while the second form uses two parameters s and t that sum to 1. Both forms generate the same line as the parameters vary.

Practice Problems

Problem 1

Find the vector equation of the line passing through points A(2,1) and B(4,3).

Solution:

Using the formula: \(\vec{r} = (1-t)\vec{a} + t\vec{b}\)

\(\vec{r} = (1-t)(2,1) + t(4,3) = (2-2t + 4t, 1-t + 3t) = (2+2t, 1+2t)\)

Vector equation: \((x,y) = (2+2t, 1+2t)\)

Problem 2

Find the coordinates of the point dividing the line segment joining A(1,2) and B(5,4) in the ratio 1:2.

Solution:

Section formula: Point dividing in ratio m:n is \(\frac{n\vec{a} + m\vec{b}}{m+n}\)

Here m=1, n=2: \(\frac{2(1,2) + 1(5,4)}{1+2} = \frac{(2,4) + (5,4)}{3} = \frac{(7,8)}{3} = (\frac{7}{3}, \frac{8}{3})\)

Using parametric form: t = m/(m+n) = 1/3

\(\vec{r} = (1-\frac{1}{3})(1,2) + \frac{1}{3}(5,4) = \frac{2}{3}(1,2) + \frac{1}{3}(5,4) = (\frac{7}{3}, \frac{8}{3})\)

Problem 3

Key Takeaways
  • Two equivalent forms: \(\vec{r} = (1-t)\vec{a} + t\vec{b}\) and \(\vec{r} = s\vec{a} + t\vec{b}\) with s+t=1
  • t=0 gives point A, t=1 gives point B, 0
  • s≥0, t≥0 in second form gives line segment AB (convex combination)
  • Direction vector is \(\vec{b} - \vec{a}\)
  • Section formula: Point dividing in ratio m:n is \(\frac{n\vec{a} + m\vec{b}}{m+n}\)
  • Used for parametric representation of line segments
