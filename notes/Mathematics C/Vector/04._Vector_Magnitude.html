Vector Components and Magnitude

A vector can be represented by its components in a coordinate system. The magnitude (length) of a vector is the distance from its initial point to its terminal point. This fundamental concept connects geometry with algebra and has applications throughout mathematics and physics.

Vector Magnitude Formula
2D Vector Magnitude
\(|\vec{a}| = \sqrt{a_1^2 + a_2^2}\)
For vector \(\vec{a} = (a_1, a_2)\), magnitude is the square root of the sum of squared components
3D Vector Magnitude
\(|\vec{a}| = \sqrt{a_1^2 + a_2^2 + a_3^2}\)
For vector \(\vec{a} = (a_1, a_2, a_3)\), includes the z-component
General nD Magnitude
\(|\vec{a}| = \sqrt{\sum_{i=1}^{n} a_i^2}\)
In n-dimensional space, magnitude is the square root of the sum of all squared components
Distance Formula Connection

Vector magnitude is the distance between two points:

  • Distance: \(d = \sqrt{(x_2 - x_1)^2 + (y_2 - y_1)^2}\) = |\(\vec{b} - \vec{a}\)|
  • Position Vector: If \(\vec{a}\) starts at origin, |\(\vec{a}\)| is distance from origin
  • Pythagorean Theorem: Magnitude formula is 2D Pythagorean theorem
  • Unit Vector: \(\hat{a} = \dfrac{\vec{a}}{|\vec{a}|}\) has magnitude 1

Direction and Unit Vectors

Direction Angle: \(\theta = \tan^{-1}\left(\dfrac{y}{x}\right)\)
Direction and Unit Vector Formulas
Direction Angle
\(\theta = \tan^{-1}\left(\dfrac{a_2}{a_1}\right)\)
Angle that the vector makes with the positive x-axis
Unit Vector
\(\hat{a} = \dfrac{\vec{a}}{|\vec{a}|} = \left(\dfrac{a_1}{|\vec{a}|}, \dfrac{a_2}{|\vec{a}|}\right)\)
Vector with same direction as \(\vec{a}\) but magnitude 1
Component Form
\(\vec{a} = |\vec{a}|(\cos\theta, \sin\theta)\)
Vector expressed in terms of magnitude and direction angle

Worked Examples

1
Problem: Find the magnitude of \(\vec{a} = (3, 4)\).

Step 1: Apply magnitude formula
|\(\vec{a}\)| = √(3² + 4²) = √(9 + 16) = √25
Step 2: Calculate square root
√25 = 5
Step 3: Verify with Pythagorean theorem
3-4-5 right triangle, hypotenuse = 5
Answer: |\(\vec{a}\)| = 5
2
Problem: Find the unit vector in the direction of \(\vec{v} = (5, 12)\).

Step 1: Find magnitude
|\(\vec{v}\)| = √(5² + 12²) = √(25 + 144) = √169 = 13
Step 2: Divide by magnitude
\(\hat{v} = \dfrac{\vec{v}}{|\vec{v}|} = \dfrac{(5, 12)}{13} = \left(\dfrac{5}{13}, \dfrac{12}{13}\right)\)
Step 3: Verify unit vector
|\(\hat{v}\)| = √((5/13)² + (12/13)²) = √((25 + 144)/169) = √(169/169) = 1
Answer: Unit vector \(\hat{v} = \left(\dfrac{5}{13}, \dfrac{12}{13}\right)\)
3
Problem: Find the direction angle of \(\vec{a} = (-3, 4)\).

Step 1: Use arctangent formula
\(\theta = \tan^{-1}\left(\dfrac{4}{-3}\right) = \tan^{-1}\left(-\dfrac{4}{3}\right)\)
Step 2: Calculate angle
\(\tan^{-1}(-1.333) ≈ -53.13^\circ\)
Step 3: Adjust for quadrant (second quadrant)
Since x < 0 and y > 0, θ = 180° - 53.13° = 126.87°
Answer: Direction angle θ ≈ 126.87°

Applications

Physics and Engineering

  • Displacement: Distance traveled in a straight line
  • Velocity: Speed is the magnitude of velocity vector
  • Force: Magnitude determines force strength
  • Work: W = F × d × cosθ (force magnitude times distance)

    • Computer Graphics and Games

  • Object Positioning: Distance calculations for collision detection
  • Lighting: Distance attenuation for realistic lighting
  • AI Pathfinding: Distance heuristics for navigation
  • Animation: Vector interpolation and scaling
    • Real-World Connection:

    GPS systems use vector magnitude to calculate distances between coordinates. The distance formula is essential for navigation apps to determine travel distances and estimated arrival times.

    θ = tan⁻¹(-12/5) = tan⁻¹(-2.4) ≈ -67.38°

    Key Takeaways

    • Magnitude Formula: \(|\vec{a}| = \sqrt{a_1^2 + a_2^2}\) for 2D vectors
    • Distance Connection: Vector magnitude equals distance between points
    • Pythagorean Theorem: Magnitude formula is generalized Pythagorean theorem
    • Unit Vectors: \(\hat{a} = \vec{a}/|\vec{a}|\) has magnitude 1, same direction
    • Direction Angle: \(\theta = \tan^{-1}(a_2/a_1)\) from positive x-axis
    • Component Form: \(\vec{a} = |\vec{a}|(\cos\theta, \sin\theta)\)
    • Zero Vector: |\(\vec{0}\)| = 0
    • Applications in physics, navigation, graphics, and engineering
