Vector Equation of a Straight Line
The vector equation of a line passing through a point A with position vector \(\vec{a}\) and parallel to a direction vector \(\vec{d}\) is given by \(\vec{r} = \vec{a} + t\vec{d}\), where t is a real parameter.
$$ \vec{r} = \vec{a} + t\vec{d} $$
where t ∈ ℝ (real numbers)

Mathematical Foundation

A straight line in vector form is defined by:

  • Point on the line: A position vector \(\vec{a}\) of a known point A on the line
  • Direction vector: A non-zero vector \(\vec{d}\) parallel to the line
  • Parameter: A real number t that varies to generate all points on the line
$$ \vec{r} = \vec{a} + t\vec{d}, \quad t \in \mathbb{R} $$

As t varies over all real numbers, \(\vec{r}\) traces out the entire line. The direction vector \(\vec{d}\) determines the slope and direction of the line.

Practice Problems

Problem 1

Find the vector equation of the line passing through the point A(2,3) and parallel to the vector \(\vec{d} = (4,5)\).

Solution:

The vector equation is: \(\vec{r} = \vec{a} + t\vec{d}\)

\(\vec{r} = (2,3) + t(4,5)\)

Position vector form: \((x,y) = (2+4t, 3+5t)\)

Problem 2

Find the coordinates of the point on the line \(\vec{r} = (1,2) + t(3,4)\) when t = -1.

Solution:

Substitute t = -1: \(\vec{r} = (1,2) + (-1)(3,4) = (1,2) + (-3,-4) = (-2,-2)\)

The point is (-2, -2).

Problem 3

Find the direction vector of the line passing through points A(1,2) and B(4,6).

Solution:

Direction vector \(\vec{d} = \overrightarrow{AB} = (4-1, 6-2) = (3,4)\)

Vector equation: \(\vec{r} = (1,2) + t(3,4)\)

Key Takeaways
  • Vector equation: \(\vec{r} = \vec{a} + t\vec{d}\) where \(\vec{a}\) is a point on the line and \(\vec{d}\) is direction vector
  • Parameter t varies over all real numbers to generate all points on the line
  • Direction vector \(\vec{d}\) is parallel to the line and non-zero
  • For two points A and B, direction vector is \(\overrightarrow{AB}\)
  • Cartesian form: x = a₁ + t d₁, y = a₂ + t d₂
  • Used for parametric representation of lines in any dimension
