Properties of Scalar Multiplication of Vectors

Mathematical Foundation

Scalar Multiplication Basics

Scalar multiplication combines a real number (scalar) with a vector to produce a new vector. This operation scales the vector's magnitude and can change its direction. The properties of scalar multiplication work together with vector addition to create a complete algebraic system.

Fundamental Properties of Scalar Multiplication
1. Associativity with Scalars
\(k(l\vec{a}) = (kl)\vec{a}\)
Multiplying a vector by a product of scalars is the same as multiplying by each scalar sequentially
2. Distributivity over Scalar Addition
\((k + l)\vec{a} = k\vec{a} + l\vec{a}\)
A vector multiplied by a sum of scalars equals the sum of the vector multiplied by each scalar
3. Distributivity over Vector Addition
\(k(\vec{a} + \vec{b}) = k\vec{a} + k\vec{b}\)
A scalar multiplied by a sum of vectors equals the sum of the scalar multiplied by each vector
4. Scalar Identity
\(1 \cdot \vec{a} = \vec{a}\)
Multiplying by 1 leaves the vector unchanged
5. Scalar Zero
\(0 \cdot \vec{a} = \vec{0}\)
Multiplying by 0 gives the zero vector
Geometric Interpretation

When you multiply a vector by a scalar k:

  • Magnitude: |k\vec{a}| = |k| × |\vec{a}| (scaled by absolute value of k)
  • Direction: Same as \vec{a} if k > 0, opposite if k < 0
  • Special cases: k = 0 gives zero vector, k = -1 gives additive inverse
  • Unit vector: \vec{a}/|\vec{a}| has magnitude 1, same direction as \vec{a}

Worked Examples

1
Problem: Given \(\vec{a} = (2, 3)\), find \(3\vec{a}\) and \(-2\vec{a}\).

Step 1: Multiply each component by the scalar
\(3\vec{a} = 3(2, 3) = (6, 9)\)
Step 2: For negative scalar
\(-2\vec{a} = -2(2, 3) = (-4, -6)\)
Step 3: Verify magnitudes
|\(\vec{a}\)| = √(2² + 3²) = √13 ≈ 3.606
|3\(\vec{a}\)| = √(6² + 9²) = √(36 + 81) = √117 ≈ 10.817 = 3 × 3.606
|-2\(\vec{a}\)| = √((-4)² + (-6)²) = √(16 + 36) = √52 ≈ 7.211 = 2 × 3.606
Answer: \(3\vec{a} = (6, 9)\), \(-2\vec{a} = (-4, -6)\)
2
Problem: Verify the distributive property: \(2(\vec{a} + \vec{b}) = 2\vec{a} + 2\vec{b}\) where \(\vec{a} = (1, 2)\) and \(\vec{b} = (3, 1)\).

Step 1: Calculate left side: \(2(\vec{a} + \vec{b})\)
\(\vec{a} + \vec{b} = (1+3, 2+1) = (4, 3)\)
\(2(4, 3) = (8, 6)\)
Step 2: Calculate right side: \(2\vec{a} + 2\vec{b}\)
\(2\vec{a} = 2(1, 2) = (2, 4)\)
\(2\vec{b} = 2(3, 1) = (6, 2)\)
\(2\vec{a} + 2\vec{b} = (2+6, 4+2) = (8, 6)\)
Step 3: Verify equality
Both sides equal (8, 6), distributive property holds
Answer: Distributive property verified: both sides give (8, 6)
3
Problem: Find the unit vector in the direction of \(\vec{v} = (5, 12)\).

Step 1: Find the magnitude of \(\vec{v}\)
|\(\vec{v}\)| = √(5² + 12²) = √(25 + 144) = √169 = 13
Step 2: Divide by magnitude to get unit vector
\(\hat{v} = \dfrac{\vec{v}}{|\vec{v}|} = \dfrac{(5, 12)}{13} = \left(\dfrac{5}{13}, \dfrac{12}{13}\right)\)
Step 3: Verify unit vector properties
|\(\hat{v}\)| = √((5/13)² + (12/13)²) = √((25 + 144)/169) = √(169/169) = √1 = 1
Answer: Unit vector \(\hat{v} = \left(\dfrac{5}{13}, \dfrac{12}{13}\right)\)

Applications

Physics and Engineering

  • Force Scaling: Forces can be scaled by coefficients (e.g., 2F means twice the force)
  • Velocity Vectors: Speed changes scale velocity vectors
  • Unit Vectors: Direction vectors with magnitude 1 for navigation and orientation
  • Electric Fields: Field strength scaling in electrostatics

    • Computer Graphics and Games

  • Object Scaling: Resize 3D models and sprites
  • Animation: Smooth transitions and interpolations
  • Physics Engines: Mass, acceleration, and force calculations
  • AI Pathfinding: Direction vectors and movement calculations
    • Real-World Connection:

    In GPS navigation, velocity vectors are scaled by time to calculate displacement. If a car moves at 60 km/h north for 2 hours, its displacement vector is 120 km north (scalar multiplication by time).

    Key Takeaways

    • Scalar Associativity: \(k(l\vec{a}) = (kl)\vec{a}\) - scalars multiply together first
    • Distributivity over Scalars: \((k + l)\vec{a} = k\vec{a} + l\vec{a}\) - distribute scalars across vectors
    • Distributivity over Vectors: \(k(\vec{a} + \vec{b}) = k\vec{a} + k\vec{b}\) - distribute scalars across vector sums
    • Magnitude Scaling: |k\vec{a}| = |k| × |\vec{a}| - absolute value scaling
    • Direction: Same direction if k > 0, opposite if k < 0
    • Unit Vectors: \(\hat{v} = \vec{v}/|\vec{v}|\) - vectors with magnitude 1
    • Zero Properties: 0·\(\vec{a}\) = \(\vec{0}\), 1·\(\vec{a}\) = \(\vec{a}\)
    • Applications in physics, graphics, engineering, and navigation
