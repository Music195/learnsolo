Condition for Three Points to be Collinear
Three points A, B, and C are collinear (lie on the same straight line) if and only if the vector \(\overrightarrow{AB}\) and \(\overrightarrow{AC}\) are parallel, or if C coincides with A.
$$ \overrightarrow{AC} = k\overrightarrow{AB} $$
for some real number \(k\)

Mathematical Foundation

Three points A, B, and C are collinear if they lie on the same straight line. This condition can be expressed using vectors in several equivalent ways:

$$ \overrightarrow{AC} = k\overrightarrow{AB} \quad \text{or} \quad \overrightarrow{BC} = m\overrightarrow{BA} \quad \text{or} \quad \overrightarrow{CA} = n\overrightarrow{CB} $$

The scalar k represents the position of point C relative to A and B. If k = 0, then C coincides with A. If k = 1, then C coincides with B. If 0 < k < 1, then C lies between A and B.

This condition is equivalent to the area of triangle ABC being zero, or the vectors \(\overrightarrow{AB}\) and \(\overrightarrow{AC}\) being linearly dependent.

Practice Problems

Problem 1

Determine whether the points A(2,3), B(4,7), and C(6,11) are collinear.

Solution:

Check if \(\overrightarrow{AC} = k\overrightarrow{AB}\)

\(\overrightarrow{AB} = (4-2, 7-3) = (2,4)\)

\(\overrightarrow{AC} = (6-2, 11-3) = (4,8) = 2(2,4) = 2\overrightarrow{AB}\)

Since \(\overrightarrow{AC} = 2\overrightarrow{AB}\), the points are collinear with k = 2.

Problem 2

Find the value of k such that the points A(1,2), B(3,4), and C(k,6) are collinear.

Solution:

\(\overrightarrow{AB} = (3-1, 4-2) = (2,2)\)

\(\overrightarrow{AC} = (k-1, 6-2) = (k-1, 4)\)

Set \((k-1, 4) = m(2,2) = (2m, 2m)\)

So: k-1 = 2m and 4 = 2m ⇒ m = 2, k-1 = 4 ⇒ k = 5

The value is k = 5.

Problem 3

Show that the points A(1,-1), B(2,1), and C(3,3) are not collinear.

Solution:

\(\overrightarrow{AB} = (2-1, 1-(-1)) = (1,2)\)

\(\overrightarrow{AC} = (3-1, 3-(-1)) = (2,4)\)

Check if (2,4) = k(1,2) = (k, 2k)

2 = k and 4 = 2k ⇒ k = 2, but 4 ≠ 2×2 = 4? Wait, 4 = 2×2, so they are collinear!

Actually, these points ARE collinear with k = 2.

Key Takeaways
  • Three points are collinear if \(\overrightarrow{AC} = k\overrightarrow{AB}\) for some k
  • If k = 0, C coincides with A; if k = 1, C coincides with B
  • If 0 < k < 1, C lies between A and B on the line
  • Equivalent to area of triangle ABC being zero
  • Vectors \(\overrightarrow{AB}\) and \(\overrightarrow{AC}\) are linearly dependent
  • Used to check alignment in geometry and computer graphics
