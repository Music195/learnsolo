Position Vector of Division Points

Division Point Formulas

The position vector of a point dividing a line segment in a given ratio provides a powerful tool for coordinate geometry and vector analysis.

Division Point Formulas
For points A(\(\vec{a}\)) and B(\(\vec{b}\)), dividing AB in ratio m:n
Internal Division: \(\vec{r} = \frac{n\vec{a} + m\vec{b}}{m+n}\)
External Division: \(\vec{r} = \frac{-n\vec{a} + m\vec{b}}{m-n}\)
Midpoint: \(\vec{r} = \frac{\vec{a} + \vec{b}}{2}\)
Section Formula Relationship
\(\vec{r} = \frac{n\vec{a} + m\vec{b}}{m+n} = \frac{(m+n)\vec{a} + (m+n)\vec{b} - n\vec{a} - m\vec{b} + n\vec{a} + m\vec{b}}{m+n}\)
Derived from the weighted average of position vectors

Mathematical Foundation

The division formulas are derived from the concept of weighted averages and the properties of position vectors.

Internal Division Derivation
\(\vec{r} = \vec{a} + \frac{n}{m+n}(\vec{b} - \vec{a}) = \frac{m\vec{a} + n\vec{a} + n\vec{b} - n\vec{a}}{m+n} = \frac{m\vec{a} + n\vec{b}}{m+n}\)
Point divides segment from A to B in ratio m:n internally
External Division Derivation
\(\vec{r} = \vec{b} + \frac{n}{m-n}(\vec{a} - \vec{b}) = \frac{m\vec{b} - n\vec{b} + n\vec{a} - n\vec{b}}{m-n} = \frac{n\vec{a} + (m-n)\vec{b}}{m-n}\)
Point divides extended line in ratio m:n externally
Midpoint Special Case
\(\vec{r} = \frac{\vec{a} + \vec{b}}{2} = \frac{1\vec{a} + 1\vec{b}}{1+1}\)
When m = n = 1, we get the midpoint formula

Practice Problems

Test your understanding with these practice problems.

1
Find the position vector dividing points A(3,5) and B(9,7) in ratio 1:2 internally
Internal division formula: \(\vec{r} = \frac{n\vec{a} + m\vec{b}}{m+n}\)
Here m=1, n=2, so \(\vec{r} = \frac{2(3,5) + 1(9,7)}{1+2} = \frac{(6,10) + (9,7)}{3} = \frac{(15,17)}{3} = (5, \frac{17}{3})\)
Position vector = \((5, \frac{17}{3})\)
2
Find the position vector dividing points A(2,4) and B(6,8) externally in ratio 3:1
External division formula: \(\vec{r} = \frac{-n\vec{a} + m\vec{b}}{m-n}\)
Here m=3, n=1, so \(\vec{r} = \frac{-1(2,4) + 3(6,8)}{3-1} = \frac{(-2,-4) + (18,24)}{2} = \frac{(16,20)}{2} = (8,10)\)
Position vector = (8,10)
3
Find the midpoint of points A(1,3) and B(7,9)
Midpoint formula: \(\vec{r} = \frac{\vec{a} + \vec{b}}{2}\)
\(\vec{r} = \frac{(1,3) + (7,9)}{2} = \frac{(8,12)}{2} = (4,6)\)
Position vector = (4,6)

Key Takeaways

  • Internal division uses \(\frac{n\vec{a} + m\vec{b}}{m+n}\) for ratio m:n
  • External division uses \(\frac{-n\vec{a} + m\vec{b}}{m-n}\) for ratio m:n
  • Midpoint is special case when m = n = 1
  • Formulas work for any dimension vectors
  • Essential for coordinate geometry and vector analysis
  • Used in interpolation and weighted averaging
