Parabola: Basic Properties

Core Concept: Parabola

A parabola is the set of all points in a plane that are equidistant from a fixed point (focus) and a fixed line (directrix). It is one of the four conic sections and has many important applications in physics, engineering, and mathematics.

Standard Equation: \( y^2 = 4px \quad (p \neq 0) \)
  • Vertex: Origin O(0,0)
  • Focus: Point F(p, 0)
  • Directrix: Line x = -p
  • Axis of symmetry: x-axis
  • Latus rectum length: 4|p|
  • Focal parameter: p
Key Geometric Property

For any point P on the parabola, the distance from P to the focus equals the distance from P to the directrix.

Distance to focus = Distance to directrix
PF = PM (where M is foot of perpendicular from P to directrix)

Mathematical Foundation

Distance Property
For point P(x,y) on parabola y² = 4px:
Distance to focus: √[(x-p)² + y²]
Distance to directrix: |x - (-p)| = |x + p|
Set equal: √[(x-p)² + y²] = |x + p|
Square both sides: (x-p)² + y² = (x + p)²
Expand: x² - 2px + p² + y² = x² + 2px + p²
Simplify: y² = 4px
Alternative Forms
Horizontal parabola: y² = 4px
Vertical parabola: x² = 4py
General form: (y - k)² = 4p(x - h)
Vertex at (h,k), focus at (h+p,k)

Worked Examples

1

Find the focus, directrix, and latus rectum of the parabola y² = 8x.

Equation: y² = 8x
Compare with y² = 4px: 4p = 8 ⇒ p = 2
Vertex: (0,0)
Focus: (p,0) = (2,0)
Directrix: x = -p = x = -2
Latus rectum length: 4|p| = 8
Latus rectum: x = 2, from y = -4 to y = 4
2

Find the equation of parabola with vertex (1,2) and focus (3,2).

Vertex: (h,k) = (1,2)
Focus: (h+p,k) = (3,2)
So h+p = 3, h=1 ⇒ p = 2
General form: (y - k)² = 4p(x - h)
(y - 2)² = 4×2(x - 1)
(y - 2)² = 8(x - 1)
3

Test if point (4,4) lies on the parabola y² = 8x.

Point: (4,4), Equation: y² = 8x
Left side: 4² = 16
Right side: 8×4 = 32
16 ≠ 32, so point does not lie on parabola
Distance to focus: √[(4-2)² + 4²] = √[4 + 16] = √20 = 2√5 ≈ 4.47
Distance to directrix: |4 - (-2)| = 6
4.47 ≠ 6, confirming point is not on parabola

Applications and Importance

Real-World Applications
  • Physics: Projectile motion follows parabolic trajectory
  • Engineering: Suspension bridges, arches, reflectors
  • Optics: Parabolic mirrors focus parallel light rays
  • Antennas: Satellite dishes use parabolic surfaces
  • Architecture: Arches and curved structures
  • Mathematics: Optimization problems, calculus applications

Key Takeaways

  • Parabola: set of points equidistant from focus and directrix
  • Standard form y² = 4px has vertex at origin, opens right
  • Focus located at (p,0), directrix at x = -p
  • Latus rectum length is 4|p|, parallel to directrix
  • Distance property: PF = PM for any point P on parabola
  • Vertex is midpoint between focus and directrix
  • Fundamental conic section with wide applications
