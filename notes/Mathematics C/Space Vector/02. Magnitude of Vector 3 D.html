Main Formula:
For vector $\vec{a} = (a_1, a_2, a_3)$, the magnitude is:
$$|\vec{a}| = \sqrt{a_1^2 + a_2^2 + a_3^2}$$

Understanding Vector Magnitude

The magnitude (or length) of a vector $\vec{a} = (a_1, a_2, a_3)$ represents the distance from the origin (0,0,0) to the point (a‚ÇÅ, a‚ÇÇ, a‚ÇÉ) in 3D space.

Geometric interpretation:
  • The magnitude is always a non-negative real number
  • It represents the "length" or "size" of the vector
  • A vector with magnitude 1 is called a unit vector
  • A vector with magnitude 0 is the zero vector
  • The magnitude is denoted by $||\vec{a}||$ or simply $|\vec{a}|$

Derivation Using Pythagorean Theorem

The formula is derived by applying the Pythagorean theorem twice:

Step-by-step derivation:

  1. Consider the vector $\vec{a} = (a_1, a_2, a_3)$ from origin to point A(a‚ÇÅ, a‚ÇÇ, a‚ÇÉ)
  2. The projection of A onto the xy-plane is point B(a‚ÇÅ, a‚ÇÇ, 0)
  3. Distance from origin to B: $OB = \sqrt{a_1^2 + a_2^2}$ (2D distance)
  4. Distance from B to A: $BA = |a_3|$ (vertical distance)
  5. By Pythagorean theorem: $OA^2 = OB^2 + BA^2$
  6. $OA^2 = (a_1^2 + a_2^2) + a_3^2 = a_1^2 + a_2^2 + a_3^2$
  7. Therefore: $OA = |\vec{a}| = \sqrt{a_1^2 + a_2^2 + a_3^2}$

Worked Examples

Example 1: Basic Vector

Find the magnitude of vector $\vec{v} = (3, 4, 5)$.

Solution:

$|\vec{v}| = \sqrt{3^2 + 4^2 + 5^2} = \sqrt{9 + 16 + 25} = \sqrt{50} = 5\sqrt{2} \approx 7.07$

Example 2: Unit Vector

Find the magnitude of vector $\vec{u} = \left(\frac{1}{\sqrt{3}}, \frac{1}{\sqrt{3}}, \frac{1}{\sqrt{3}}\right)$.

Solution:

$|\vec{u}| = \sqrt{\left(\frac{1}{\sqrt{3}}\right)^2 + \left(\frac{1}{\sqrt{3}}\right)^2 + \left(\frac{1}{\sqrt{3}}\right)^2} = \sqrt{\frac{1}{3} + \frac{1}{3} + \frac{1}{3}} = \sqrt{1} = 1$

This is a unit vector!

Example 3: Zero Vector

Find the magnitude of vector $\vec{0} = (0, 0, 0)$.

Solution:

$|\vec{0}| = \sqrt{0^2 + 0^2 + 0^2} = \sqrt{0} = 0$

The zero vector has zero magnitude.

Practice Problems

Problem 1: Calculate the magnitude of vector $\vec{a} = (2, 3, 6)$.

Solution:

$|\vec{a}| = \sqrt{2^2 + 3^2 + 6^2} = \sqrt{4 + 9 + 36} = \sqrt{49} = 7$
Problem 2: Find the magnitude of vector $\vec{b} = (-1, -2, 3)$.

Solution:

$|\vec{b}| = \sqrt{(-1)^2 + (-2)^2 + 3^2} = \sqrt{1 + 4 + 9} = \sqrt{14} \approx 3.74$
Problem 3: Determine if $\vec{c} = \left(\frac{3}{5}, \frac{4}{5}, 0\right)$ is a unit vector.

Solution:

$|\vec{c}| = \sqrt{\left(\frac{3}{5}\right)^2 + \left(\frac{4}{5}\right)^2 + 0^2} = \sqrt{\frac{9}{25} + \frac{16}{25}} = \sqrt{\frac{25}{25}} = \sqrt{1} = 1$

Yes, this is a unit vector.
Problem 4: Find the magnitude of vector $\vec{d} = (1, 1, 1, 1)$ in 4D space.

Solution:

The formula generalizes to n dimensions: $|\vec{d}| = \sqrt{1^2 + 1^2 + 1^2 + 1^2} = \sqrt{4} = 2$
Problem 5: If a vector has magnitude 10 and points in the direction of $\vec{u} = (3, 4, 0)$, find its components.

Solution:

First, find the unit vector in the direction of $\vec{u}$:
$|\vec{u}| = \sqrt{3^2 + 4^2 + 0^2} = \sqrt{9 + 16} = \sqrt{25} = 5$
Unit vector: $\hat{u} = \left(\frac{3}{5}, \frac{4}{5}, 0\right)$

Vector with magnitude 10: $10 \times \hat{u} = \left(10 \times \frac{3}{5}, 10 \times \frac{4}{5}, 0\right) = (6, 8, 0)$
