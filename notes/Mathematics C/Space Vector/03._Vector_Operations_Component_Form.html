Vector Addition (Âíå)

The sum of two vectors $\vec{a}$ and $\vec{b}$ is found by adding their corresponding components:

$$ \begin{array}{l} (a_1, a_2, a_3) + (b_1, b_2, b_3) \\ = (a_1 + b_1, a_2 + b_2, a_3 + b_3) \end{array} $$
Example: $(2, 3, 4) + (1, -2, 3) = (3, 1, 7)$

Vector Subtraction (Â∑Æ)

The difference of vector $\vec{b}$ from vector $\vec{a}$ is found by subtracting the corresponding components:

$$ \begin{array}{l} (a_1, a_2, a_3) - (b_1, b_2, b_3) \\ = (a_1 - b_1, a_2 - b_2, a_3 - b_3) \end{array} $$
Example: $(5, 7, 2) - (1, 3, -1) = (4, 4, 3)$

Scalar Multiplication (ÂÆüÊï∞ÂÄç)

The product of a scalar $k$ and a vector $\vec{a}$ is found by multiplying each component of $\vec{a}$ by $k$:

$$ \begin{array}{l} k(a_1, a_2, a_3) \\ = (ka_1, ka_2, ka_3) \end{array} $$

(Note: $k$ is a real number)

Example: $2(3, -1, 4) = (6, -2, 8)$
üí° Key Concepts:
  • Vector operations in 3D (and other dimensions) are performed component-wise.
  • Vector Addition: Add corresponding components. Geometrically, this follows the parallelogram law or head-to-tail rule.
  • Vector Subtraction: Subtract corresponding components. $\vec{a} - \vec{b}$ is equivalent to $\vec{a} + (-\vec{b})$.
  • Scalar Multiplication: Multiply each component by the scalar. This scales the vector's magnitude and/or reverses its direction.
    • If $k > 0$, the direction is unchanged, magnitude scales by $|k|$.
    • If $k < 0$, the direction is reversed, magnitude scales by $|k|$.
    • If $k = 0$, the result is the zero vector $(0,0,0)$.
  • These operations are fundamental in linear algebra, physics, computer graphics, and many other fields.

Worked Examples

Example 1: Vector Addition

Find the sum of vectors $\vec{a} = (4, -2, 3)$ and $\vec{b} = (-1, 5, 2)$.

Solution:

$\vec{a} + \vec{b} = (4 + (-1), -2 + 5, 3 + 2) = (3, 3, 5)$

Example 2: Vector Subtraction

Find the difference $\vec{a} - \vec{b}$ where $\vec{a} = (7, 1, -3)$ and $\vec{b} = (2, -4, 1)$.

Solution:

$\vec{a} - \vec{b} = (7 - 2, 1 - (-4), -3 - 1) = (5, 5, -4)$

Example 3: Scalar Multiplication

Find $3\vec{v}$ where $\vec{v} = (2, -1, 4)$.

Solution:

$3\vec{v} = (3 \times 2, 3 \times (-1), 3 \times 4) = (6, -3, 12)$

Example 4: Combined Operations

Find $2\vec{a} - 3\vec{b}$ where $\vec{a} = (1, 2, 3)$ and $\vec{b} = (-1, 0, 2)$.

Solution:

$2\vec{a} = (2, 4, 6)$

$3\vec{b} = (-3, 0, 6)$

$2\vec{a} - 3\vec{b} = (2 - (-3), 4 - 0, 6 - 6) = (5, 4, 0)$

Practice Problems

Problem 1: Calculate $\vec{a} + \vec{b}$ where $\vec{a} = (3, -1, 4)$ and $\vec{b} = (2, 5, -3)$.

Solution:

$\vec{a} + \vec{b} = (3 + 2, -1 + 5, 4 + (-3)) = (5, 4, 1)$
Problem 2: Find $\vec{p} - \vec{q}$ where $\vec{p} = (6, 2, -1)$ and $\vec{q} = (1, -3, 4)$.

Solution:

$\vec{p} - \vec{q} = (6 - 1, 2 - (-3), -1 - 4) = (5, 5, -5)$
Problem 3: Compute $-2\vec{v}$ where $\vec{v} = (4, -3, 7)$.

Solution:

$-2\vec{v} = (-2 \times 4, -2 \times (-3), -2 \times 7) = (-8, 6, -14)$
Problem 4: Evaluate $3\vec{a} + 2\vec{b} - \vec{c}$ where $\vec{a} = (1, 0, 2)$, $\vec{b} = (-1, 3, 1)$, and $\vec{c} = (2, -1, 0)$.

Solution:

$3\vec{a} = (3, 0, 6)$
$2\vec{b} = (-2, 6, 2)$
$\vec{c} = (2, -1, 0)$
$3\vec{a} + 2\vec{b} - \vec{c} = (3 + (-2) + (-2), 0 + 6 + (-(-1)), 6 + 2 + (-0)) = (-1, 7, 8)$
Problem 5: If $\vec{u} + \vec{v} = (5, 7, -2)$ and $\vec{u} - \vec{v} = (1, -1, 4)$, find $\vec{u}$ and $\vec{v}$.

Solution:

Let $\vec{u} = (u_1, u_2, u_3)$ and $\vec{v} = (v_1, v_2, v_3)$
Adding the equations: $2\vec{u} = (6, 6, 2)$ ‚Üí $\vec{u} = (3, 3, 1)$
Subtracting the equations: $2\vec{v} = (4, -8, 6)$ ‚Üí $\vec{v} = (2, -4, 3)$

  • Vector operations are performed component-wise in each dimension
  • Vector addition: $(a_x, a_y, a_z) + (b_x, b_y, b_z) = (a_x + b_x, a_y + b_y, a_z + b_z)$
  • Vector subtraction: $(a_x, a_y, a_z) - (b_x, b_y, b_z) = (a_x - b_x, a_y - b_y, a_z - b_z)$
  • Scalar multiplication: $k(a_x, a_y, a_z) = (k a_x, k a_y, k a_z)$
