Main Formula:
The distance between the origin O(0,0,0) and a point P(a, b, c) in 3D space is:
$$OP = \sqrt{a^2 + b^2 + c^2}$$

Derivation of the Distance Formula

The distance formula in 3D space is derived using the Pythagorean theorem applied twice:

Step-by-step derivation:
  1. Let Q be the projection of point P onto the xy-plane. Q has coordinates (a,b,0)
  2. The distance OQ in the xy-plane is: $OQ = \sqrt{a^2 + b^2}$
  3. Consider right triangle â–³OQP with right angle at Q
  4. The distance QP (vertical leg) is: $QP = |c - 0| = |c| = c$ (assuming c â‰¥ 0)
  5. By Pythagorean theorem: $OP^2 = OQ^2 + QP^2$
  6. Substitute: $OP^2 = (a^2 + b^2) + c^2 = a^2 + b^2 + c^2$
  7. Therefore: $OP = \sqrt{a^2 + b^2 + c^2}$

Worked Examples

Example 1: Basic Calculation

Find the distance from the origin to the point P(2, 3, 6).

Solution:

$OP = \sqrt{2^2 + 3^2 + 6^2} = \sqrt{4 + 9 + 36} = \sqrt{49} = 7$

The distance is 7 units.

Example 2: Decimal Coordinates

Find the distance from the origin to the point P(1.5, 2.5, 3.5).

Solution:

$OP = \sqrt{(1.5)^2 + (2.5)^2 + (3.5)^2} = \sqrt{2.25 + 6.25 + 12.25} = \sqrt{20.75} \approx 4.56$

The distance is approximately 4.56 units.

Example 3: Negative Coordinates

Find the distance from the origin to the point P(-2, -3, 4).

Solution:

$OP = \sqrt{(-2)^2 + (-3)^2 + 4^2} = \sqrt{4 + 9 + 16} = \sqrt{29} \approx 5.39$

The distance is approximately 5.39 units. (Note: squares of negatives are positive)

Practice Problems

The distance is 3 units.

The distance is 13 units.

The distance is approximately 6.23 units.
Problem 5: What point is exactly 5 units from the origin?

Solution:

Any point P(a,b,c) where $\sqrt{a^2 + b^2 + c^2} = 5$

Squaring both sides: $a^2 + b^2 + c^2 = 25$

Examples: P(3,4,0), P(0,3,4), P(5,0,0), etc.

All points on the sphere with radius 5 centered at the origin.

ðŸŽ¯ Key Takeaways

  • The 3D distance formula is $OP = \sqrt{a^2 + b^2 + c^2}$ for point P(a,b,c)
  • This is an extension of the 2D distance formula using Pythagorean theorem twice
  • The distance is always positive (or zero for the origin)
  • Squares of negative coordinates become positive in the formula
  • This formula gives the magnitude of the position vector $\vec{OP} = \langle a,b,c \rangle$
  • Points at equal distance from origin lie on a sphere
  • The formula generalizes to n dimensions: $\sqrt{x_1^2 + x_2^2 + \dots + x_n^2}$
