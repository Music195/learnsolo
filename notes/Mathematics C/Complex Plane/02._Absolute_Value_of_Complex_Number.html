Absolute Value of a Complex Number

Core Concept: Modulus of Complex Numbers

The absolute value (or modulus) of a complex number measures its distance from the origin in the complex plane. This geometric interpretation connects complex numbers to the familiar concept of distance in the real plane.

The absolute value of a complex number \( z = a + bi \) is:

|z| = |a + bi| = \sqrt{a^2 + b^2}
Key Properties:
  • When z is real, |z| equals the absolute value of the real number
  • |z| ‚â• 0 for all complex numbers
  • |z| = 0 only when z = 0
  • |z| represents distance from origin to point (a,b)

Geometric Interpretation

Distance from Origin: The modulus |z| represents the straight-line distance from the origin (0,0) to the point (a,b) in the complex plane. This is exactly the same as the distance formula in analytic geometry.
Circle Definition: The set of all complex numbers with modulus r forms a circle of radius r centered at the origin. This is fundamental for understanding complex number equations and inequalities.
