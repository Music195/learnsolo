Rotation About the Origin

Core Concept: Complex Rotation

Multiplication by a complex number on the unit circle rotates points in the complex plane about the origin. This fundamental property connects complex multiplication to geometric transformations.

For any complex number z and unit complex number e^(iθ) = cos θ + i sin θ:

e^{iθ} \cdot z = \text{rotation of } z \text{ by angle } θ

The point e^(iθ)z is obtained by rotating z counterclockwise about the origin by angle θ.

Key Properties:
  • Unit circle: |e^(iθ)| = 1, so rotation preserves distance from origin
  • Angle addition: Multiple rotations add angles
  • Composition: Rotation by θ₁ then θ₂ = rotation by θ₁ + θ₂
  • Counterclockwise: Positive angles rotate counterclockwise
  • Periodicity: Rotation by 2π returns to original position

Mathematical Foundation

Rotation Formula
z = a + bi
e^{iθ}z = (cos θ + i sin θ)(a + bi)
= a cos θ - b sin θ + i(a sin θ + b cos θ)
= (a cos θ - b sin θ) + i(a sin θ + b cos θ)
Matrix Representation
\begin{pmatrix} \cos\theta & -\sin\theta \\ \sin\theta & \cos\theta \end{pmatrix} \begin{pmatrix} a \\ b \end{pmatrix} = \begin{pmatrix} a\cos\theta - b\sin\theta \\ a\sin\theta + b\cos\theta \end{pmatrix}
Geometric Interpretation: The rotation matrix above is the standard rotation matrix in 2D geometry. Complex multiplication provides an elegant way to represent and compute rotations.

Worked Examples

1

Rotate the point z = 1 + i by 90° counterclockwise about the origin.

z = 1 + i, θ = 90° = π/2 radians
Rotation factor: e^(iπ/2) = cos(π/2) + i sin(π/2) = 0 + i×1 = i
Rotated point: i × z = i × (1 + i) = i×1 + i×i = i + i² = i + (-1) = -1 + i
Geometrically: (1,1) rotates 90° counterclockwise to (-1,1)
Verification: The point moved from first quadrant to second quadrant
2

Find the rotation that maps z = 2 + 3i to w = -2 + 3i.

Original: z = 2 + 3i, Target: w = -2 + 3i
We need: e^(iθ)z = w
So: e^(iθ) = w/z = (-2 + 3i)/(2 + 3i)
Multiply numerator and denominator by conjugate of denominator
= (-2 + 3i)(2 - 3i) / (2 + 3i)(2 - 3i) = (-4 - 6i + 9i - 9)/(-4 + 9) = (-13 - 6i + 9i)/5 = (-13 + 3i)/5
= -13/5 + 3i/5 = -2.6 + 0.6i
arg(-2.6 + 0.6i) ≈ 180° - tan⁻¹(0.6/2.6) ≈ 180° - 13° ≈ 167°
So the rotation angle is approximately 167°
3

Rotate z = √3 + i by 30° counterclockwise.

z = √3 + i, θ = 30° = π/6 radians
Rotation factor: e^(iπ/6) = cos(π/6) + i sin(π/6) = √3/2 + i/2
Rotated point: (√3/2 + i/2)(√3 + i)
= √3/2 × √3 + √3/2 × i + i/2 × √3 + i/2 × i
= 3/2 + i√3/2 + i√3/2 + i²/2 = 3/2 + i√3 - 1/2 = (3/2 - 1/2) + i√3 = 1 + i√3
The point (√3, 1) rotated by 30° gives (1, √3)

Applications in Mathematics and Physics

Key Applications
  • Signal processing: Phase shifting in frequency domain

  • Computer graphics: 2D transformations and animations
  • Electrical engineering: Phasor analysis in AC circuits
  • Navigation: Coordinate system rotations
  • Fractal geometry: Rotational symmetries in patterns

Key Takeaways

  • e^(iθ)z rotates z by angle θ counterclockwise about origin
  • Rotation preserves distance from origin: |e^(iθ)z| = |z|
  • Multiple rotations: e^(iθ₁)e^(iθ₂)z = e^(i(θ₁+θ₂))z
  • Matrix form connects to linear algebra rotations
  • Essential for complex analysis and geometric transformations
  • Unit circle elements generate all rotations
