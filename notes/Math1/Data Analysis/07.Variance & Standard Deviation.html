Variance & Standard Deviation (分散・標準偏差)

Data Analysis
For data values \( x_1, x_2, \ldots, x_n \) with mean \( \overline{x} \):
Variance:
\( s^2 = \dfrac{1}{n} \left\{ (x_1 - \overline{x})^2 + (x_2 - \overline{x})^2 + \cdots + (x_n - \overline{x})^2 \right\} \)

Standard Deviation:
\( s = \sqrt{ \dfrac{1}{n} \left\{ (x_1 - \overline{x})^2 + (x_2 - \overline{x})^2 + \cdots + (x_n - \overline{x})^2 \right\} } \)
💡 Variance and standard deviation are measures of how spread out the data is. The standard deviation is the square root of the variance and is in the same unit as the data.
