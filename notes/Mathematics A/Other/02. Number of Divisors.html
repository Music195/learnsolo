Number of Divisors

Core Concept

The number of divisors (also called the divisor function) tells us how many positive divisors a number has. For any natural number with prime factorization:

$N = p_1^{a_1} \times p_2^{a_2} \times \cdots \times p_k^{a_k}$

The number of positive divisors $d(N)$ is given by:

Number of Divisors Formula

$d(N) = (a_1 + 1)(a_2 + 1) \cdots (a_k + 1)$

This formula works because each prime factor can contribute different powers (from 0 to the maximum exponent) to form divisors.

Detailed Examples

Example 1: Simple Prime Number

For a prime number like 7:

$7 = 7^1$

Number of divisors: $(1 + 1) = 2$

The divisors are: 1 and 7

Example 2: Prime Power

For $2^4 = 16$:

$16 = 2^4$

Number of divisors: $(4 + 1) = 5$

The divisors are: $2^0, 2^1, 2^2, 2^3, 2^4 = 1, 2, 4, 8, 16$

Example 3: Multiple Prime Factors

For $60 = 2^2 \times 3^1 \times 5^1$:

$60 = 2^2 \times 3^1 \times 5^1$

Number of divisors: $(2 + 1)(1 + 1)(1 + 1) = 3 \times 2 \times 2 = 12$

The divisors are: 1, 2, 3, 4, 5, 6, 10, 12, 15, 20, 30, 60

Applications

  • Number Theory: Used in studying perfect numbers, abundant numbers, and deficient numbers
  • Cryptography: Important in RSA algorithm for finding numbers with specific divisor properties
  • Computer Science: Used in algorithms for finding factors and optimizing computations
  • Mathematics Competitions: Common in problems involving highly composite numbers

Practice Problems

1

Find the number of positive divisors of 72.

First, factorize 72: $72 = 2^3 \times 3^2$

Number of divisors: $(3+1)(2+1) = 4 \times 3 = 12$

The divisors are: 1, 2, 3, 4, 6, 8, 9, 12, 18, 24, 36, 72

2

How many positive divisors does 100 have?

Factorize 100: $100 = 2^2 \times 5^2$

Number of divisors: $(2+1)(2+1) = 3 \times 3 = 9$

The divisors are: 1, 2, 4, 5, 10, 20, 25, 50, 100

3

Find the number of divisors of 210.

Factorize 210: $210 = 2^1 \times 3^1 \times 5^1 \times 7^1$

Number of divisors: $(1+1)(1+1)(1+1)(1+1) = 2 \times 2 \times 2 \times 2 = 16$

The divisors are: 1, 2, 3, 5, 6, 7, 10, 14, 15, 21, 30, 35, 42, 70, 105, 210

4

If a number has exactly 12 positive divisors, what could its prime factorization be?

Possible factorizations where $(a+1)(b+1)\cdots = 12$:

  • $12 = 12 \times 1$: Not possible (would be $p^{11}$)
  • $12 = 6 \times 2$: $p^5 \times q^1$
  • $12 = 4 \times 3$: $p^3 \times q^2$
  • $12 = 3 \times 2 \times 2$: $p^2 \times q^1 \times r^1$

Key Takeaways

  • The number of divisors depends only on the exponents in the prime factorization
  • Each prime factor contributes (exponent + 1) to the total divisor count
  • Numbers with many small prime factors tend to have more divisors
  • The formula works for any natural number greater than 1
  • Prime numbers have exactly 2 divisors: 1 and themselves
