Basic Properties of Probability

Axioms of Probability

Three Fundamental Axioms

  1. Non-negativity: For any event A, P(A) ≥ 0
  2. Normalization: The probability of the sample space S is 1, P(S) = 1
  3. Additivity: For mutually exclusive events A and B, P(A ∪ B) = P(A) + P(B)

These three axioms form the foundation of probability theory. All other probability rules can be derived from these axioms.

Basic Properties

1. Probability Range

For any event A: 0 ≤ P(A) ≤ 1

  • P(A) = 0: Impossible event
  • 0 < P(A) < 1: Possible event
  • P(A) = 1: Certain event

2. Impossible and Certain Events

P(∅) = 0 (Empty set - impossible event)

P(S) = 1 (Sample space - certain event)

3. Complement Rule

For any event A: P(A') = 1 - P(A)

Where A' is the complement of A (everything not in A)

Addition Law

Addition Law of Probability

For any two events A and B:

P(A ∪ B) = P(A) + P(B) - P(A ∩ B)

This is the general addition law for any two events

Special Case: Mutually Exclusive Events

When A and B are mutually exclusive (A ∩ B = ∅):

P(A ∪ B) = P(A) + P(B)

No overlap between events, so no subtraction needed

Detailed Examples

Example 1: Complement Rule

A fair coin is tossed. What is the probability of not getting heads?

P(Heads) = 1/2 = 0.5
Complement: P(Not Heads) = P(Tails)
Formula: P(A') = 1 - P(A) = 1 - 0.5 = 0.5
Same result: P(Tails) = 1/2 = 0.5

Example 2: Addition Law (Mutually Exclusive)

A die is rolled. What is the probability of getting a 2 or a 4?

P(2) = 1/6, P(4) = 1/6
Events are mutually exclusive (can't get both 2 and 4)
Formula: P(A ∪ B) = P(A) + P(B) = 1/6 + 1/6 = 2/6 = 1/3

Example 3: Addition Law (General)

In a class of 30 students, 12 play soccer, 8 play basketball, and 3 play both. What is the probability a randomly selected student plays soccer or basketball?

P(Soccer) = 12/30 = 2/5, P(Basketball) = 8/30 = 4/15
P(Both) = 3/30 = 1/10
Formula: P(A ∪ B) = P(A) + P(B) - P(A ∩ B)
Calculation: 2/5 + 4/15 - 1/10 = (12/30 + 8/30 - 3/30) = 17/30

Example 4: Probability Range

Check if 1.5 can be a valid probability.

Probability must satisfy: 0 ≤ P(A) ≤ 1
1.5 > 1, so it violates the upper bound
Therefore, 1.5 is not a valid probability

Applications

  • Quality Control: Probability of defective products in manufacturing
  • Medical Testing: False positive/negative rates in diagnostics
  • Risk Assessment: Calculating combined probabilities of multiple risks
  • Insurance: Determining coverage for multiple events
  • Computer Science: Error probabilities in data transmission
  • Sports Analytics: Combined probabilities of different game outcomes

Practice Problems

1

If P(A) = 0.3, what is P(A')?

Using complement rule: P(A') = 1 - P(A) = 1 - 0.3 = 0.7

2

A card is drawn from a deck. What is the probability of not drawing a heart?

P(Heart) = 13/52 = 1/4

P(Not Heart) = 1 - P(Heart) = 1 - 1/4 = 3/4

There are 39 non-heart cards out of 52 total.

3

In a group of 50 people, 25 like tea, 20 like coffee, and 10 like both. How many like tea or coffee?

Using addition law: P(A ∪ B) = P(A) + P(B) - P(A ∩ B)

25 + 20 - 10 = 35 people like tea or coffee

4

Which of these cannot be a probability: -0.2, 0.5, 1.2, 0.95?

Probability must satisfy: 0 ≤ P ≤ 1

-0.2 is less than 0 (invalid)

1.2 is greater than 1 (invalid)

0.5 and 0.95 are both valid probabilities

Key Takeaways

  • Axioms: Three fundamental rules (non-negativity, normalization, additivity)
  • Range: All probabilities satisfy 0 ≤ P(A) ≤ 1
  • Complement: P(A') = 1 - P(A) for any event A
  • Impossible: P(∅) = 0, certain: P(S) = 1
  • Addition Law: P(A ∪ B) = P(A) + P(B) - P(A ∩ B)
  • Mutually Exclusive: If A ∩ B = ∅, then P(A ∪ B) = P(A) + P(B)
  • Derived Rules: All other probability rules come from these basic properties
