Subset Properties
Basic Subset Relationship
A = {1, 2} and B = {1, 2, 3, 4}

Since every element of A (1 and 2) is also in B, we write A ⊆ B.

This is also a proper subset: A ⊂ B (since A ≠ B).

Equal Sets are Subsets
C = {x, y, z} and D = {x, y, z}

If A = B, then A ⊆ B and B ⊆ A.

Equal sets are subsets of each other.

Power Set Concept
S = {1, 2}, P(S) = {∅, {1}, {2}, {1, 2}}

The power set of S contains all possible subsets of S.

∅ ⊆ S, {1} ⊆ S, {2} ⊆ S, {1, 2} ⊆ S

The Empty Set
∅ = {} (The Empty Set)
Empty Set Properties

The empty set contains no elements.

∅ = {}

Key Property: The empty set is a subset of every set.

∅ ⊆ A for any set A
Empty Set Examples

The set of all positive integers greater than 5 and less than 2:

{x ∈ ℤ⁺ | 5 < x < 2} = ∅

The set of all real numbers that are both positive and negative:

{x ∈ ℝ | x > 0 and x < 0} = ∅
Singleton Set vs Empty Set

A singleton set has exactly one element:

{5} ≠ ∅

The empty set has zero elements, while {∅} has one element (the empty set itself).

Practice Problems
Problem 1: Subset Relationships

Given sets A = {1, 2, 3}, B = {1, 2}, C = {1, 2, 3, 4}, determine which of the following are true:

a) B ⊆ A

b) A ⊆ C

c) C ⊆ A

d) A ⊆ B

a) B ⊆ A: True (every element of B is in A)

b) A ⊆ C: True (every element of A is in C)

c) C ⊆ A: False (4 is in C but not in A)

d) A ⊆ B: False (3 is in A but not in B)

Problem 2: Empty Set Subset

Is the empty set ∅ a subset of the set A = {1, 2, 3}? Explain why.

Solution: Yes, ∅ ⊆ A

The empty set is a subset of every set because there are no elements in ∅ that are not in A. The subset definition requires that every element of the first set is in the second set. Since there are no elements to check, this condition is vacuously true.

Problem 3: Proper Subsets

List all proper subsets of the set {1, 2}.

Solution: ∅, {1}, {2}

The proper subsets are all subsets except the set itself. The power set of {1, 2} is {∅, {1}, {2}, {1, 2}}, so the proper subsets are ∅, {1}, and {2}.

Problem 4: Set-Builder Empty Set

Write the set {x ∈ ℝ | x² = -1} in roster form.

Solution: ∅

There are no real numbers x such that x² = -1, since the square of any real number is non-negative. Therefore, this set is empty.

Key Takeaways
✓ A ⊆ B means every element of A is also an element of B
✓ A ⊂ B means A is a proper subset of B (A ⊆ B and A ≠ B)
✓ The empty set ∅ is a subset of every set
✓ Equal sets are subsets of each other
✓ The power set contains all possible subsets of a set
✓ ∅ = {} (empty set notation)
