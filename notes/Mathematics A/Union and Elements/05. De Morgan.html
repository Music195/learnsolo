De Morgan's Laws

Core Concept: De Morgan's Laws

De Morgan's Laws describe how the complement operation interacts with union and intersection operations. These fundamental laws are essential for set theory, logic, and computer science, providing a way to simplify complex set expressions.

De Morgan's First Law

$\overline{A \cup B} = \overline{A} \cap \overline{B}$

De Morgan's Second Law

$\overline{A \cap B} = \overline{A} \cup \overline{B}$

These laws state that the complement of a union is the intersection of complements, and the complement of an intersection is the union of complements.

$\overline{A \cup B} = \overline{A} \cap \overline{B}$
Complement of Union
$\overline{A \cap B} = \overline{A} \cup \overline{B}$
Complement of Intersection

Worked Examples

1

Verify De Morgan's First Law: $\overline{A \cup B} = \overline{A} \cap \overline{B}$

Let U = {1, 2, 3, 4, 5, 6}, A = {1, 2, 3}, B = {3, 4, 5}

A ∪ B = {1, 2, 3, 4, 5}
$\overline{A ∪ B}$ = {6}
$\overline{A}$ = {4, 5, 6}, $\overline{B}$ = {1, 2, 6}
$\overline{A} ∩ \overline{B}$ = {6}
Both sides equal: {6}
2

Verify De Morgan's Second Law: $\overline{A ∩ B} = \overline{A} ∪ \overline{B}$

Let U = {1, 2, 3, 4, 5, 6}, A = {1, 2, 3}, B = {2, 3, 4}

A ∩ B = {2, 3}
$\overline{A ∩ B}$ = {1, 4, 5, 6}
$\overline{A}$ = {4, 5, 6}, $\overline{B}$ = {1, 5, 6}
$\overline{A} ∪ \overline{B}$ = {1, 4, 5, 6}
Both sides equal: {1, 4, 5, 6}

Key Takeaways

  • De Morgan's First Law: The complement of a union is the intersection of complements
  • De Morgan's Second Law: The complement of an intersection is the union of complements
  • These laws allow simplification of complex set expressions
  • Essential for set theory, logic, and computer science applications
  • Can be extended to more than two sets using mathematical induction
