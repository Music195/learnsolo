Arithmetic Mean - Geometric Mean Inequality
The AM-GM inequality states that for positive real numbers, the arithmetic mean is always greater than or equal to the geometric mean. This fundamental inequality has numerous applications in mathematics and optimization.
Two Variables: $$\frac{a + b}{2} \geq \sqrt{ab}$$
$$\text{Equality when } a = b$$
Three Variables: $$\frac{a + b + c}{3} \geq \sqrt[3]{abc}$$
$$\text{Equality when } a = b = c$$
General Case: $$\frac{a_1 + a_2 + \dots + a_n}{n} \geq \sqrt[n]{a_1 a_2 \dots a_n}$$
$$\text{Equality when all } a_i \text{ equal}$$
For positive real numbers, the arithmetic mean is always greater than or equal to the geometric mean. Equality holds if and only if all numbers are equal. This inequality is fundamental in optimization and has applications in probability, statistics, and analysis.

Inequality Properties

Two Variables

$$\frac{a + b}{2} \geq \sqrt{ab}$$
Key Points:
• AM ≥ GM for positive a,b
• Equality when a = b
• Proof: (√a - √b)² ≥ 0
• Used in optimization

Three Variables

$$\frac{a + b + c}{3} \geq \sqrt[3]{abc}$$
Key Points:
• Extends to three variables
• Equality when a = b = c
• Proof by induction
• Applications in geometry

General Case

$$\frac{\sum a_i}{n} \geq \sqrt[n]{\prod a_i}$$
Key Points:
• Holds for any n positive numbers
• Equality when all equal
• Used in Jensen's inequality
• Applications in information theory

Proof Method

(√a - √b)² ≥ 0
Standard Proof:
• Expand: a + b - 2√(ab) ≥ 0
• Divide by 2: (a+b)/2 ≥ √(ab)
• Uses square non-negativity
• Can be extended to n variables
Mathematical Foundation

The AM-GM inequality is one of the most important inequalities in mathematics:

  • Convex Functions: AM-GM follows from convexity of exponential function
  • Jensen's Inequality: AM-GM is a special case of Jensen's inequality
  • Optimization: Used to find maximum/minimum values of functions
  • Probability: Applications in expected values and variance
  • Information Theory: Used in entropy inequalities
  • Geometry: Applications in triangle inequalities and areas
  • Analysis: Used in proving convergence and limits
  • Algebra: Applications in polynomial inequalities

The inequality becomes equality when all variables are equal, and strict inequality otherwise.

Practice Problems

Problem 1

Verify the AM-GM inequality for a = 2 and b = 8.

Solution:

AM = (2 + 8)/2 = 5

GM = √(2 × 8) = √16 = 4

5 ≥ 4 ✓

Result: Inequality holds

Problem 2

For what values of x does equality hold in AM-GM for positive numbers a and b?

Solution:

Equality holds when a = b

This is the only case where AM = GM

For all other positive a ≠ b, AM > GM

Result: Equality when a = b

Problem 3

Use AM-GM to prove that for positive x, y: (x + y)/2 ≥ √(xy)

Solution:

Consider (√x - √y)² ≥ 0

Expand: x + y - 2√(xy) ≥ 0

Divide by 2: (x + y)/2 ≥ √(xy)

Result: Proof complete

Key Takeaways
  • AM-GM states that arithmetic mean ≥ geometric mean for positive numbers
  • Equality holds if and only if all numbers are equal
  • The inequality is strict (>) unless all variables are equal
  • AM-GM can be proven using the square of differences
  • It generalizes to any number of positive variables
  • Used extensively in optimization and inequality proofs
  • Applications in probability, statistics, and analysis
  • Can be used to find maximum products with fixed sums
