Square Inequalities
Square inequalities involve relationships between squared quantities. Understanding these properties is crucial for solving inequalities and working with absolute values.
Strict Inequality: $$a^2 > b^2 \iff |a| > |b|$$
$$\text{(for } a > 0, b > 0: a^2 > b^2 \iff a > b\text{)}$$
Non-strict Inequality: $$a^2 \geq b^2 \iff |a| \geq |b|$$
$$\text{(for } a > 0, b > 0: a^2 \geq b^2 \iff a \geq b\text{)}$$
Equality: $$a^2 = b^2 \iff |a| = |b|$$
For positive real numbers, squaring preserves the inequality direction. For general real numbers, we must consider absolute values since squares are always non-negative.

Inequality Properties

Positive Numbers

a > 0, b > 0
Rules:
• a² > b² ⇔ a > b
• a² ≥ b² ⇔ a ≥ b
• a² < b² ⇔ a < b
• Squaring preserves order

General Real Numbers

a, b ∈ ℝ
Rules:
• a² > b² ⇔ |a| > |b|
• a² ≥ b² ⇔ |a| ≥ |b|
• a² = b² ⇔ |a| = |b|
• Must use absolute values

Solving Inequalities

x² > k
Steps:
• Take square root of both sides
• Consider positive and negative roots
• x > √k and x < -√k
• Domain restrictions apply

Absolute Value

|x| < a
Equivalent to:
• -a < x < a
• |x| ≤ a ⇔ -a ≤ x ≤ a
• |x| > a ⇔ x > a or x < -a
• Useful for square inequalities
Mathematical Foundation

Square inequalities are based on the fundamental properties of squares and absolute values:

  • Monotonicity: Squaring is increasing for positive numbers
  • Absolute Values: |a|² = a² always holds
  • Order Preservation: For a,b > 0, a > b implies a² > b²
  • Square Roots: √(a²) = |a|, not just a
  • Inequality Direction: Squaring can change inequality direction for negative numbers
  • Critical Points: Squares have minimum at zero
  • Parabolic Behavior: Quadratic functions are always ≥ minimum value
  • Completing Square: Method for solving quadratic inequalities

These properties are essential for solving quadratic inequalities and understanding function behavior.

\begin{aligned} \text{Solving } x^2 > a: \\ \text{Case 1: } a \geq 0 \to x > \sqrt{a} \text{ or } x < -\sqrt{a} \\ \text{Case 2: } a < 0 \to \text{all real numbers} \end{aligned}

Practice Problems

Problem 1

Compare 7² and 6². Which is larger and why?

Solution:

Since both 7 and 6 are positive: 7 > 6

Therefore: 7² > 6² (49 > 36)

Result: 7² is larger than 6²

Problem 2

Solve the inequality x² > 9 and express the solution in interval notation.

Solution:

x² > 9

Take square root: |x| > 3

This means: x > 3 or x < -3

Result: (-∞, -3) ∪ (3, ∞)

Problem 3

Determine if (-2)² > (-3)² is true or false, and explain why.

Solution:

(-2)² = 4, (-3)² = 9

4 > 9 is false

Since squares are always non-negative, we compare absolute values: |-2| = 2, |-3| = 3, so 2 < 3

Result: False, because | -2 | < | -3 |

Key Takeaways
  • For positive numbers: a² > b² ⇔ a > b (order preserved)
  • For general real numbers: a² > b² ⇔ |a| > |b| (use absolute values)
  • Squaring both sides of an inequality may change the direction if numbers can be negative
  • To solve x² > k, take square roots and consider both positive and negative cases
  • x² ≥ 0 for all real x, with equality only when x = 0
  • Absolute value inequalities |x| < a are equivalent to -a < x < a
  • When solving quadratic inequalities, consider the parabola's behavior
  • Domain restrictions are important when taking square roots
