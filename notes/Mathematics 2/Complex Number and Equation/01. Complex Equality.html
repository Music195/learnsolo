Equality of Complex Numbers
Two complex numbers are equal if and only if their real parts are equal and their imaginary parts are equal. This fundamental property allows us to solve equations involving complex numbers.
Complex Equality: $$a + bi = c + di \iff a = c \text{ and } b = d$$
Zero Complex Number: $$a + bi = 0 \iff a = 0 \text{ and } b = 0$$
Conjugate Equality: $$z = \bar{z} \iff \Im(z) = 0$$

Mathematical Foundation

A complex number is defined as \(z = a + bi\), where \(a\) and \(b\) are real numbers, and \(i = \sqrt{-1}\).

$$z_1 = a + bi, \quad z_2 = c + di$$

Two complex numbers are equal if and only if their real parts are equal and their imaginary parts are equal:

$$a + bi = c + di \iff a = c \text{ and } b = d$$

This is because complex numbers can be represented as ordered pairs (a, b) in the complex plane, and equality of ordered pairs requires both components to be equal.

The complex number zero is unique: \(0 = 0 + 0i\), and it's the only complex number that equals its own negative.

Practice Problems

Problem 1

Solve for x and y: (2x + 3) + (x + 2y)i = 7 + 5i

Solution:

Real parts: 2x + 3 = 7 ⇒ 2x = 4 ⇒ x = 2

Imaginary parts: x + 2y = 5 ⇒ 2 + 2y = 5 ⇒ 2y = 3 ⇒ y = 3/2

Therefore: x = 2, y = 1.5

Problem 2

Find the complex number z such that z + \bar{z} = 4 + 6i and z - \bar{z} = 2 - 8i

Solution:

Let z = a + bi, then \bar{z} = a - bi

z + \bar{z} = (a + bi) + (a - bi) = 2a = 4 + 6i ⇒ 2a = 4 ⇒ a = 2

z - \bar{z} = (a + bi) - (a - bi) = 2bi = 2 - 8i ⇒ 2b = -8 ⇒ b = -4

Therefore: z = 2 - 4i

Problem 3

Determine if the equation (z + 1)^2 = z^2 + 2z + 1 has complex solutions.

Solution:

Expand both sides: z² + 2z + 1 = z² + 2z + 1

The equation simplifies to 0 = 0, which is always true.

Therefore, every complex number z is a solution.

Key Takeaways
  • Two complex numbers are equal iff their real parts are equal and imaginary parts are equal
  • Complex numbers can be treated as ordered pairs (a, b) in the complex plane
  • The complex number 0 is unique: 0 + 0i
  • A complex number equals its conjugate iff it's purely real
  • Complex equality allows solving systems of equations with complex coefficients
