Quadratic Formula
The quadratic formula provides the solutions to any quadratic equation ax² + bx + c = 0. With complex numbers, it can find solutions even when the discriminant is negative.
Standard Form: $$x = \frac{-b \pm \sqrt{b^2 - 4ac}}{2a}$$
Alternative Form: $$x = \frac{-b' \pm \sqrt{b'^2 - ac}}{a}$$
Discriminant: $$D = b^2 - 4ac$$
Discriminant and Solution Types
D > 0
Two distinct real roots
Parabola crosses x-axis twice
D = 0
One repeated real root
Parabola touches x-axis
D < 0
Two complex conjugate roots
Parabola doesn't cross x-axis
Complex Roots
Form: α ± βi
Conjugate pairs

Mathematical Foundation

The quadratic formula is derived by completing the square for the general quadratic equation ax² + bx + c = 0:

$$ax^2 + bx + c = 0$$ $$x^2 + \frac{b}{a}x + \frac{c}{a} = 0$$ $$x^2 + \frac{b}{a}x = -\frac{c}{a}$$ $$\left(x + \frac{b}{2a}\right)^2 - \left(\frac{b}{2a}\right)^2 = -\frac{c}{a}$$ $$\left(x + \frac{b}{2a}\right)^2 = \frac{b^2 - 4ac}{4a^2}$$ $$x + \frac{b}{2a} = \pm \frac{\sqrt{b^2 - 4ac}}{2a}$$ $$x = \frac{-b \pm \sqrt{b^2 - 4ac}}{2a}$$

The discriminant D = b² - 4ac determines the nature of the roots:

  • D > 0: Two distinct real roots
  • D = 0: One repeated real root
  • D < 0: Two complex conjugate roots

Practice Problems

Problem 1

Solve 2x² - 7x + 3 = 0 using the quadratic formula.

Solution:

a = 2, b = -7, c = 3

D = (-7)² - 4(2)(3) = 49 - 24 = 25

x = [7 ± √25]/(2×2) = [7 ± 5]/4

x = 12/4 = 3 or x = 2/4 = 0.5

Problem 2

Find the complex roots of x² + 4x + 13 = 0.

Solution:

a = 1, b = 4, c = 13

D = 4² - 4(1)(13) = 16 - 52 = -36

x = [-4 ± √(-36)]/(2×1) = [-4 ± 6i]/2

x = -2 + 3i or x = -2 - 3i

Problem 3

Determine the nature of roots for 3x² + 6x + 3 = 0 without solving.

Solution:

a = 3, b = 6, c = 3

D = 6² - 4(3)(3) = 36 - 36 = 0

D = 0, so one repeated real root

Key Takeaways
  • Quadratic formula: x = [-b ± √(b² - 4ac)]/(2a)
  • Discriminant D = b² - 4ac determines root types
  • D > 0: two distinct real roots
  • D = 0: one repeated real root
  • D < 0: two complex conjugate roots
  • Complex roots always come in conjugate pairs
