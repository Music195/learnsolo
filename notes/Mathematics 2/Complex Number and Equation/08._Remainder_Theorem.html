Remainder Theorem
The Remainder Theorem provides a powerful shortcut for finding remainders when polynomials are divided by linear factors.
Remainder Theorem: $$P(x) \div (x - k) \implies \text{Remainder} = P(k)$$
Factor Theorem: $$P(k) = 0 \implies (x - k) \text{ is a factor of } P(x)$$
Division Algorithm: $$P(x) = (x - k)Q(x) + R$$

Remainder Theorem

P(x) = (x - k)Q(x) + R
When dividing P(x) by (x - k), the remainder R equals P(k).

This works because P(k) = (k - k)Q(k) + R = R

Factor Theorem

P(k) = 0 ⇒ (x - k) divides P(x)
If k is a root of P(x), then (x - k) is a factor of P(x).

Conversely, if (x - k) divides P(x), then k is a root.

Multiple Roots

P(k) = P'(k) = 0 ⇒ (x - k)² divides P(x)
If k is a repeated root, then (x - k) appears multiple times as a factor.

The multiplicity equals the number of times (x - k) divides P(x).

Mathematical Foundation

The Remainder Theorem follows directly from the Division Algorithm for polynomials:

$$\text{If } P(x) = (x - k)Q(x) + R, \text{ where } R \text{ is a constant}$$

Substituting x = k into both sides:

$$\begin{align} P(k) &= (k - k)Q(k) + R \\ P(k) &= 0 \cdot Q(k) + R \\ P(k) &= R \end{align}$$

Therefore, the remainder when P(x) is divided by (x - k) is simply P(k).

The Factor Theorem is a direct consequence: if P(k) = 0, then R = 0, so (x - k) divides P(x) exactly.

Practice Problems

Problem 1

Find the remainder when x³ - 3x² + 4x - 2 is divided by x - 1.

Solution:

By Remainder Theorem, remainder = P(1)

P(1) = 1³ - 3(1)² + 4(1) - 2 = 1 - 3 + 4 - 2 = 0

Remainder = 0

Problem 2

Determine if x + 1 is a factor of x³ + 2x² - x - 2.

Solution:

Check P(-1): (-1)³ + 2(-1)² - (-1) - 2 = -1 + 2 + 1 - 2 = 0

Since P(-1) = 0, x + 1 is a factor.

Problem 3

Find the value of k such that x - 2 is a factor of x³ - kx² + 3x - 6.

Solution:

Since x - 2 is a factor, P(2) = 0

2³ - k(2)² + 3(2) - 6 = 0

8 - 4k + 6 - 6 = 0

8 - 4k = 0 ⇒ k = 2

Key Takeaways
  • Remainder when P(x) ÷ (x - k) equals P(k)
  • If P(k) = 0, then (x - k) is a factor of P(x)
  • Works for any polynomial and any complex number k
  • Much faster than polynomial long division
  • Essential for finding rational roots of polynomial equations
  • Foundation for synthetic division and polynomial factorization
