Square Root of Negative Numbers
The square root of a negative number introduces the imaginary unit i, where i² = -1. This fundamental concept extends the real number system to the complex numbers.

Square Root of Negative: $$\sqrt{-a} = \sqrt{a}i \quad (a > 0)$$
Two Square Roots: $$\pm \sqrt{-a} = \pm \sqrt{a}i$$

Mathematical Foundation

The square root of a negative number cannot exist in the real number system because the square of any real number is non-negative. To solve this problem, we introduce the imaginary unit i:

$$i = \sqrt{-1}, \quad i^2 = -1$$

Using this definition, we can find the square root of any negative number:

$$\sqrt{-a} = \sqrt{a \cdot (-1)} = \sqrt{a} \cdot \sqrt{-1} = \sqrt{a}i \quad (a > 0)$$

Since squaring can produce both positive and negative results, there are actually two square roots:

$$\pm \sqrt{-a} = \pm \sqrt{a}i$$

The imaginary unit has interesting powers that cycle every 4 steps:

$$i^1 = i, \quad i^2 = -1, \quad i^3 = -i, \quad i^4 = 1, \quad i^5 = i, \quad \dots$$

Practice Problems

Problem 1

Calculate √(-16) and express all square roots.

Solution:

√(-16) = √(16 × (-1)) = √16 × √(-1) = 4i

All square roots: ±4i

Problem 2

Simplify i⁸ and i²⁵.

Solution:

i⁸ = (i⁴)² = 1² = 1

i²⁵ = i^(24+1) = (i⁴)^6 × i = 1^6 × i = i

Problem 3

Solve x² + 4 = 0 for complex x.

Solution:

x² = -4

x = ±√(-4) = ±2i

Key Takeaways
  • The imaginary unit i is defined as i = √(-1) with i² = -1
  • Square root of negative numbers: √(-a) = √a × i for a > 0
  • There are always two square roots of any non-zero complex number
  • Powers of i cycle every 4: i, -1, -i, 1, i, -1, -i, 1, ...
  • Complex numbers extend the real number system to solve previously impossible equations
