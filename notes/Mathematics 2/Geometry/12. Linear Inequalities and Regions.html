Linear inequalities create half-planes divided by their boundary lines. The inequality symbol determines whether the boundary is included and which half-plane contains the solutions.
Types of Linear Inequalities

Strict Upper Region

Points above the line (excluded)
$$y > mx + b$$
Region: All points above the line
Boundary: Dashed line (not included)
Test point: (0,0) determines shading side
Half-plane: Open region

Strict Lower Region

Points below the line (excluded)
$$y < mx + b$$
Region: All points below the line
Boundary: Dashed line (not included)
Test point: (0,0) determines shading side
Half-plane: Open region

Inclusive Region

Points above including the line
$$y \geq mx + b$$
Region: All points above or on the line
Boundary: Solid line (included)
Test point: (0,0) determines shading side

Mathematical Foundation

Linear inequalities represent regions in the coordinate plane bounded by lines:

  • Inequality symbols: > (greater than), < (less than), ≥ (greater than or equal), ≤ (less than or equal)
  • Half-planes: Each linear inequality divides the plane into two half-planes
  • Boundary line: ax + by + c = 0 or y = mx + b
  • Test point method: Substitute a point not on the boundary to determine which half-plane satisfies the inequality
  • Strict vs inclusive: > and < exclude boundary (open regions), ≥ and ≤ include boundary (closed regions)
  • Graphing procedure: 1) Graph boundary line, 2) Choose test point, 3) Shade appropriate region
  • Vertical boundaries: x > a, x < a, x ≥ a, x ≤ a create vertical half-planes
  • System of inequalities: Intersection of half-planes forms polygonal regions
\begin{aligned} \text{General form:} \\ ax + by + c &> 0 \\ ax + by + c &< 0 \\ ax + by + c &\geq 0 \\ ax + by + c &\leq 0 \\ \\ \text{Slope-intercept form:} \\ y &> mx + b \\ y &< mx + b \\ y &\geq mx + b \\ y &\leq mx + b \end{aligned}

Practice Problems

Problem 1

Graph the inequality y ≥ x + 2. Describe the solution region.

Solution:

Boundary line: y = x + 2 (solid line)

Test point (0,0): 0 ≥ 0 + 2 ⇒ 0 ≥ 2? False

Shade region below the line

Result: Region below y = x + 2, including the line

Problem 2

Graph 2x - y < 4 using the test point method.

Solution:

Convert to y > 2x - 4 (dashed line)

Test point (0,0): 0 > 2(0) - 4 ⇒ 0 > -4? True

Shade region above the line

Result: Region above y = 2x - 4, excluding the line

Problem 3

Determine which region satisfies x + 3y ≥ 6.

Solution:

Convert to y ≥ (-1/3)x + 2 (solid line)

Test point (0,0): 0 ≥ (-1/3)(0) + 2 ⇒ 0 ≥ 2? False

Shade region above the line

Result: Region above y = (-1/3)x + 2, including the line

Key Takeaways
  • Linear inequalities divide the plane into half-planes with boundary lines
  • Strict inequalities (>, <) use dashed boundaries and exclude the line
  • Inclusive inequalities (≥, ≤) use solid boundaries and include the line
  • Test point method: substitute (0,0) or any point not on boundary to determine shading
  • If test point satisfies inequality, shade its side; otherwise shade opposite side
  • Convert general form ax + by + c > 0 to slope-intercept form y > mx + b for graphing
  • Solution region is always a half-plane (infinite region)
  • Essential for linear programming, optimization, and system constraints
