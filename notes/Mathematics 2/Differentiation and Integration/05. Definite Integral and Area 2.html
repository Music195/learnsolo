Definite Integral and Area (Case 2)
When a function lies below the x-axis, the definite integral represents the negative of the area between the curve and the x-axis. This case requires special consideration for area calculations.
Definite Integral: $$\int_a^b f(x)\,dx = F(b) - F(a)$$
Area (Case 2): $$S = \int_a^b \color{#f57c00}{-f(x)}\,dx$$
Condition: $$f(x) \leq 0 \quad \forall x \in [a,b]$$

Case 1: f(x) ≥ 0

S = ∫ f(x) dx
When the function is above the x-axis, the definite integral directly gives the area between the curve and the x-axis.

Case 2: f(x) ≤ 0

S = ∫ -f(x) dx
When the function is below the x-axis, the definite integral is negative. To get the actual area, we integrate the absolute value or -f(x).

Case 3: Mixed

S = ∫ |f(x)| dx
When the function crosses the x-axis, we need to split the integral at the roots and handle each region separately.

Mathematical Foundation

When a function lies entirely below the x-axis (f(x) ≤ 0 for all x in [a,b]), the definite integral ∫ f(x) dx will be negative. However, the area between the curve and the x-axis is always positive.

$$\int_a^b f(x)\,dx = F(b) - F(a) < 0 \quad (\text{since } f(x) \leq 0)$$

To find the actual area, we integrate the function -f(x), which is always non-negative:

$$S = \int_a^b -f(x)\,dx = -\int_a^b f(x)\,dx > 0$$

This is equivalent to integrating the absolute value of f(x) when f(x) ≤ 0:

$$S = \int_a^b |f(x)|\,dx = \int_a^b -f(x)\,dx$$

Practice Problems

Problem 1

Find the area bounded by the curve y = -x² + 4, the x-axis, and the lines x = 0 and x = 2.

Solution:

First, find the definite integral: ∫₀² (-x² + 4) dx

= [-x³/3 + 4x]₀² = (-8/3 + 8) - (0 + 0) = (-8/3 + 24/3) = 16/3

Since the function is positive in [0,2], this gives the area directly: S = 16/3

Note: This is actually Case 1, but shows the concept.

Problem 2

Find the area bounded by the curve y = -(x-1)² + 1, the x-axis, and the lines x = 0 and x = 2.

Solution:

First, find the definite integral: ∫₀² [-(x-1)² + 1] dx

= ∫₀² [-((x-1)² - 1)] dx = ∫₀² [-x² + 2x - 1 + 1] dx = ∫₀² (-x² + 2x) dx

= [-x³/3 + x²]₀² = (-8/3 + 4) - (0 + 0) = (-8/3 + 12/3) = 4/3

Since the function is negative, the area is: S = ∫₀² -f(x) dx = -∫₀² f(x) dx = -4/3 = 4/3

Problem 3

Find the area bounded by the curve y = -sin(x), the x-axis, and the lines x = 0 and x = π.

Solution:

Definite integral: ∫₀^π -sin(x) dx = [cos(x)]₀^π = cos(π) - cos(0) = (-1) - 1 = -2

Area: S = ∫₀^π -(-sin(x)) dx = ∫₀^π sin(x) dx = [-cos(x)]₀^π = -cos(π) - (-cos(0)) = -(-1) - (-1) = 1 + 1 = 2

Alternatively: S = -∫₀^π -sin(x) dx = -(-2) = 2

Key Takeaways
  • Definite integral ∫ f(x) dx gives signed area; can be negative
  • For f(x) ≤ 0, area S = ∫ -f(x) dx = -∫ f(x) dx
  • Area is always non-negative; definite integral can be negative
  • Geometrically represents region between curve and x-axis
  • Fundamental Theorem: ∫ f(x) dx = F(b) - F(a)
  • Case 2 applies when function is entirely below x-axis
