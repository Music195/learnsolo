Trigonometric Properties with Graphs

Mathematical Foundation

Trigonometric properties with graphs explore how trigonometric functions transform when angles are shifted or reflected. These properties are fundamental to understanding periodic functions and their behavior on the unit circle.

Key Properties:
  • Periodicity: $f(\theta + T) = f(\theta)$ where T is the period
  • Angle Shifts: $\sin(\theta + \pi) = -\sin\theta$, $\cos(\theta + \pi) = -\cos\theta$
  • 90° Shifts: $\sin(\theta + \frac{\pi}{2}) = \cos\theta$, $\cos(\theta + \frac{\pi}{2}) = -\sin\theta$
  • Even/Odd Functions: $\sin(-\theta) = -\sin\theta$, $\cos(-\theta) = \cos\theta$

Worked Example 1: 180° Angle Shift

If θ = π/4, find sin(θ + π) and cos(θ + π).

Solution:

θ = π/4 = 45°

sin(π/4 + π) = sin(5π/4) = sin(225°) = -sin(45°) = -√2/2

cos(π/4 + π) = cos(5π/4) = cos(225°) = -cos(45°) = -√2/2

tan(π/4 + π) = tan(5π/4) = tan(225°) = tan(45°) = 1

Worked Example 2: 90° Angle Shift

If θ = π/3, find sin(θ + π/2) and cos(θ + π/2).

Solution:

θ = π/3 = 60°

sin(π/3 + π/2) = sin(5π/6) = sin(150°) = cos(60°) = 1/2

cos(π/3 + π/2) = cos(5π/6) = cos(150°) = -sin(60°) = -√3/2

Worked Example 3: Negative Angle

If θ = π/6, find sin(-θ), cos(-θ), and tan(-θ).

Solution:

θ = π/6 = 30°

sin(-π/6) = sin(-30°) = -sin(30°) = -1/2

cos(-π/6) = cos(-30°) = cos(30°) = √3/2

tan(-π/6) = tan(-30°) = -tan(30°) = -1/√3

Visual Representations

1. Angle Shift by π (180°)
Properties:
  • $$\sin(\theta + \pi) = -\sin\theta$$
  • $$\cos(\theta + \pi) = -\cos\theta$$
  • $$\tan(\theta + \pi) = \tan\theta$$

When an angle is increased by π, the terminal point moves to the opposite side of the origin.

Point P($a$, $b$) at angle $\theta$ becomes point Q($-a$, $-b$) at angle $\theta + \pi$

2. Angle Shift by π/2 (90°)
Properties:
  • $$\sin\left(\theta + \frac{\pi}{2}\right) = \cos\theta$$
  • $$\cos\left(\theta + \frac{\pi}{2}\right) = -\sin\theta$$
  • $$\tan\left(\theta + \frac{\pi}{2}\right) = -\frac{1}{\tan\theta}$$

When an angle is increased by π/2, the terminal point rotates 90° counterclockwise.

Point P($a$, $b$) at angle $\theta$ becomes point R($-b$, $a$) at angle $\theta + \frac{\pi}{2}$

3. Negative Angle Properties
Properties:
  • $$\sin(-\theta) = -\sin\theta$$ (odd function)
  • $$\cos(-\theta) = \cos\theta$$ (even function)
  • $$\tan(-\theta) = -\tan\theta$$ (odd function)

Negative angles rotate clockwise instead of counterclockwise.

Point P($a$, $b$) at angle $\theta$ and point S($a$, $-b$) at angle $-\theta$

Practice Problems

Problem 1: If θ = π/3, find the exact values of sin(θ + π), cos(θ + π), and tan(θ + π).
Solution:
θ = π/3 = 60°
sin(π/3 + π) = sin(4π/3) = sin(240°) = -sin(60°) = -√3/2
cos(π/3 + π) = cos(4π/3) = cos(240°) = -cos(60°) = -1/2
tan(π/3 + π) = tan(4π/3) = tan(240°) = tan(60°) = √3
Problem 2: If θ = π/4, find the exact values of sin(θ + π/2), cos(θ + π/2), and tan(θ + π/2).
Solution:
θ = π/4 = 45°
sin(π/4 + π/2) = sin(3π/4) = sin(135°) = cos(45°) = √2/2
cos(π/4 + π/2) = cos(3π/4) = cos(135°) = -sin(45°) = -√2/2
tan(π/4 + π/2) = tan(3π/4) = tan(135°) = -1/tan(45°) = -1
Problem 3: If θ = 2π/3, find the exact values of sin(-θ), cos(-θ), and tan(-θ).
Solution:
θ = 2π/3 = 120°
sin(-2π/3) = sin(-120°) = -sin(120°) = -√3/2
cos(-2π/3) = cos(-120°) = cos(120°) = -1/2
tan(-2π/3) = tan(-120°) = -tan(120°) = -(-√3) = √3

Key Takeaways

  • 180° Shifts: Adding π to an angle moves the point to the opposite side of the unit circle, negating sine and cosine while preserving tangent.
  • 90° Shifts: Adding π/2 rotates the point 90° counterclockwise, swapping sine and cosine with appropriate sign changes.
  • Negative Angles: Negative angles rotate clockwise and demonstrate the even/odd properties of trigonometric functions.
  • Unit Circle Symmetry: These transformations reveal the fundamental symmetries of the unit circle and trigonometric functions.
  • Periodicity: Understanding these shifts helps comprehend the periodic nature of trigonometric functions.
