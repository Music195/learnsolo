Trigonometric Properties
Trigonometric functions exhibit fundamental properties including periodicity, symmetry, and special angle relationships that are essential for solving trigonometric equations and understanding function behavior.
Periodicity:
f(x + T) = f(x)
Even Function:
f(-x) = f(x)
Odd Function:
f(-x) = -f(x)
Sine and tangent are odd functions, cosine is even. All trigonometric functions are periodic with sine and cosine having period 2π, tangent having period π.

Fundamental Properties

Periodicity

f(x + T) = f(x)
  • Sine & Cosine: Period = 2π
  • Tangent: Period = π
  • Cosecant & Secant: Period = 2π
  • Cotangent: Period = π
  • Fundamental Period: Smallest positive T
  • Applications: Wave functions, oscillations

Symmetry Properties

\begin{aligned} \text{Even: } f(-x) &= f(x) \\ \text{Odd: } f(-x) &= -f(x) \end{aligned}
  • Even Functions: cos x, sec x
  • Odd Functions: sin x, tan x, csc x, cot x
  • Geometric Meaning: Even functions are symmetric about y-axis
  • Geometric Meaning: Odd functions are symmetric about origin
  • Integration: ∫f(x) dx from -a to a = 2∫f(x) dx from 0 to a (even)
  • Integration: ∫f(x) dx from -a to a = 0 (odd)

Complementary Angles

θ + φ = π/2
  • sin(π/2 - x) = cos x
  • cos(π/2 - x) = sin x
  • tan(π/2 - x) = cot x
  • cot(π/2 - x) = tan x
  • sec(π/2 - x) = csc x
  • csc(π/2 - x) = sec x
  • Applications: Right triangle relationships

Supplementary Angles

θ + φ = π
  • sin(π - x) = sin x
  • cos(π - x) = -cos x
  • tan(π - x) = -tan x
  • cot(π - x) = -cot x
  • sec(π - x) = -sec x
  • csc(π - x) = csc x
  • Applications: Quadrant relationships
Mathematical Foundation

Trigonometric properties are derived from the unit circle and fundamental geometric relationships:

  • Unit Circle Definition: Trigonometric functions are defined using coordinates on the unit circle
  • Periodicity: Functions repeat their values at regular intervals due to the circular nature
  • Symmetry: Even functions are symmetric about the y-axis, odd functions about the origin
  • Angle Relationships: Complementary angles sum to 90°, supplementary angles sum to 180°
  • Co-function Identities: sin(90° - x) = cos x, cos(90° - x) = sin x
  • Negative Angle Identities: sin(-x) = -sin x, cos(-x) = cos x, tan(-x) = -tan x
  • Period Shift Identities: sin(x + 2π) = sin x, cos(x + 2π) = cos x, tan(x + π) = tan x
  • Reflection Properties: Functions reflect over axes according to their symmetry
  • Fundamental Domains: Smallest interval containing one full period of the function
  • Group Properties: Trigonometric functions form abelian groups under addition modulo period
\begin{aligned} \text{Periodicity:} \quad f(x + nT) &= f(x) \quad \text{for integer } n \\ \text{Even function:} \quad f(-x) &= f(x) \\ \text{Odd function:} \quad f(-x) &= -f(x) \\ \text{Complementary:} \quad \sin\left(\frac{\pi}{2} - x\right) &= \cos x \\ \text{Supplementary:} \quad \sin(\pi - x) &= \sin x \end{aligned}

Practice Problems

Problem 1

Verify the periodicity of the tangent function by showing that tan(x + π) = tan x.

Solution:

Using the angle addition formula for tangent:

tan(x + π) = (tan x + tan π)/(1 - tan x tan π)

Since tan π = 0:

tan(x + π) = (tan x + 0)/(1 - tan x * 0) = tan x / 1 = tan x

Result: tan(x + π) = tan x ✓

Problem 2

Determine whether the function f(x) = cos²x - sin²x is even, odd, or neither.

Solution:

Test f(-x): cos²(-x) - sin²(-x) = cos²x - sin²x = f(x)

Since f(-x) = f(x), the function is even.

Result: Even function

Problem 3

Use complementary angle properties to simplify: sin(π/2 - α) * cos α + cos(π/2 - α) * sin α

Solution:

sin(π/2 - α) = cos α

cos(π/2 - α) = sin α

So: cos α * cos α + sin α * sin α = cos²α + sin²α = 1

Result: 1

Key Takeaways
  • Sine and cosine have period 2π, tangent and cotangent have period π
  • Cosine and secant are even functions, others are odd
  • Complementary angles: sin(90° - x) = cos x, cos(90° - x) = sin x
  • Supplementary angles: sin(180° - x) = sin x, cos(180° - x) = -cos x
  • Periodicity enables solving equations by shifting by multiples of the period
  • Symmetry properties simplify integration over symmetric intervals
  • Even functions are symmetric about y-axis, odd about origin
  • Angle relationships reduce complex expressions to simpler forms
  • Properties are fundamental to trigonometric identities and equations
  • Understanding properties enables efficient trigonometric manipulations
