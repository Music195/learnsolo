Sum of Arithmetic Sequence

Core Concept: Sum of Arithmetic Series

The sum of an arithmetic series is the total of all terms in the sequence. There are two main formulas to calculate this sum, depending on what information is available about the sequence.

Sum Formula 1: Using First and Last Terms

$S_n = \frac{n}{2}(a + l)$

Where:
$S_n$ = sum of first n terms
$n$ = number of terms
$a$ = first term
$l$ = last term

Sum Formula 2: Using First Term and Common Difference

$S_n = \frac{n}{2}[2a + (n-1)d]$

Where:
$S_n$ = sum of first n terms
$n$ = number of terms
$a$ = first term
$d$ = common difference

The sum of an arithmetic series can be visualized as the area of a trapezoid formed by the sequence terms. The first formula represents the average of the first and last terms multiplied by the number of terms.

Derivation of Sum Formulas

Method 1: Pairing Terms

Write the series forward and backward, then add them:

$S_n = a + (a+d) + (a+2d) + \dots + (l-d) + l$
$S_n = l + (l-d) + (l-2d) + \dots + (a+d) + a$
$2S_n = (a+l) + (a+l) + \dots + (a+l)$ (n times)
$2S_n = n(a+l)$
$S_n = \frac{n}{2}(a+l)$
Method 2: Using General Term

Substitute the last term $l = a + (n-1)d$ into the first formula:

$S_n = \frac{n}{2}[a + a + (n-1)d]$
$S_n = \frac{n}{2}[2a + (n-1)d]$

Worked Examples

1

Find the sum of the first 20 terms of the arithmetic sequence: 3, 7, 11, 15, ...

a = 3, d = 7-3 = 4, n = 20
S₂₀ = n/2[2a + (n-1)d]
S₂₀ = 20/2[2×3 + (20-1)×4]
S₂₀ = 10[6 + 19×4]
S₂₀ = 10[6 + 76] = 10×82 = 820
2

The sum of the first 15 terms of an arithmetic sequence is 225. The first term is 5. Find the common difference.

S₁₅ = 225, a = 5, n = 15
225 = 15/2[2×5 + (15-1)d]
225 = 15/2[10 + 14d]
225 = (15/2)×(10 + 14d)
225 × 2/15 = 10 + 14d
30 = 10 + 14d
20 = 14d ⇒ d = 20/14 = 10/7
3

Find the sum of all odd numbers from 1 to 99.

Sequence: 1, 3, 5, ..., 99
a = 1, l = 99, d = 2
Number of terms: n = (99-1)/2 + 1 = 50
S₅₀ = n/2(a + l) = 50/2(1 + 99) = 25×100 = 2500

Key Takeaways

  • Use $S_n = \frac{n}{2}(a + l)$ when you know the first and last terms
  • Use $S_n = \frac{n}{2}[2a + (n-1)d]$ when you know a and d
  • The sum represents the average of first and last terms multiplied by number of terms
  • Arithmetic series sums can be derived by pairing terms from opposite ends
  • Both formulas are equivalent and can be used interchangeably
