General Term of an Arithmetic Sequence

Core Concept: Arithmetic Sequence

An arithmetic sequence is a sequence where each term after the first is obtained by adding a constant difference (common difference) to the preceding term. The general term formula allows us to find any term in the sequence without calculating all previous terms.

General Term Formula

$a_n = a + (n-1)d$

Where:
$a_n$ = nth term
$a$ = first term
$d$ = common difference
$n$ = term position

The nth term is obtained by adding the common difference $d$ to the first term $a$, $(n-1)$ times.

Worked Examples

1

Find the 10th term of the arithmetic sequence where the first term is 5 and the common difference is 4.

a = 5, d = 4, n = 10
a₁₀ = a + (n-1)d
a₁₀ = 5 + (10-1)×4
a₁₀ = 5 + 9×4
a₁₀ = 5 + 36 = 41
2

In an arithmetic sequence, the 5th term is 18 and the 12th term is 45. Find the first term and common difference.

a₅ = a + 4d = 18
a₁₂ = a + 11d = 45
Subtract first equation from second:
(a + 11d) - (a + 4d) = 45 - 18
7d = 27 ⇒ d = 27÷7 = 3.857...
From a + 4d = 18:
a + 4×3.857 = 18
a + 15.428 = 18
a = 18 - 15.428 = 2.572
3

Find the term number of 50 in the sequence: 3, 7, 11, 15, ...

a = 3, d = 7-3 = 4
aₙ = 3 + (n-1)×4 = 50
3 + 4(n-1) = 50
4(n-1) = 47
n-1 = 47÷4 = 11.75
n = 12.75 (not integer)
50 is not a term in this sequence

Key Takeaways

  • The general term formula $a_n = a + (n-1)d$ finds any term directly
  • The first term is when n=1: $a_1 = a$
  • Each term increases by the common difference d
  • The formula can be rearranged to find a, d, or n
  • Arithmetic sequences have a linear pattern
