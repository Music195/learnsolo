Quotient Rule for Derivatives
The Quotient Rule:
$$\dfrac{d}{dx}\left[\dfrac{f(x)}{g(x)}\right] = \dfrac{f'(x)g(x) - f(x)g'(x)}{[g(x)]^2}$$
"Low d-high minus high d-low, over the square of what's below"
1. Derivation from Product Rule
Start with: $\dfrac{f(x)}{g(x)} = f(x) \cdot [g(x)]^{-1}$

Apply product rule: $\dfrac{d}{dx}[f(x) \cdot u(x)] = f'(x)u(x) + f(x)u'(x)$
where $u(x) = [g(x)]^{-1}$

So: $u'(x) = -\dfrac{g'(x)}{[g(x)]^2}$

Therefore: $\dfrac{d}{dx}\left[\dfrac{f(x)}{g(x)}\right] = f'(x) \cdot \dfrac{1}{g(x)} + f(x) \cdot \left(-\dfrac{g'(x)}{[g(x)]^2}\right)$

$= \dfrac{f'(x)}{g(x)} - \dfrac{f(x)g'(x)}{[g(x)]^2} = \dfrac{f'(x)g(x) - f(x)g'(x)}{[g(x)]^2}$
2. Special Cases
Derivative of 1/g(x):
$\dfrac{d}{dx}\left[\dfrac{1}{g(x)}\right] = -\dfrac{g'(x)}{[g(x)]^2}$

Derivative of c/g(x):
$\dfrac{d}{dx}\left[\dfrac{c}{g(x)}\right] = -\dfrac{c \cdot g'(x)}{[g(x)]^2}$

Derivative of f(x)/c:
$\dfrac{d}{dx}\left[\dfrac{f(x)}{c}\right] = \dfrac{f'(x)}{c}$ (where c ≠ 0)

4. Step-by-Step Examples
Example 1: Basic Rational Function
Find $\dfrac{d}{dx}\left(\dfrac{x^2}{x+1}\right)$

Solution:
Let $f(x) = x^2$, $g(x) = x+1$
$f'(x) = 2x$, $g'(x) = 1$

$\dfrac{d}{dx}\left(\dfrac{x^2}{x+1}\right) = \dfrac{2x(x+1) - x^2(1)}{(x+1)^2} = \dfrac{2x^2 + 2x - x^2}{(x+1)^2} = \dfrac{x^2 + 2x}{(x+1)^2}$
Example 2: Trigonometric Quotient
Find $\dfrac{d}{dx}\left(\dfrac{\sin x}{\cos x}\right)$

Solution:
This is actually $\tan x$, and its derivative is $\sec^2 x$

Let $f(x) = \sin x$, $g(x) = \cos x$
$f'(x) = \cos x$, $g'(x) = -\sin x$

$\dfrac{d}{dx}\left(\dfrac{\sin x}{\cos x}\right) = \dfrac{\cos x \cdot \cos x - \sin x (-\sin x)}{\cos^2 x} = \dfrac{\cos^2 x + \sin^2 x}{\cos^2 x} = \dfrac{1}{\cos^2 x} = \sec^2 x$
Example 3: Complex Rational Function
Find $\dfrac{d}{dx}\left(\dfrac{x^3 - 2x}{x^2 + 1}\right)$

Solution:
Let $f(x) = x^3 - 2x$, $g(x) = x^2 + 1$
$f'(x) = 3x^2 - 2$, $g'(x) = 2x$

$\dfrac{d}{dx}\left(\dfrac{x^3 - 2x}{x^2 + 1}\right) = \dfrac{(3x^2 - 2)(x^2 + 1) - (x^3 - 2x)(2x)}{(x^2 + 1)^2}$

Numerator: $(3x^2 - 2)(x^2 + 1) = 3x^4 + 3x^2 - 2x^2 - 2 = 3x^4 + x^2 - 2$
$(x^3 - 2x)(2x) = 2x^4 - 4x^2$

So: $\dfrac{3x^4 + x^2 - 2 - 2x^4 + 4x^2}{(x^2 + 1)^2} = \dfrac{x^4 + 5x^2 - 2}{(x^2 + 1)^2}$
5. Common Mistakes to Avoid
  • Wrong: $\dfrac{d}{dx}\left(\dfrac{f}{g}\right) = \dfrac{f'}{g'}$ ❌
  • Wrong: $\dfrac{d}{dx}\left(\dfrac{f}{g}\right) = \dfrac{f' \cdot g'}{g^2}$ ❌
  • Wrong: Forgetting the square in denominator ❌
  • Wrong: Getting the subtraction order wrong ❌
  • Right: $\dfrac{d}{dx}\left(\dfrac{f}{g}\right) = \dfrac{f'g - fg'}{g^2}$ ✅
6. Applications
Physics: Average velocity = displacement/time, instantaneous velocity requires quotient rule
Economics: Price elasticity = (% change in quantity)/(% change in price)
Chemistry: Rate laws involving concentration ratios
7. Practice Problems

8. Key Takeaways
  • Quotient rule: "low d-high minus high d-low over square below"
  • Always check that denominator is never zero in domain
  • Can be derived from product rule with reciprocal
  • Common mistake: forgetting the subtraction in numerator
  • Simplify result when possible
  • Consider if other rules (like chain rule) might be needed too
