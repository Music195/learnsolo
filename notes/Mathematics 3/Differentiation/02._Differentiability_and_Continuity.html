Differentiability and Continuity
Differentiability and Continuity
Key Relationship:
$$\text{Differentiable at } x = a \implies \text{Continuous at } x = a$$
But the converse is NOT true!
1. Definitions
Continuity at $x = a$: $\lim_{x \to a} f(x) = f(a)$
Differentiability at $x = a$: $\lim_{h \to 0} \frac{f(a+h) - f(a)}{h}$ exists

3. Examples
Example 1: Differentiable Function
$f(x) = x^2$ is differentiable (and thus continuous) at $x = 0$
Example 2: Continuous but Not Differentiable
$f(x) = |x|$ is continuous at $x = 0$ but not differentiable there
Example 3: Discontinuous Function
$f(x) = \begin{cases} x & x \geq 0 \\ x+1 & x < 0 \end{cases}$ is discontinuous at $x = 0$
4. Why Differentiability Implies Continuity
If $f'(a)$ exists, then:
$\lim_{h \to 0} \frac{f(a+h) - f(a)}{h} = f'(a)$
$\therefore \lim_{h \to 0} [f(a+h) - f(a)] = 0$
$\therefore \lim_{h \to 0} f(a+h) = f(a)$
So $f$ is continuous at $x = a$
5. Common Points of Non-Differentiability
  • Cusps: Sharp points like $f(x) = |x|$ at $x = 0$
  • Vertical tangents: Infinite slope like $f(x) = x^{1/3}$ at $x = 0$
  • Discontinuities: Jumps or holes in the function
  • Corners: Abrupt changes in direction
6. Applications
Physics: Velocity exists only if position is continuous
Economics: Marginal functions require continuity
Engineering: Material stress analysis
7. Practice Problems
Problem 1: Determine if $f(x) = x^3$ is differentiable at $x = 0$.
Problem 2: Is $f(x) = |x - 1|$ differentiable at $x = 1$?
Problem 3: Where is $f(x) = \sqrt[3]{x}$ not differentiable?
8. Key Takeaways
  • Differentiability is a stronger condition than continuity
  • All differentiable functions are continuous
  • Not all continuous functions are differentiable
  • Look for cusps, corners, and discontinuities
  • Vertical tangents indicate infinite derivatives
