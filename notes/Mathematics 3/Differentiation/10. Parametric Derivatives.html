Parametric Derivatives

Basic Parametric Derivative Formulas

$\dfrac{dy}{dx} = \dfrac{\dfrac{dy}{dt}}{\dfrac{dx}{dt}} = \dfrac{g'(t)}{f'(t)}$
$\dfrac{d^2y}{dx^2} = \dfrac{d}{dx}\left(\dfrac{dy}{dx}\right) = \dfrac{\dfrac{d}{dt}\left(\dfrac{dy}{dx}\right)}{\dfrac{dx}{dt}}$
$\dfrac{d^2y}{dx^2} = \dfrac{\dfrac{d^2y}{dt^2} \cdot \dfrac{dx}{dt} - \dfrac{dy}{dt} \cdot \dfrac{d^2x}{dt^2}}{\left(\dfrac{dx}{dt}\right)^3}$

Understanding Parametric Derivatives

Parametric equations define curves using a parameter t, where both x and y are expressed as functions of t. The derivative dy/dx represents the slope of the curve at any point, found by dividing the rate of change of y with respect to t by the rate of change of x with respect to t.

Key Concepts:

  • Parameter as Time: Think of t as time - dx/dt is velocity in x-direction, dy/dt is velocity in y-direction
  • Slope Relationship: The slope dy/dx is the ratio of vertical speed to horizontal speed
  • Second Derivative: The second derivative involves the chain rule and quotient rule applied to the first derivative
  • Vertical Tangents: When dx/dt = 0, the derivative is undefined (vertical tangent line)
  • Horizontal Tangents: When dy/dt = 0, the slope is zero (horizontal tangent line)

Examples

Example 1: Basic Parametric Derivative

Find $\dfrac{dy}{dx}$ for the parametric equations: $x = t^2$, $y = t^3$
$\dfrac{dx}{dt} = 2t$, $\dfrac{dy}{dt} = 3t^2$

$\dfrac{dy}{dx} = \dfrac{\dfrac{dy}{dt}}{\dfrac{dx}{dt}} = \dfrac{3t^2}{2t} = \dfrac{3t}{2} \quad (t \neq 0)$
At t = 1: slope = 3/2 = 1.5. At t = 2: slope = 6/2 = 3. The slope increases as t increases.

Example 2: Circle Parametric Equations

Find $\dfrac{dy}{dx}$ for $x = \cos t$, $y = \sin t$ (unit circle)
$\dfrac{dx}{dt} = -\sin t$, $\dfrac{dy}{dt} = \cos t$

$\dfrac{dy}{dx} = \dfrac{\cos t}{-\sin t} = -\dfrac{\cos t}{\sin t} = -\cot t$
This gives the slope of the tangent line to the unit circle at parameter t. At t = π/4, slope = -cot(π/4) = -1.

Example 3: Second Derivative

Find $\dfrac{d^2y}{dx^2}$ for $x = t^2$, $y = t^3$
From Example 1: $\dfrac{dy}{dx} = \dfrac{3t}{2}$

$\dfrac{d^2y}{dx^2} = \dfrac{d}{dx}\left(\dfrac{3t}{2}\right) = \dfrac{d}{dt}\left(\dfrac{3t}{2}\right) \cdot \dfrac{dt}{dx} = \dfrac{3}{2} \cdot \dfrac{1}{\dfrac{dx}{dt}} = \dfrac{3}{2} \cdot \dfrac{1}{2t} = \dfrac{3}{4t}$
Using the formula: $\dfrac{d^2y}{dx^2} = \dfrac{\dfrac{d}{dt}\left(\dfrac{dy}{dx}\right)}{\dfrac{dx}{dt}} = \dfrac{\dfrac{d}{dt}\left(\dfrac{3t}{2}\right)}{\dfrac{dx}{dt}} = \dfrac{\dfrac{3}{2}}{2t} = \dfrac{3}{4t}$

Example 4: Ellipse

Find $\dfrac{dy}{dx}$ for the ellipse: $x = 3\cos t$, $y = 2\sin t$
$\dfrac{dx}{dt} = -3\sin t$, $\dfrac{dy}{dt} = 2\cos t$

$\dfrac{dy}{dx} = \dfrac{2\cos t}{-3\sin t} = -\dfrac{2\cos t}{3\sin t} = -\dfrac{2}{3}\cot t$
The slope is negative when sin t and cos t have the same sign, and positive when they have opposite signs.

Example 5: Cycloid

Find $\dfrac{dy}{dx}$ for the cycloid: $x = t - \sin t$, $y = 1 - \cos t$
$\dfrac{dx}{dt} = 1 - \cos t$, $\dfrac{dy}{dt} = \sin t$

$\dfrac{dy}{dx} = \dfrac{\sin t}{1 - \cos t}$
This parametric curve traces the path of a point on a rolling circle. The derivative becomes undefined when t = 2πk (cusp points).

Example 6: Tangent Line

Find the equation of the tangent line to $x = t^2 + 1$, $y = t^3 - t$ at t = 1
At t = 1: $x = 1^2 + 1 = 2$, $y = 1^3 - 1 = 0$

$\dfrac{dx}{dt} = 2t$, $\dfrac{dy}{dt} = 3t^2 - 1$

At t = 1: $\dfrac{dx}{dt} = 2$, $\dfrac{dy}{dt} = 3 - 1 = 2$

$\dfrac{dy}{dx} = \dfrac{2}{2} = 1$

Tangent line: $y - 0 = 1(x - 2)$ or $y = x - 2$
First find the point (2, 0), then compute the slope dy/dx = 1, then use point-slope form.

Derivation of Parametric Derivatives

First Derivative

Using the chain rule:

$\dfrac{dy}{dx} = \dfrac{dy}{dt} \cdot \dfrac{dt}{dx}$

But $\dfrac{dt}{dx} = \dfrac{1}{\dfrac{dx}{dt}}$

Therefore: $\dfrac{dy}{dx} = \dfrac{dy}{dt} \cdot \dfrac{1}{\dfrac{dx}{dt}} = \dfrac{\dfrac{dy}{dt}}{\dfrac{dx}{dt}}$

Second Derivative

The second derivative can be found by differentiating the first derivative with respect to x:

$\dfrac{d^2y}{dx^2} = \dfrac{d}{dx}\left(\dfrac{dy}{dx}\right) = \dfrac{d}{dt}\left(\dfrac{dy}{dx}\right) \cdot \dfrac{dt}{dx} = \dfrac{d}{dt}\left(\dfrac{\dfrac{dy}{dt}}{\dfrac{dx}{dt}}\right) \cdot \dfrac{1}{\dfrac{dx}{dt}}$

Using quotient rule: $\dfrac{d}{dt}\left(\dfrac{u}{v}\right) = \dfrac{u'v - uv'}{v^2}$ where u = dy/dt, v = dx/dt

$\dfrac{d^2y}{dx^2} = \dfrac{\dfrac{d^2y}{dt^2} \cdot \dfrac{dx}{dt} - \dfrac{dy}{dt} \cdot \dfrac{d^2x}{dt^2}}{\left(\dfrac{dx}{dt}\right)^2} \cdot \dfrac{1}{\dfrac{dx}{dt}} = \dfrac{\dfrac{d^2y}{dt^2} \cdot \dfrac{dx}{dt} - \dfrac{dy}{dt} \cdot \dfrac{d^2x}{dt^2}}{\left(\dfrac{dx}{dt}\right)^3}$

Applications

Physics: Projectile Motion

Projectile motion is naturally parametric: x = (v₀ cos θ)t, y = (v₀ sin θ)t - (1/2)gt². The derivative dy/dx gives the trajectory slope.

Engineering: Robotics and CNC Machines

Robot arms and CNC machines often use parametric equations to define tool paths. Derivatives help calculate velocities and accelerations.

Computer Graphics: Bézier Curves

Bézier curves in computer graphics are defined parametrically. Derivatives are used for curve smoothing and animation.

Astronomy: Planetary Orbits

Kepler's laws describe planetary orbits parametrically. Derivatives help analyze orbital velocities and accelerations.

Mechanics: Cycloid Paths

The cycloid (Example 5) is the path traced by a point on a rolling wheel. Used in gear design and brachistochrone problems.

Practice Problems

Problem 1

Find $\dfrac{dy}{dx}$ for $x = 2t + 1$, $y = t^2 - 3t$

Show Solution

$\dfrac{dx}{dt} = 2$, $\dfrac{dy}{dt} = 2t - 3$
$\dfrac{dy}{dx} = \dfrac{2t - 3}{2} = t - \dfrac{3}{2}$

Problem 2

Find $\dfrac{dy}{dx}$ for $x = e^t$, $y = e^{2t}$

Show Solution

$\dfrac{dx}{dt} = e^t$, $\dfrac{dy}{dt} = 2e^{2t}$
$\dfrac{dy}{dx} = \dfrac{2e^{2t}}{e^t} = 2e^t$

Problem 3

Find the slope at t = π/4 for $x = \sin t$, $y = \cos 2t$

Show Solution

$\dfrac{dx}{dt} = \cos t$, $\dfrac{dy}{dt} = -2\sin 2t$
At t = π/4: $\dfrac{dx}{dt} = \cos(\pi/4) = \sqrt{2}/2$, $\dfrac{dy}{dt} = -2\sin(\pi/2) = -2$
Slope = (-2)/(\sqrt{2}/2) = -2 * 2/\sqrt{2} = -2\sqrt{2}$

Problem 4

Find $\dfrac{d^2y}{dx^2}$ for $x = t^2$, $y = t^3$

Show Solution

From earlier: $\dfrac{dy}{dx} = \dfrac{3t}{2}$
$\dfrac{d^2y}{dx^2} = \dfrac{3}{4t}$

Problem 5

Find the equation of the tangent line at t = 1 for $x = t^3$, $y = t^2 + 2$

Show Solution

At t = 1: x = 1, y = 3
$\dfrac{dx}{dt} = 3t^2$, $\dfrac{dy}{dt} = 2t$
At t = 1: slope = (2)/(3) = 2/3
Tangent: y - 3 = (2/3)(x - 1)

Key Takeaways

Fundamental Formula

$\dfrac{dy}{dx} = \dfrac{\dfrac{dy}{dt}}{\dfrac{dx}{dt}}$ - divide y-velocity by x-velocity to get slope.

Vertical Tangents

When $\dfrac{dx}{dt} = 0$, the derivative is undefined, indicating a vertical tangent line.

Second Derivative Complexity

The second derivative formula is more complex, involving both first and second derivatives with respect to t.

Physical Interpretation

Think of t as time: dx/dt and dy/dt are velocities, dy/dx is the path slope.

Applications Everywhere

Parametric equations appear in physics (projectiles), engineering (robotics), graphics (curves), and astronomy (orbits).
