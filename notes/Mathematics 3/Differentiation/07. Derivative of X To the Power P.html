Derivative of x to the Power p
Derivative of $x^p$
The Power Rule:
$$\dfrac{d}{dx}(x^n) = n x^{n-1}$$
Where n is any real number (rational or irrational)
1. Understanding the Power Rule
For positive integer powers:
$\dfrac{d}{dx}(x^2) = 2x$, $\dfrac{d}{dx}(x^3) = 3x^2$, $\dfrac{d}{dx}(x^4) = 4x^3$

For negative powers:
$\dfrac{d}{dx}(x^{-1}) = -x^{-2}$, $\dfrac{d}{dx}(x^{-2}) = -2x^{-3}$

For fractional powers:
$\dfrac{d}{dx}(\sqrt{x}) = \dfrac{d}{dx}(x^{1/2}) = \dfrac{1}{2}x^{-1/2} = \dfrac{1}{2\sqrt{x}}$

Special case (n = 0):
$\dfrac{d}{dx}(x^0) = \dfrac{d}{dx}(1) = 0$

3. Step-by-Step Examples
Example 1: Positive Integer Power
Find $\dfrac{d}{dx}(x^5)$

Solution:
Using the power rule: $\dfrac{d}{dx}(x^5) = 5x^{5-1} = 5x^4$
Example 2: Negative Power
Find $\dfrac{d}{dx}\left(\dfrac{1}{x^3}\right) = \dfrac{d}{dx}(x^{-3})$

Solution:
$\dfrac{d}{dx}(x^{-3}) = -3x^{-3-1} = -3x^{-4} = -\dfrac{3}{x^4}$
Example 3: Fractional Power (Square Root)
Find $\dfrac{d}{dx}(\sqrt{x}) = \dfrac{d}{dx}(x^{1/2})$

Solution:
$\dfrac{d}{dx}(x^{1/2}) = \dfrac{1}{2}x^{1/2 - 1} = \dfrac{1}{2}x^{-1/2} = \dfrac{1}{2\sqrt{x}}$
Example 4: Cube Root
Find $\dfrac{d}{dx}(\sqrt[3]{x}) = \dfrac{d}{dx}(x^{1/3})$

Solution:
$\dfrac{d}{dx}(x^{1/3}) = \dfrac{1}{3}x^{1/3 - 1} = \dfrac{1}{3}x^{-2/3} = \dfrac{1}{3x^{2/3}} = \dfrac{1}{3\sqrt[3]{x^2}}$
4. Proof of the Power Rule
For integer n:
Using the definition: $\dfrac{d}{dx}(x^n) = \lim_{h \to 0} \dfrac{(x+h)^n - x^n}{h}$

Expand $(x+h)^n = x^n + nx^{n-1}h + \dfrac{n(n-1)}{2}x^{n-2}h^2 + \cdots + h^n$

So: $\dfrac{(x+h)^n - x^n}{h} = nx^{n-1} + \dfrac{n(n-1)}{2}x^{n-2}h + \cdots + h^{n-1}$

As $h \to 0$, all terms with h vanish except $nx^{n-1}$

For general real exponents: Requires logarithmic differentiation
5. Common Power Rule Patterns
  • Constants: $\dfrac{d}{dx}(c) = 0$
  • Linear: $\dfrac{d}{dx}(x) = 1$
  • Quadratic: $\dfrac{d}{dx}(x^2) = 2x$
  • Cubic: $\dfrac{d}{dx}(x^3) = 3x^2$
  • Square root: $\dfrac{d}{dx}(\sqrt{x}) = \dfrac{1}{2\sqrt{x}}$
  • Cube root: $\dfrac{d}{dx}(\sqrt[3]{x}) = \dfrac{1}{3x^{2/3}}$
  • Reciprocals: $\dfrac{d}{dx}(x^{-n}) = -nx^{-n-1}$
6. Applications
Physics: Position functions, gravitational potential energy ∝ 1/r
Biology: Population growth models, surface area to volume ratios
Economics: Cost functions, diminishing returns

8. Key Takeaways
  • Power rule applies to any real exponent
  • Subtract 1 from the exponent
  • Coefficient becomes the new coefficient
  • Negative exponents: power becomes more negative
  • Fractional exponents: denominator becomes numerator
  • Zero exponent: derivative is zero
  • Most fundamental rule in differential calculus
