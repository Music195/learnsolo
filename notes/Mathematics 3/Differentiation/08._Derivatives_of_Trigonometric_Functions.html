Derivatives of Trigonometric Functions

Basic Trigonometric Derivatives

$\dfrac{d}{dx}[\sin x] = \cos x$
$\dfrac{d}{dx}[\cos x] = -\sin x$
$\dfrac{d}{dx}[\tan x] = \sec^2 x = \dfrac{1}{\cos^2 x} = 1 + \tan^2 x$
$\dfrac{d}{dx}[\cot x] = -\csc^2 x$
$\dfrac{d}{dx}[\sec x] = \sec x \tan x$
$\dfrac{d}{dx}[\csc x] = -\csc x \cot x$

Understanding Trigonometric Derivatives

The derivatives of trigonometric functions are fundamental in calculus because they connect the rates of change of angles with the trigonometric ratios. These derivatives are derived from the definitions of the trigonometric functions and the concept of limits.

Key Relationships:

  • Sine and Cosine: The derivative of sine is cosine, and the derivative of cosine is negative sine. This creates a cycle: sin → cos → -sin → -cos → sin
  • Tangent: The derivative of tangent equals secant squared, which can also be expressed as 1 + tan²(x)
  • Reciprocal Functions: The derivatives of secant, cosecant, and cotangent follow patterns based on their relationships to sine and cosine

Examples

Example 1: Derivative of sin(x)

Find $\dfrac{d}{dx}[\sin x]$
$\dfrac{d}{dx}[\sin x] = \cos x$
The derivative of sine is cosine. At x = 0: sin(0) = 0, cos(0) = 1, so the slope is 1.

Example 2: Derivative of cos(x)

Find $\dfrac{d}{dx}[\cos x]$
$\dfrac{d}{dx}[\cos x] = -\sin x$
The derivative of cosine is negative sine. At x = 0: cos(0) = 1, -sin(0) = 0, so the slope is 0.

Example 3: Derivative of tan(x)

Find $\dfrac{d}{dx}[\tan x]$
$\dfrac{d}{dx}[\tan x] = \sec^2 x = \dfrac{1}{\cos^2 x} = 1 + \tan^2 x$
Using quotient rule: tan(x) = sin(x)/cos(x), so derivative is [cos(x)cos(x) - sin(x)(-sin(x))]/cos²(x) = [cos²(x) + sin²(x)]/cos²(x) = 1/cos²(x) = sec²(x)

Example 4: Derivative of sin(2x)

Find $\dfrac{d}{dx}[\sin(2x)]$
$\dfrac{d}{dx}[\sin(2x)] = \cos(2x) \cdot 2 = 2\cos(2x)$
Using chain rule: derivative of sin(u) where u = 2x is cos(u) · du/dx = cos(2x) · 2

Example 5: Derivative of cos³(x)

Find $\dfrac{d}{dx}[\cos^3 x]$
$\dfrac{d}{dx}[\cos^3 x] = 3\cos^2 x \cdot (-\sin x) = -3\cos^2 x \sin x$
Using power rule and chain rule: derivative of [cos(x)]³ is 3[cos(x)]² · d/dx[cos(x)] = 3cos²(x) · (-sin(x))

Derivation of Trigonometric Derivatives

Derivative of sin(x)

Using the limit definition:

$\dfrac{d}{dx}[\sin x] = \lim_{h \to 0} \dfrac{\sin(x+h) - \sin x}{h} = \lim_{h \to 0} \dfrac{\sin x \cos h + \cos x \sin h - \sin x}{h}$

$= \lim_{h \to 0} \dfrac{\sin x (\cos h - 1) + \cos x \sin h}{h} = \lim_{h \to 0} \left[ \sin x \cdot \dfrac{\cos h - 1}{h} + \cos x \cdot \dfrac{\sin h}{h} \right]$

$= \sin x \cdot 0 + \cos x \cdot 1 = \cos x$

Derivative of cos(x)

Using the limit definition:

$\dfrac{d}{dx}[\cos x] = \lim_{h \to 0} \dfrac{\cos(x+h) - \cos x}{h} = \lim_{h \to 0} \dfrac{\cos x \cos h - \sin x \sin h - \cos x}{h}$

$= \lim_{h \to 0} \dfrac{\cos x (\cos h - 1) - \sin x \sin h}{h} = \lim_{h \to 0} \left[ \cos x \cdot \dfrac{\cos h - 1}{h} - \sin x \cdot \dfrac{\sin h}{h} \right]$

$= \cos x \cdot 0 - \sin x \cdot 1 = -\sin x$

Applications

Physics: Simple Harmonic Motion

The position of a mass on a spring is given by x(t) = A sin(ωt). The velocity is the derivative: v(t) = Aω cos(ωt), and acceleration is v'(t) = -Aω² sin(ωt).

Engineering: AC Circuit Analysis

In AC circuits, voltage and current often follow sinusoidal patterns. The derivatives help analyze power, phase relationships, and circuit behavior.

Mechanics: Circular Motion

For an object moving in a circle with position (cos θ, sin θ), the velocity components involve the derivatives of sine and cosine.

Wave Functions

Many wave phenomena (sound, light, electromagnetic waves) are described by trigonometric functions, and their derivatives give wave velocities and accelerations.

Practice Problems

Problem 1

Find the derivative: $\dfrac{d}{dx}[3\sin x + 2\cos x]$

Show Solution

$\dfrac{d}{dx}[3\sin x + 2\cos x] = 3\cos x - 2\sin x$

Problem 2

Find the derivative: $\dfrac{d}{dx}[\sin^2 x]$

Show Solution

$\dfrac{d}{dx}[\sin^2 x] = 2\sin x \cos x = \sin 2x$

Problem 3

Find the derivative: $\dfrac{d}{dx}[\tan(3x)]$

Show Solution

$\dfrac{d}{dx}[\tan(3x)] = \sec^2(3x) \cdot 3 = 3\sec^2(3x)$

Problem 4

Find the derivative: $\dfrac{d}{dx}[\cos(x^2)]$

Show Solution

$\dfrac{d}{dx}[\cos(x^2)] = -\sin(x^2) \cdot 2x = -2x\sin(x^2)$

Problem 5

Find the derivative: $\dfrac{d}{dx}[\dfrac{\sin x}{1 + \cos x}]$

Show Solution

Using quotient rule: $\dfrac{(\cos x)(1 + \cos x) - \sin x (-\sin x)}{(1 + \cos x)^2} = \dfrac{\cos x + \cos^2 x + \sin^2 x}{(1 + \cos x)^2} = \dfrac{\cos x + 1}{(1 + \cos x)^2} = \dfrac{1 + \cos x}{(1 + \cos x)^2} = \dfrac{1}{1 + \cos x}$

Key Takeaways

Memorize the Basics

$\dfrac{d}{dx}[\sin x] = \cos x$ and $\dfrac{d}{dx}[\cos x] = -\sin x$ are fundamental. All other trigonometric derivatives build from these.

Tangent Derivative

$\dfrac{d}{dx}[\tan x] = \sec^2 x$ can be remembered as "tangent's derivative is secant squared."

Chain Rule Integration

When differentiating composite functions like sin(2x) or cos(x²), always apply the chain rule: derivative of outer function times derivative of inner function.

Reciprocal Functions

The derivatives of secant, cosecant, and cotangent follow predictable patterns based on their relationships to sine and cosine.

Practical Applications

These derivatives are essential in physics (SHM, waves), engineering (circuits, signals), and many other fields involving periodic phenomena.
