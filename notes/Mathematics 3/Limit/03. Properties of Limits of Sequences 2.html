These additional properties help us work with inequalities and bounded sequences:

üßÄ The Squeeze Theorem (Sandwich Theorem)

If a sequence is squeezed between two other sequences that both converge to the same limit, then the middle sequence must also converge to that limit.

\( a_n \leq c_n \leq b_n \quad \text{and} \quad \lim_{n \to \infty} a_n = \lim_{n \to \infty} b_n = L \implies \lim_{n \to \infty} c_n = L \)

üîç Examples

Example 1: Order Preservation

Given that \( \frac{1}{n+1} \leq \frac{1}{\sqrt{n}} \) for n ‚â• 1, and both sequences converge, what can we say about their limits?

\( \lim_{n \to \infty} \frac{1}{n+1} = 0 \leq \lim_{n \to \infty} \frac{1}{\sqrt{n}} = 0 \)

Both limits are 0, which satisfies the inequality.

Example 2: Squeeze Theorem

Find \( \lim_{n \to \infty} \frac{\sin n}{n} \).

We know that \( -1 \leq \sin n \leq 1 \) for all n, so:

\( -\frac{1}{n} \leq \frac{\sin n}{n} \leq \frac{1}{n} \)

As \( n \to \infty \), \( \lim_{n \to \infty} -\frac{1}{n} = 0 \) and \( \lim_{n \to \infty} \frac{1}{n} = 0 \)

Therefore, by the Squeeze Theorem: \( \lim_{n \to \infty} \frac{\sin n}{n} = 0 \)

Example 3: Bounded Sequences

If \( a_n \leq 5 \) for all n and \( \lim_{n \to \infty} a_n = 3 \), what can we conclude?

By order preservation, if \( a_n \leq 5 \) for all n, then \( \lim a_n \leq 5 \). Since the limit is 3, this is satisfied.

Problem 1: Use the Squeeze Theorem to find \( \lim_{n \to \infty} \frac{\cos n}{n^2} \).

We know that \( -1 \leq \cos n \leq 1 \) for all n, so:

\( -\frac{1}{n^2} \leq \frac{\cos n}{n^2} \leq \frac{1}{n^2} \)

As \( n \to \infty \), both bounding sequences go to 0:

\( \lim_{n \to \infty} -\frac{1}{n^2} = 0 \) and \( \lim_{n \to \infty} \frac{1}{n^2} = 0 \)

Therefore, \( \lim_{n \to \infty} \frac{\cos n}{n^2} = 0 \)
Problem 2: Given that \( \frac{n}{n+1} \leq a_n \leq 1 \) for all n ‚â• 1, and \( \lim_{n \to \infty} \frac{n}{n+1} = 1 \), find \( \lim_{n \to \infty} a_n \).

We have \( \frac{n}{n+1} \leq a_n \leq 1 \), and \( \lim_{n \to \infty} \frac{n}{n+1} = 1 \).

By order preservation, \( \lim_{n \to \infty} a_n \geq 1 \).

Since a_n ‚â§ 1 for all n, and the sequence is bounded above by 1, if it converges, the limit must be ‚â§ 1.

Therefore, \( \lim_{n \to \infty} a_n = 1 \).

Note: This uses the fact that if a sequence is bounded and has a lower bound approaching the upper bound, it must converge to the upper bound.

Problem 3: Show that \( \lim_{n \to \infty} \frac{(-1)^n}{n} = 0 \) using the Squeeze Theorem.

We know that \( -\frac{1}{n} \leq \frac{(-1)^n}{n} \leq \frac{1}{n} \).

When (-1)^n = 1: \( \frac{1}{n} \geq \frac{(-1)^n}{n} \geq -\frac{1}{n} \)

When (-1)^n = -1: \( -\frac{1}{n} \leq \frac{(-1)^n}{n} \leq \frac{1}{n} \)

So in both cases: \( -\frac{1}{n} \leq \frac{(-1)^n}{n} \leq \frac{1}{n} \)

As \( n \to \infty \), \( \lim_{n \to \infty} -\frac{1}{n} = 0 \) and \( \lim_{n \to \infty} \frac{1}{n} = 0 \)

Therefore, by the Squeeze Theorem: \( \lim_{n \to \infty} \frac{(-1)^n}{n} = 0 \)

üéØ Key Takeaways

  • Order Preservation: Limits preserve inequalities between sequences
  • Squeeze Theorem: If a sequence is squeezed between two sequences with the same limit, it converges to that limit
  • Useful for limits involving bounded functions like sin(n), cos(n)
  • Essential for proving convergence of oscillating sequences
  • Combines with other properties for complex limit calculations
  • Critical for understanding bounded sequences and their convergence
