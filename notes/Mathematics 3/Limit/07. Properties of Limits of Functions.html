Properties of Limits of Functions

üìö Core Concepts

Function limits have algebraic properties that allow us to evaluate complex limits by breaking them down into simpler parts. These properties are analogous to the properties of limits of sequences.

If $\lim_{x \to a} f(x) = \alpha$ and $\lim_{x \to a} g(x) = \beta$ (both finite), then:
‚Ä¢ Constants and scalars work as expected
‚Ä¢ Sums and products behave predictably
‚Ä¢ Quotients work when the denominator limit isn't zero
‚Ä¢ Order is preserved in inequalities
‚Ä¢ The squeeze theorem allows us to find limits of bounded functions
Important: These properties assume the individual limits exist and are finite. Special care is needed when dealing with infinite limits or indeterminate forms.

üìä Algebraic Properties

1. Constant Multiple

$\lim_{x \to a} k f(x) = k \lim_{x \to a} f(x) = k\alpha$

Constants can be factored out of limits.

2. Sum & Difference

$\lim_{x \to a} [f(x) + g(x)] = \alpha + \beta$
$\lim_{x \to a} [f(x) - g(x)] = \alpha - \beta$

Limits of sums are sums of limits.

3. Product

$\lim_{x \to a} f(x)g(x) = \alpha \cdot \beta$

Limits of products are products of limits.

4. Quotient

$\lim_{x \to a} \dfrac{f(x)}{g(x)} = \dfrac{\alpha}{\beta}$

Provided $\beta \neq 0$ and $g(x) \neq 0$ near a.

üîç Order Properties & Squeeze Theorem

5. Order Preservation

If $f(x) \leq g(x)$ near a, then $\alpha \leq \beta$

Inequality relations are preserved in limits.

6. Squeeze Theorem

If $f(x) \leq h(x) \leq g(x)$ near a
and $\lim_{x \to a} f(x) = \lim_{x \to a} g(x) = L$
then $\lim_{x \to a} h(x) = L$

Sandwich bounded functions to find their limits.

üéØ The Squeeze Theorem in Action

The squeeze theorem is one of the most powerful tools for evaluating limits. It allows you to find the limit of a function that is "squeezed" between two other functions that have the same limit.

Classic Example: $\lim_{x \to 0} x^2 \sin(1/x)$

We know $-1 \leq \sin(1/x) \leq 1$, so $-x^2 \leq x^2 \sin(1/x) \leq x^2$. As $x \to 0$, both bounds approach 0, so the limit is 0.

üí° Examples

Example 1: Evaluate $\lim_{x \to 2} (3x + 5)$.

Using properties: $\lim_{x \to 2} 3x + \lim_{x \to 2} 5 = 3 \cdot 2 + 5 = 11$.
Example 2: Find $\lim_{x \to 1} \dfrac{x^2 - 1}{x - 1}$.

Direct substitution gives 0/0. Using properties: $\lim_{x \to 1} (x+1) = 2$.
Example 3 (Squeeze Theorem): Evaluate $\lim_{x \to 0} x \sin(1/x)$.

We know $-|x| \leq x \sin(1/x) \leq |x|$, and both bounds ‚Üí 0, so limit = 0.

  • Asymptotes: Finding horizontal and vertical asymptotes
  • Indeterminate Forms: Resolving 0/0 and ‚àû/‚àû forms

üìù Practice Problems

Problem 1: Evaluate $\lim_{x \to 3} (2x^2 - 5x + 1)$.

Using properties: $2(3)^2 - 5(3) + 1 = 2(9) - 15 + 1 = 18 - 15 + 1 = 4$.
Problem 2: Find $\lim_{x \to 4} \dfrac{x-4}{x^2-16}$.

Direct substitution gives 0/0. Factor: $\dfrac{x-4}{(x-4)(x+4)} = \dfrac{1}{x+4}$. Limit as x‚Üí4 is 1/8.
Problem 3: Use the squeeze theorem to evaluate $\lim_{x \to 0} x^3 \cos(1/x)$.

We know $-x^3 \leq x^3 \cos(1/x) \leq x^3$. Both bounds ‚Üí 0 as x‚Üí0, so limit = 0.
Problem 4: Given $\lim_{x \to 2} f(x) = 3$ and $\lim_{x \to 2} g(x) = 5$, find $\lim_{x \to 2} [2f(x) + g(x)]$.

Using properties: $2 \cdot 3 + 5 = 6 + 5 = 11$.

üéØ Key Takeaways

  • Limits obey algebraic rules for convergent functions
  • Constants factor out, sums become sums of limits
  • Products and quotients work with finite non-zero denominators
  • Order is preserved in inequalities
  • Squeeze theorem is powerful for bounded oscillating functions
  • Properties extend to infinite limits with modifications
  • Essential for evaluating complex limits and derivatives
