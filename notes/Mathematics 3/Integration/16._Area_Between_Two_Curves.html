Area Between Two Curves

The area between two curves is a powerful application of definite integrals that allows us to find the area enclosed by two functions. This concept extends the idea of area under a single curve to regions bounded by multiple curves.

Why This Matters:

Many real-world problems involve finding areas between curves, such as the area between supply and demand curves in economics, or the area between velocity and acceleration curves in physics. Integration provides the exact mathematical tool for these calculations.

Basic Formula

Standard Case: If f(x) ≥ g(x) for all x in [a, b], the area S between the curves y = f(x) and y = g(x) from x = a to x = b is:

\( S = \int_a^b [f(x) - g(x)] \, dx \)

General Case: When the curves intersect or cross, the area is:

\( S = \int_a^b |f(x) - g(x)| \, dx \)

Vertical Strips Method

The area can be thought of as a collection of vertical strips, each of width dx and height |f(x) - g(x)|.

Each strip contributes an area of |f(x) - g(x)| dx to the total area.

Summing all strips gives: \( S = \int_a^b |f(x) - g(x)| \, dx \)

Finding Intersection Points

To find the exact area between curves, you need to determine the limits of integration by finding where the curves intersect:

Solve: f(x) = g(x)

The solutions x = c₁, c₂, ..., cn determine the intervals where you can integrate.

Example: Finding Intersections

For f(x) = x² and g(x) = x, solve x² = x:

x² - x = 0 → x(x - 1) = 0 → x = 0, x = 1

Intersection points: (0, 0) and (1, 1)

Examples

Example 1: Simple Case (f ≥ g)

Find the area between y = x² and y = x from x = 0 to x = 1.

Solution:

First, check which is larger: for x ∈ [0, 1], x ≥ x² (since x - x² = x(1 - x) ≥ 0).

So f(x) = x, g(x) = x², and f(x) ≥ g(x).

Area = \( \int_0^1 (x - x^2) \, dx = \left[ \frac{x^2}{2} - \frac{x^3}{3} \right]_0^1 = \frac{1}{2} - \frac{1}{3} = \frac{1}{6} \)

Example 2: Curves That Cross

Find the area between y = x² - 1 and y = 1 - x² from x = -1 to x = 1.

Solution:

These curves are symmetric. At x = 0: both equal -1 + 1 = 0.

For |x| < 1, 1 - x² ≥ x² - 1, so upper curve is y = 1 - x².

Area = \( \int_{-1}^1 |(x^2 - 1) - (1 - x^2)| \, dx = \int_{-1}^1 |2x^2 - 2| \, dx = \int_{-1}^1 2|x^2 - 1| \, dx \)

Since x² - 1 ≤ 0 on [-1, 1], |x² - 1| = 1 - x².

Area = \( 2 \int_{-1}^1 (1 - x^2) \, dx = 4 \int_0^1 (1 - x^2) \, dx = 4 \left[ x - \frac{x^3}{3} \right]_0^1 = 4\left(1 - \frac{1}{3}\right) = \frac{8}{3} \)

Example 3: Multiple Intersections

Find the area between y = x³ and y = x from x = -1 to x = 1.

Solution:

Intersections: x³ = x → x³ - x = 0 → x(x² - 1) = 0 → x = -1, 0, 1

On [-1, 0]: x³ ≤ x (since x³ is odd, x is odd), so upper is y = x

On [0, 1]: x³ ≥ x (x³ - x = x(x² - 1) ≥ 0), so upper is y = x³

Area = \( \int_{-1}^0 (x - x^3) \, dx + \int_0^1 (x^3 - x) \, dx \)

Both integrals give the same value 1/4, so total area = 1/2

Practice Problems

Problem 1

Find the area between y = √x and y = x² from x = 0 to x = 1.

Solution:

Intersections: √x = x² → x = x⁴ → x⁴ - x = 0 → x(x³ - 1) = 0 → x = 0, 1

On [0, 1]: √x ≥ x², so upper curve is y = √x

Area = \( \int_0^1 (\sqrt{x} - x^2) \, dx = \left[ \frac{2}{3} x^{3/2} - \frac{x^3}{3} \right]_0^1 = \frac{2}{3} - \frac{1}{3} = \frac{1}{3} \)

Problem 2

Find the area between y = e^x and y = e^{-x} from x = 0 to x = 1.

Solution:

On [0, 1]: e^x ≥ e^{-x}, so upper curve is y = e^x

Area = \( \int_0^1 (e^x - e^{-x}) \, dx = [e^x + e^{-x}]_0^1 = (e + e^{-1}) - (1 + 1) = e + \frac{1}{e} - 2 \)

≈ 2.718 + 0.368 - 2 = 1.086

Problem 3

Find the area between y = sin x and y = cos x from x = 0 to x = π/4.

Solution:

On [0, π/4]: sin x and cos x intersect at π/4 where both equal √2/2

For x ∈ [0, π/4]: cos x ≥ sin x, so upper curve is y = cos x

Area = \( \int_0^{\pi/4} (\cos x - \sin x) \, dx = [\sin x + \cos x]_0^{\pi/4} = (\frac{\sqrt{2}}{2} + \frac{\sqrt{2}}{2}) - (0 + 1) = \sqrt{2} - 1 \)

≈ 1.414 - 1 = 0.414

Key Takeaways

  • Area = ∫[f(x) - g(x)] dx when f(x) ≥ g(x) on [a, b]
  • Use |f(x) - g(x)| when curves cross or intersect
  • Find intersection points by solving f(x) = g(x)
  • Break into subintervals if curves switch which is on top
  • Always sketch the curves to visualize the region
  • Vertical strips method helps understand the concept
  • Integration gives exact areas for complex regions
  • Applications in physics, economics, and engineering
