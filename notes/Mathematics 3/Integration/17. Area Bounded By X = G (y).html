Area Bounded by x = g(y)

When curves are expressed as functions of y rather than x, we need a different approach to find areas. This occurs when the curve is vertical or nearly vertical, making horizontal strips more appropriate than vertical strips.

When to Use This Method:

Use integration with respect to y when the region is more naturally described by horizontal strips, or when the boundaries are expressed as functions of y. This is particularly useful for regions bounded by vertical lines or curves that are steep.

Basic Formula

Standard Case: If g(y) ≥ 0 for all y in [c, d], the area S bounded by x = g(y), the y-axis, and the lines y = c, y = d is:

\( S = \int_c^d g(y) \, dy \)

Horizontal Strips Method:

The area can be thought of as a collection of horizontal strips, each of width g(y) and height dy.

Each strip contributes an area of g(y) dy to the total area.

Comparison with Vertical Strips

Vertical strips: Area = ∫ [f(x) - g(x)] dx (integrate with respect to x)

Horizontal strips: Area = ∫ [f(y) - g(y)] dy (integrate with respect to y)

The choice depends on which variable is easier to express the boundaries in terms of.

General Case: Area Between Two Curves

The area between x = f(y) and x = g(y) from y = c to y = d is:

\( S = \int_c^d |f(y) - g(y)| \, dy \)

If f(y) ≥ g(y) on [c, d]:

\( S = \int_c^d (f(y) - g(y)) \, dy \)

When to Switch to dy Integration

  • When curves are expressed as x = f(y)
  • When vertical strips would require splitting the region
  • When the region has vertical boundaries
  • When dx integration leads to more complex expressions
  • When the curve is nearly vertical

Examples

Example 1: Simple Case

Find the area bounded by x = y², the y-axis, and y = 1, y = 2.

Solution:

The curve x = y² is a parabola opening to the right. Since y² ≥ 0, the area is:

\( S = \int_1^2 y^2 \, dy = \left[ \frac{y^3}{3} \right]_1^2 = \frac{8}{3} - \frac{1}{3} = \frac{7}{3} \)

The area is 7/3 square units.

Example 2: Between Two Curves

Find the area between x = y² and x = y + 2 from y = 0 to y = 2.

Solution:

First, find which curve is to the right: at y = 1, y² = 1, y + 2 = 3, so x = y + 2 is farther right.

Area = \( \int_0^2 ((y + 2) - y^2) \, dy = \int_0^2 (y + 2 - y^2) \, dy \)

\( = \left[ \frac{y^2}{2} + 2y - \frac{y^3}{3} \right]_0^2 = \left( \frac{4}{2} + 4 - \frac{8}{3} \right) - 0 = \left(2 + 4 - \frac{8}{3}\right) = 6 - \frac{8}{3} = \frac{10}{3} \)

Example 3: Region Bounded by Vertical Lines

Find the area of the region bounded by y = x², y = 4, and the y-axis.

Solution:

This region is bounded by y = x² (parabola), y = 4 (horizontal line), and x = 0 (y-axis).

Express in terms of y: for each y from 0 to 4, x goes from 0 to √y.

Area = \( \int_0^4 \sqrt{y} \, dy = \int_0^4 y^{1/2} \, dy = \left[ \frac{2}{3} y^{3/2} \right]_0^4 = \frac{2}{3} (8) = \frac{16}{3} \)

Practice Problems

Problem 1

Find the area bounded by x = √y, the y-axis, and y = 4.

Solution:

The curve x = √y is the upper half of a parabola. Area = \( \int_0^4 \sqrt{y} \, dy = \int_0^4 y^{1/2} \, dy \)

\( = \left[ \frac{2}{3} y^{3/2} \right]_0^4 = \frac{2}{3} (8) = \frac{16}{3} \)

Problem 2

Find the area between x = y² and x = 2y from y = 0 to y = 2.

Solution:

At y = 1: y² = 1, 2y = 2, so x = 2y is farther right.

Area = \( \int_0^2 (2y - y^2) \, dy = \left[ y^2 - \frac{y^3}{3} \right]_0^2 = (4 - \frac{8}{3}) - 0 = \frac{12}{3} - \frac{8}{3} = \frac{4}{3} \)

Problem 3

Find the area of the region bounded by y = x², x = 0, and y = 1.

Solution:

The region is bounded by the parabola y = x², the y-axis x = 0, and the line y = 1.

For each y from 0 to 1, x goes from 0 to √y.

Area = \( \int_0^1 \sqrt{y} \, dy = \left[ \frac{2}{3} y^{3/2} \right]_0^1 = \frac{2}{3} \)

Key Takeaways

  • Use dy integration when curves are expressed as x = f(y)
  • Horizontal strips: each contributes width × dy to the area
  • Area = ∫ g(y) dy when bounded by x = g(y), y-axis, y = c, y = d
  • Between curves: Area = ∫ |f(y) - g(y)| dy
  • Switch to dy when dx integration is difficult or complex
  • Vertical boundaries often suggest dy integration
  • Sketch the region to determine appropriate integration variable
  • Both dx and dy methods give the same result for the same region
