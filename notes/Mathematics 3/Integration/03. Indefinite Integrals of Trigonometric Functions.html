Indefinite Integrals of Trigonometric Functions

Basic Trigonometric Integrals

The most fundamental trigonometric integrals are the antiderivatives of sine and cosine functions:

Basic Rules:
  • \( \int \sin x \, dx = -\cos x + C \)
  • \( \int \cos x \, dx = \sin x + C \)
  • \( \int \sec^2 x \, dx = \tan x + C \)
  • \( \int \csc^2 x \, dx = -\cot x + C \)

Why These Rules Work

These integrals are derived directly from the derivatives of trigonometric functions:

  • \( \frac{d}{dx}[-\cos x] = \sin x \), so \( \int \sin x \, dx = -\cos x + C \)
  • \( \frac{d}{dx}[\sin x] = \cos x \), so \( \int \cos x \, dx = \sin x + C \)
  • \( \frac{d}{dx}[\tan x] = \sec^2 x \), so \( \int \sec^2 x \, dx = \tan x + C \)
  • \( \frac{d}{dx}[-\cot x] = \csc^2 x \), so \( \int \csc^2 x \, dx = -\cot x + C \)

Advanced Trigonometric Integrals

More complex trigonometric integrals often require trigonometric identities:

Using Pythagorean Identities:
  • \( \int \tan x \, dx = \int \frac{\sin x}{\cos x} \, dx = -\ln|\cos x| + C = \ln|\sec x| + C \)
  • \( \int \cot x \, dx = \int \frac{\cos x}{\sin x} \, dx = \ln|\sin x| + C \)
  • \( \int \sec x \, dx = \ln|\sec x + \tan x| + C \) (more advanced)
  • \( \int \csc x \, dx = -\ln|\csc x + \cot x| + C \) (more advanced)

Visual Guide

The trigonometric integrals represent areas under trigonometric curves.

Examples

Example 1: Basic Sine Integral

Find \( \int \sin(3x) \, dx \)

Using the substitution method: Let u = 3x, then du = 3 dx, so dx = du/3

\( \int \sin(3x) \, dx = \int \sin u \cdot \frac{du}{3} = \frac{1}{3} \int \sin u \, du = \frac{1}{3} (-\cos u) + C = -\frac{1}{3} \cos(3x) + C \)

Example 2: Secant Squared

Find \( \int \sec^2(2x) \, dx \)

Using substitution: Let u = 2x, then du = 2 dx, so dx = du/2

\( \int \sec^2(2x) \, dx = \int \sec^2 u \cdot \frac{du}{2} = \frac{1}{2} \int \sec^2 u \, du = \frac{1}{2} \tan u + C = \frac{1}{2} \tan(2x) + C \)

Example 3: Tangent Function

Find \( \int \tan x \, dx \)

Rewrite using identity: \( \tan x = \frac{\sin x}{\cos x} \)

\( \int \tan x \, dx = \int \frac{\sin x}{\cos x} \, dx \)

Let u = cos x, then du = -sin x dx, so -du = sin x dx

\( \int \frac{\sin x}{\cos x} \, dx = \int \frac{-du}{u} = -\int \frac{du}{u} = -\ln|u| + C = -\ln|\cos x| + C = \ln|\sec x| + C \)

Example 4: Cosecant Squared

Find \( \int \csc^2(4x) \, dx \)

Using substitution: Let u = 4x, then du = 4 dx, so dx = du/4

\( \int \csc^2(4x) \, dx = \int \csc^2 u \cdot \frac{du}{4} = \frac{1}{4} \int \csc^2 u \, du = \frac{1}{4} (-\cot u) + C = -\frac{1}{4} \cot(4x) + C \)

Integration Techniques for Trigonometric Functions

  • Direct Integration: For basic functions like sin x, cos x, sec²x, csc²x
  • Substitution: For functions like sin(kx), cos(kx), sec²(kx), etc.
  • Trigonometric Identities: Rewrite complex expressions using identities
  • Integration by Parts: For products involving trigonometric functions
  • Reduction Formulas: For higher powers of trigonometric functions

Applications

  • Physics: Simple harmonic motion, wave functions, pendulum problems
  • Engineering: AC circuit analysis, signal processing
  • Mathematics: Fourier series, differential equations
  • Real-world: Sound waves, electromagnetic waves, oscillatory motion

Practice Problems

1. Find \( \int \sin(5x) \, dx \)

Solution:

Let u = 5x, then du = 5 dx, so dx = du/5

\( \int \sin(5x) \, dx = \int \sin u \cdot \frac{du}{5} = \frac{1}{5} \int \sin u \, du = \frac{1}{5} (-\cos u) + C = -\frac{1}{5} \cos(5x) + C \)

2. Find \( \int \cos(2x) \, dx \)

Solution:

Let u = 2x, then du = 2 dx, so dx = du/2

\( \int \cos(2x) \, dx = \int \cos u \cdot \frac{du}{2} = \frac{1}{2} \int \cos u \, du = \frac{1}{2} \sin u + C = \frac{1}{2} \sin(2x) + C \)

3. Find \( \int \sec^2(3x) \, dx \)

Solution:

Let u = 3x, then du = 3 dx, so dx = du/3

\( \int \sec^2(3x) \, dx = \int \sec^2 u \cdot \frac{du}{3} = \frac{1}{3} \int \sec^2 u \, du = \frac{1}{3} \tan u + C = \frac{1}{3} \tan(3x) + C \)

4. Find \( \int \tan x \, dx \)

Solution:

Rewrite: \( \tan x = \frac{\sin x}{\cos x} \)

Let u = cos x, then du = -sin x dx, so -du = sin x dx

\( \int \frac{\sin x}{\cos x} \, dx = \int \frac{-du}{u} = -\ln|u| + C = -\ln|\cos x| + C = \ln|\sec x| + C \)

5. Find \( \int \csc^2 x \, dx \)

Solution:

Direct integration: \( \int \csc^2 x \, dx = -\cot x + C \)

Check: \( \frac{d}{dx}[-\cot x] = -\left(-\csc^2 x\right) = \csc^2 x \)

6. Find \( \int \cot(4x) \, dx \)

Solution:

Rewrite: \( \cot(4x) = \frac{\cos(4x)}{\sin(4x)} \)

Let u = sin(4x), then du = 4 cos(4x) dx, so cos(4x) dx = du/4

\( \int \frac{\cos(4x)}{\sin(4x)} \, dx = \int \frac{du/4}{u} = \frac{1}{4} \int \frac{du}{u} = \frac{1}{4} \ln|u| + C = \frac{1}{4} \ln|\sin(4x)| + C \)

Key Takeaways

  • \( \int \sin x \, dx = -\cos x + C \), \( \int \cos x \, dx = \sin x + C \)
  • \( \int \sec^2 x \, dx = \tan x + C \), \( \int \csc^2 x \, dx = -\cot x + C \)
  • Use substitution for functions like sin(kx), cos(kx), sec²(kx)
  • Complex trigonometric integrals often require identities and algebraic manipulation
  • Always verify your answers by differentiation
  • Watch for domain restrictions with logarithmic results
