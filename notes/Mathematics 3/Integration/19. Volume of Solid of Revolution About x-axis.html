Volume of Solid of Revolution about x-axis

Volume of Solids of Revolution

A solid of revolution is created by rotating a plane region about an axis. When we rotate about the x-axis, each point in the plane sweeps out a circle, creating a three-dimensional solid. The volume can be found using the method of cylindrical shells or the disk/washer method.

The Disk Method:

When rotating about the x-axis, cross sections perpendicular to the x-axis are circular disks. The volume is the integral of the areas of these disks.

The Disk Method Formula

Basic Case: If the region bounded by y = f(x), x = a, x = b, and the x-axis is rotated about the x-axis, then:

\( V = \pi \int_a^b [f(x)]^2 \, dx \)

Geometric Interpretation:

At each x, the cross section is a circle with radius f(x), so area = π[f(x)]².

The volume element is π[f(x)]² dx.

The Washer Method

If the region is between two curves y = f(x) and y = g(x), with f(x) ≥ g(x):

\( V = \pi \int_a^b ([f(x)]^2 - [g(x)]^2) \, dx \)

This creates a "washer" shape with outer radius f(x) and inner radius g(x).

Examples

Example 1: Basic Disk Method

Find the volume when y = √x from x = 0 to x = 4 is rotated about the x-axis.

Solution:

V = \( \pi \int_0^4 (\sqrt{x})^2 \, dx = \pi \int_0^4 x \, dx = \pi \left[ \frac{x^2}{2} \right]_0^4 = \pi \cdot \frac{16}{2} = 8\pi \)

The volume is 8π cubic units.

Example 2: Washer Method

Find the volume when the region between y = x² and y = √x is rotated about the x-axis.

Solution:

First find intersection: x² = √x → x² - x^{1/2} = 0 → x^{1/2}(x^{3/2} - 1) = 0 → x = 0, 1

On [0, 1]: √x ≥ x², so outer radius = √x, inner radius = x²

V = \( \pi \int_0^1 ((\sqrt{x})^2 - (x^2)^2) \, dx = \pi \int_0^1 (x - x^4) \, dx = \pi \left[ \frac{x^2}{2} - \frac{x^5}{5} \right]_0^1 = \pi \left( \frac{1}{2} - \frac{1}{5} \right) = \pi \cdot \frac{3}{10} = \frac{3\pi}{10} \)

Example 3: Region Above x-axis

Find the volume when y = sin x from x = 0 to x = π is rotated about the x-axis.

Solution:

V = \( \pi \int_0^\pi (\sin x)^2 \, dx = \pi \int_0^\pi \frac{1 - \cos 2x}{2} \, dx = \pi \cdot \frac{1}{2} \int_0^\pi (1 - \cos 2x) \, dx \)

\( = \frac{\pi}{2} \left[ x - \frac{\sin 2x}{2} \right]_0^\pi = \frac{\pi}{2} (\pi - 0) = \frac{\pi^2}{2} \)

The volume is π²/2 cubic units.

Practice Problems

Problem 1

Find the volume when y = x² from x = 0 to x = 2 is rotated about the x-axis.

Solution:

V = \( \pi \int_0^2 (x^2)^2 \, dx = \pi \int_0^2 x^4 \, dx = \pi \left[ \frac{x^5}{5} \right]_0^2 = \pi \cdot \frac{32}{5} = \frac{32\pi}{5} \)

Problem 2

Find the volume when the region between y = x and y = x² from x = 0 to x = 1 is rotated about the x-axis.

Solution:

On [0, 1]: x ≥ x², so outer radius = x, inner radius = x²

V = \( \pi \int_0^1 (x^2 - (x^2)^2) \, dx = \pi \int_0^1 (x^2 - x^4) \, dx = \pi \left[ \frac{x^3}{3} - \frac{x^5}{5} \right]_0^1 = \pi \left( \frac{1}{3} - \frac{1}{5} \right) = \pi \cdot \frac{2}{15} = \frac{2\pi}{15} \)

Problem 3

Find the volume when y = e^{-x} from x = 0 to x = 1 is rotated about the x-axis.

Solution:

V = \( \pi \int_0^1 (e^{-x})^2 \, dx = \pi \int_0^1 e^{-2x} \, dx = \pi \left[ -\frac{1}{2} e^{-2x} \right]_0^1 = \pi \left( -\frac{1}{2} e^{-2} - \left( -\frac{1}{2} \right) \right) = \pi \left( -\frac{1}{2e^2} + \frac{1}{2} \right) = \frac{\pi}{2} \left(1 - \frac{1}{e^2}\right) \)

Key Takeaways

  • Disk method: V = π∫[f(x)]² dx for region above x-axis
  • Washer method: V = π∫([f(x)]² - [g(x)]²) dx for region between curves
  • Cross sections are circles perpendicular to axis of rotation
  • Radius of each disk is the distance from axis to curve
  • Choose appropriate method based on region geometry
  • Sketch the solid to understand the cross sections
  • Integration with respect to x when rotating about x-axis
  • Volume element is π(radius)² dx
