Indefinite Integrals of \( x^\alpha \)

Basic Concepts

Integration is the reverse process of differentiation. The indefinite integral of a function \( f(x) \) is denoted by \( \int f(x) \, dx \) and represents the family of all antiderivatives of \( f(x) \).

Key Notation:
  • \( \int f(x) \, dx = F(x) + C \) where \( F'(x) = f(x) \)
  • \( C \) is the constant of integration
  • The integral represents all functions that differ by a constant

The Power Rule for Integration

Power Rule:

\( \int x^\alpha \, dx = \frac{x^{\alpha+1}}{\alpha+1} + C \) for \( \alpha \neq -1 \)

This is the fundamental rule for integrating power functions.

Special Case (α = -1):

\( \int \frac{1}{x} \, dx = \ln|x| + C \)

When α = -1, the denominator becomes zero, so we use the natural logarithm.

Derivation of the Power Rule

The power rule can be derived from the power rule for differentiation:

  • If \( \frac{d}{dx}[x^n] = n x^{n-1} \), then \( \frac{d}{dx}[x^n] = n x^{n-1} \)
  • To reverse this: if \( \frac{d}{dx}[F(x)] = x^\alpha \), then \( F(x) = \frac{x^{\alpha+1}}{\alpha+1} + C \)
  • Check: \( \frac{d}{dx}\left[\frac{x^{\alpha+1}}{\alpha+1} + C\right] = \frac{(\alpha+1) x^\alpha}{\alpha+1} = x^\alpha \)

Visual Guide

The integral represents the area under the curve. Different values of C shift the entire family of functions vertically.

Examples

Example 1: Basic Power Rule

Find \( \int x^3 \, dx \)

Using the power rule with α = 3:

\( \int x^3 \, dx = \frac{x^{3+1}}{3+1} + C = \frac{x^4}{4} + C \)

Check: \( \frac{d}{dx}\left[\frac{x^4}{4} + C\right] = \frac{4x^3}{4} = x^3 \)

Example 2: Fractional Power

Find \( \int x^{1/2} \, dx \)

Using the power rule with α = 1/2:

\( \int x^{1/2} \, dx = \frac{x^{1/2+1}}{1/2+1} + C = \frac{x^{3/2}}{3/2} + C = \frac{2}{3} x^{3/2} + C \)

Check: \( \frac{d}{dx}\left[\frac{2}{3} x^{3/2} + C\right] = \frac{2}{3} \cdot \frac{3}{2} x^{1/2} = x^{1/2} \)

Example 3: Special Case (α = -1)

Find \( \int \frac{1}{x} \, dx \)

This is the special case where α = -1:

\( \int \frac{1}{x} \, dx = \ln|x| + C \)

Check: \( \frac{d}{dx}[\ln|x| + C] = \frac{1}{x} \)

Note: We use absolute value because the natural log is only defined for positive numbers.

Example 4: Negative Exponent

Find \( \int x^{-2} \, dx \)

Using the power rule with α = -2:

\( \int x^{-2} \, dx = \frac{x^{-2+1}}{-2+1} + C = \frac{x^{-1}}{-1} + C = -\frac{1}{x} + C \)

Check: \( \frac{d}{dx}\left[-\frac{1}{x} + C\right] = \frac{1}{x^2} = x^{-2} \)

Domain Considerations

The domain of the integral depends on the original function:

  • Integer powers: All real numbers (except x = 0 for negative integers)
  • Fractional powers: Depends on the denominator (even roots need non-negative values)
  • Negative powers: Cannot include x = 0 in the domain
  • Logarithmic case: x ≠ 0, and we use |x| for the absolute value

Applications

  • Physics: Work done by variable force, velocity from acceleration
  • Economics: Marginal cost and revenue functions
  • Biology: Population growth models
  • Engineering: Center of mass calculations
  • Geometry: Areas and volumes of shapes

Practice Problems

1. Find \( \int x^5 \, dx \)

Solution:

Using the power rule with α = 5: \( \int x^5 \, dx = \frac{x^{5+1}}{5+1} + C = \frac{x^6}{6} + C \)

Check: \( \frac{d}{dx}\left[\frac{x^6}{6} + C\right] = \frac{6x^5}{6} = x^5 \)

2. Find \( \int \sqrt{x} \, dx \)

Solution:

Write \( \sqrt{x} = x^{1/2} \), so α = 1/2: \( \int x^{1/2} \, dx = \frac{x^{1/2+1}}{1/2+1} + C = \frac{x^{3/2}}{3/2} + C = \frac{2}{3} x^{3/2} + C \)

Check: \( \frac{d}{dx}\left[\frac{2}{3} x^{3/2} + C\right] = \frac{2}{3} \cdot \frac{3}{2} x^{1/2} = x^{1/2} = \sqrt{x} \)

3. Find \( \int x^{-3} \, dx \)

Solution:

Using the power rule with α = -3: \( \int x^{-3} \, dx = \frac{x^{-3+1}}{-3+1} + C = \frac{x^{-2}}{-2} + C = -\frac{1}{2} x^{-2} + C = -\frac{1}{2x^2} + C \)

Check: \( \frac{d}{dx}\left[-\frac{1}{2x^2} + C\right] = -\frac{1}{2} \cdot (-2) x^{-3} = x^{-3} \)

4. Find \( \int \frac{1}{\sqrt{x}} \, dx \)

Solution:

Write \( \frac{1}{\sqrt{x}} = x^{-1/2} \), so α = -1/2: \( \int x^{-1/2} \, dx = \frac{x^{-1/2+1}}{-1/2+1} + C = \frac{x^{1/2}}{1/2} + C = 2 x^{1/2} + C = 2\sqrt{x} + C \)

Check: \( \frac{d}{dx}\left[2\sqrt{x} + C\right] = 2 \cdot \frac{1}{2} x^{-1/2} = x^{-1/2} = \frac{1}{\sqrt{x}} \)

5. Find \( \int x^{2/3} \, dx \)

Solution:

Using the power rule with α = 2/3: \( \int x^{2/3} \, dx = \frac{x^{2/3+1}}{2/3+1} + C = \frac{x^{5/3}}{5/3} + C = \frac{3}{5} x^{5/3} + C \)

Check: \( \frac{d}{dx}\left[\frac{3}{5} x^{5/3} + C\right] = \frac{3}{5} \cdot \frac{5}{3} x^{2/3} = x^{2/3} \)

6. Verify that \( \frac{d}{dx}\left[\frac{x^4}{4} + C\right] = x^3 \)

Solution:

Let F(x) = \( \frac{x^4}{4} + C \)

F'(x) = \( \frac{d}{dx}\left[\frac{x^4}{4} + C\right] = \frac{4x^3}{4} + 0 = x^3 \)

Therefore, \( \frac{d}{dx}\left[\frac{x^4}{4} + C\right] = x^3 \), which confirms that \( \int x^3 \, dx = \frac{x^4}{4} + C \)

Key Takeaways

  • The power rule: \( \int x^\alpha \, dx = \frac{x^{\alpha+1}}{\alpha+1} + C \) for α ≠ -1
  • Special case: \( \int \frac{1}{x} \, dx = \ln|x| + C \)
  • Integration is the reverse of differentiation
  • The constant C represents the family of all antiderivatives
  • Domain restrictions apply for fractional and negative powers
  • Always check your answer by differentiating
