Volume of Solids of Revolution About the y-axis

When rotating a region about the y-axis, we use a different approach than rotation about the x-axis. The cross sections are still circular, but now the radius is measured horizontally from the y-axis to the curve. This often requires expressing the curve as x = g(y) rather than y = f(x).

Key Difference:

Rotation about the y-axis typically requires integration with respect to y, and the radius is the x-coordinate of points on the curve.

The Disk Method for y-axis Rotation

Basic Case: If the region bounded by x = g(y), the y-axis, and y = c, y = d is rotated about the y-axis, then:

\( V = \pi \int_c^d [g(y)]^2 \, dy \)

Geometric Interpretation:

At each y, the cross section is a circle with radius g(y) (the x-coordinate), so area = π[g(y)]².

The volume element is π[g(y)]² dy.

The Washer Method for y-axis

If the region is between two curves x = f(y) and x = h(y), with f(y) ≥ h(y):

\( V = \pi \int_c^d ([f(y)]^2 - [h(y)]^2) \, dy \)

This creates a "washer" with outer radius f(y) and inner radius h(y).

Examples

Example 1: Basic Disk Method

Find the volume when x = √y from y = 0 to y = 4 is rotated about the y-axis.

Solution:

The curve x = √y is already expressed in terms of y. The region bounded by x = √y, y-axis, and y = 0 to 4.

V = \( \pi \int_0^4 (\sqrt{y})^2 \, dy = \pi \int_0^4 y \, dy = \pi \left[ \frac{y^2}{2} \right]_0^4 = \pi \cdot \frac{16}{2} = 8\pi \)

The volume is 8π cubic units.

Example 2: Converting from y = f(x) to x = g(y)

Find the volume when y = x² from x = 0 to x = 2 is rotated about the y-axis.

Solution:

First, express in terms of y: x = √y (since y = x², x ≥ 0).

The region goes from y = 0 to y = 4 (when x = 2, y = 4).

V = \( \pi \int_0^4 (\sqrt{y})^2 \, dy = \pi \int_0^4 y \, dy = \pi \left[ \frac{y^2}{2} \right]_0^4 = \pi \cdot \frac{16}{2} = 8\pi \)

The volume is 8π cubic units.

Example 3: Washer Method

Find the volume when the region between x = y² and x = √y is rotated about the y-axis.

Solution:

First find intersection: y² = √y → y⁴ = y → y⁴ - y = 0 → y(y³ - 1) = 0 → y = 0, 1

On [0, 1]: √y ≥ y², so outer radius = √y, inner radius = y²

V = \( \pi \int_0^1 ((\sqrt{y})^2 - (y^2)^2) \, dy = \pi \int_0^1 (y - y^4) \, dy = \pi \left[ \frac{y^2}{2} - \frac{y^5}{5} \right]_0^1 = \pi \left( \frac{1}{2} - \frac{1}{5} \right) = \pi \cdot \frac{3}{10} = \frac{3\pi}{10} \)

Practice Problems

Problem 1

Find the volume when x = y/2 from y = 0 to y = 4 is rotated about the y-axis.

Solution:

V = \( \pi \int_0^4 \left(\frac{y}{2}\right)^2 \, dy = \pi \int_0^4 \frac{y^2}{4} \, dy = \frac{\pi}{4} \left[ \frac{y^3}{3} \right]_0^4 = \frac{\pi}{4} \cdot \frac{64}{3} = \frac{16\pi}{3} \)

Problem 2

Find the volume when y = √x from x = 0 to x = 4 is rotated about the y-axis.

Solution:

Express in terms of y: x = y², from y = 0 to y = 2

V = \( \pi \int_0^2 (y^2)^2 \, dy = \pi \int_0^2 y^4 \, dy = \pi \left[ \frac{y^5}{5} \right]_0^2 = \pi \cdot \frac{32}{5} = \frac{32\pi}{5} \)

Problem 3

Find the volume when the region between x = y² and x = 2 - y² from y = -1 to y = 1 is rotated about the y-axis.

Solution:

On [-1, 1]: 2 - y² ≥ y², so outer radius = √(2 - y²), inner radius = |y|

Actually, x = 2 - y² and x = y² intersect at y = ±1

V = \( \pi \int_{-1}^1 ((2 - y^2)^2 - (y^2)^2) \, dy = \pi \int_{-1}^1 (4 - 4y^2 + y^4 - y^4) \, dy = \pi \int_{-1}^1 (4 - 4y^2) \, dy \)

\( = \pi \left[ 4y - \frac{4y^3}{3} \right]_{-1}^1 = \pi \left( (4 - \frac{4}{3}) - (-4 + \frac{4}{3}) \right) = \pi \left( \frac{8}{3} - (-\frac{8}{3}) \right) = \pi \cdot \frac{16}{3} = \frac{16\pi}{3} \)

Key Takeaways

  • Disk method: V = π∫[g(y)]² dy where x = g(y) is the distance from y-axis
  • Washer method: V = π∫([f(y)]² - [h(y)]²) dy for regions between curves
  • Often need to express curves as x = g(y) rather than y = f(x)
  • Integration is with respect to y when rotating about y-axis
  • Cross sections are circles perpendicular to y-axis
  • Radius equals the x-coordinate of points on the curve
  • Sketch the region and determine which curve is farther from y-axis
  • Use symmetry when possible to simplify calculations
