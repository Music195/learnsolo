Velocity and Acceleration (Straight Line)

Fundamental Motion Relationships

When an object moves along a straight line, position, velocity, and acceleration are connected through derivatives:

$x(t) \xrightarrow{\frac{d}{dt}} v(t) \xrightarrow{\frac{d}{dt}} a(t)$
$\text{Position} \to \text{Velocity} \to \text{Acceleration}$

Where:

  • $v(t) = \frac{dx}{dt} = x'(t)$ - rate of position change
  • $a(t) = \frac{dv}{dt} = \frac{d^2x}{dt^2} = x''(t)$ - rate of velocity change

Mathematical Foundations

Straight line motion analysis uses calculus to describe how objects move, change speed, and change direction. The relationships between position, velocity, and acceleration form the foundation of kinematics.

Sign Interpretation and Motion Analysis

Velocity Signs

  • v(t) > 0: Moving right/forward/up
  • v(t) < 0: Moving left/backward/down
  • v(t) = 0: Instantaneously at rest

Acceleration Signs

  • a(t) > 0: Speeding up (if v > 0) or slowing down (if v < 0)
  • a(t) < 0: Slowing down (if v > 0) or speeding up (if v < 0)
  • a(t) = 0: Constant velocity

When is the Object Speeding Up or Slowing Down?

The key is to check if velocity and acceleration have the same sign:

Velocity Acceleration Same Sign? Result
v > 0 (moving right) a > 0 (pushing right) Yes Speeding up
v > 0 (moving right) a < 0 (pushing left) No Slowing down
v < 0 (moving left) a < 0 (pushing left) Yes Speeding up
v < 0 (moving left) a > 0 (pushing right) No Slowing down

Rule: Speeding up when v and a have the same sign; slowing down when they have opposite signs.

Worked Examples

Example 1: Complete Motion Analysis

A particle moves with position $x(t) = t^3 - 6t^2 + 9t$. Find velocity, acceleration, and analyze motion.

Step 1: Velocity: v(t) = 3t² - 12t + 9 = 3(t - 1)(t - 3)

Step 2: Acceleration: a(t) = 6t - 12 = 6(t - 2)

Step 3: Critical points: v(t) = 0 ⇒ t = 1, 3

Step 4: Motion intervals:

  • t < 1: v > 0 (right), a < 0 (opposite) → slowing down
  • 1 < t < 2: v < 0 (left), a < 0 (same) → speeding up
  • 2 < t < 3: v < 0 (left), a > 0 (opposite) → slowing down
  • t > 3: v > 0 (right), a > 0 (same) → speeding up

Example 2: Projectile Motion

Ball thrown upward: h(t) = 20t - 5t² m. Find max height and impact time.

Step 1: Velocity: v(t) = 20 - 10t

Step 2: Acceleration: a(t) = -10 m/s² (gravity)

Step 3: Max height when v(t) = 0: 20 - 10t = 0 ⇒ t = 2 s

Step 4: Max height: h(2) = 20(2) - 5(4) = 40 - 20 = 20 m

Step 5: Ground impact: h(t) = 0 ⇒ 20t - 5t² = 0 ⇒ t(20 - 5t) = 0 ⇒ t = 4 s

Example 3: Trigonometric Motion

Find velocity at t = 2 for x(t) = sin(2t).

Step 1: v(t) = 2cos(2t)

Step 2: v(2) = 2cos(4) ≈ 2(-0.654) ≈ -1.31 units/time

Step 3: Negative velocity means moving in negative direction

Practice Problems

Problem 1

A particle moves with x(t) = t⁴ - 4t³. Find when acceleration is zero and analyze motion.

Step 1: v(t) = 4t³ - 12t², a(t) = 12t² - 24t = 12t(t - 2)

Step 2: a(t) = 0 ⇒ t = 0, 2

Step 3: At these times, velocity is constant (not changing)

Step 4: Motion analysis: Initially slowing down, then speeding up

Problem 2

Car moves with x(t) = 10 + 5t - t². When is it speeding up?

Step 1: v(t) = 5 - 2t, a(t) = -2

Step 2: Constant deceleration

Step 3: Speeding up when v and a same sign: v < 0 and a < 0

Step 4: v < 0 when 5 - 2t < 0 ⇒ t > 2.5

Step 5: Speeding up for t > 2.5 seconds

Problem 3

Object moves with x(t) = e^{-t}sin(t). Find when velocity and acceleration are zero.

Step 1: v(t) = e^{-t}cos(t) - e^{-t}sin(t) = e^{-t}(cos(t) - sin(t))

Step 2: a(t) = -e^{-t}(cos(t) - sin(t)) + e^{-t}(-sin(t) - cos(t)) = -e^{-t}(2cos(t))

Step 3: v(t) = 0 ⇒ cos(t) = sin(t) ⇒ t = π/4 + nπ

Step 4: a(t) = 0 ⇒ cos(t) = 0 ⇒ t = π/2 + nπ

Problem 4

Find total distance traveled by x(t) = t³ - 6t² + 9t from t=0 to t=4.

Step 1: v(t) = 3t² - 12t + 9, changes sign at t=1,3

Step 2: Intervals: [0,1] right, [1,3] left, [3,4] right

Step 3: Distance = ∫|v(t)|dt from 0 to 4

Step 4: = ∫v(t)dt from 0 to 1 + ∫(-v(t))dt from 1 to 3 + ∫v(t)dt from 3 to 4

Step 5: = [t³-6t²+9t] from 0 to 1 + [-t³+6t²-9t] from 1 to 3 + [t³-6t²+9t] from 3 to 4

Step 6: = 4 + (18-4) + (16-36) = 4 + 14 - 20 = -2 units

Key Takeaways

  • Relationships: Position → Velocity → Acceleration (via differentiation)
  • Velocity: v(t) = dx/dt, includes direction, can be negative
  • Acceleration: a(t) = dv/dt = d²x/dt², rate of velocity change
  • Speed vs Velocity: Speed = |v(t)|, always ≥ 0
  • Direction: v > 0 means moving in positive direction
  • Speeding up: When v and a have the same sign
  • Slowing down: When v and a have opposite signs
  • Constant speed: When a = 0
  • Displacement: x(t₂) - x(t₁), can be zero even with motion
  • Total distance: ∫|v(t)|dt, always ≥ 0
