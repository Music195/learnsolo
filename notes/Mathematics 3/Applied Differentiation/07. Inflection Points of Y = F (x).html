Inflection Points of y = f(x)

Definition of Inflection Points

A point $(a, f(a))$ on the curve $y = f(x)$ is called an inflection point if the concavity of $f$ changes at $x = a$.

The curve changes from:

  • Concave upward (∪) to concave downward (∩), or
  • Concave downward (∩) to concave upward (∪)

Necessary condition: If $f''(a)$ exists, then $f''(a) = 0$

Mathematical Foundations

Inflection points mark the transition between concave upward and concave downward regions of a function. They are critical for understanding curve behavior and occur where the second derivative is zero or undefined.

Finding Inflection Points: Step-by-Step

Algorithm for Finding Inflection Points

  1. Compute the second derivative f''(x)
  2. Find all points where f''(x) = 0
  3. Check for points where f''(x) is undefined
  4. For each candidate point, test concavity on both sides
  5. Verify that concavity actually changes at the point

Testing Concavity Change

For a candidate point c:

  • Check f''(x) for x < c and x > c
  • If signs are different, c is an inflection point
  • If signs are the same, c is not an inflection point
  • Be careful with points where f''(x) changes from zero to non-zero

Worked Examples

Example 1: Cubic Function

Find inflection points of $f(x) = x^3 - 3x^2 + 2$.

Step 1: f'(x) = 3x² - 6x, f''(x) = 6x - 6 = 6(x - 1)

Step 2: f''(x) = 0 ⇒ x = 1

Step 3: Test intervals: x < 1 and x > 1

Step 4: x < 1: f''(x) < 0 (concave down)

Step 5: x > 1: f''(x) > 0 (concave up)

Step 6: Concavity changes at x = 1, so it's an inflection point

Example 2: Quartic Function

Find inflection points of $f(x) = x^4 - 2x^2$.

Step 1: f'(x) = 4x³ - 4x, f''(x) = 12x² - 4

Step 2: 12x² - 4 = 0 ⇒ x² = 1/3 ⇒ x = ±1/√3

Step 3: Test intervals: (-∞, -1/√3), (-1/√3, 1/√3), (1/√3, ∞)

Step 4: |x| < 1/√3: f''(x) < 0 (concave down)

Step 5: |x| > 1/√3: f''(x) > 0 (concave up)

Step 6: Two inflection points at x = ±1/√3

Example 3: Trigonometric Function

Find inflection points of $f(x) = \sin(x)$.

Step 1: f'(x) = cos(x), f''(x) = -sin(x)

Step 2: -sin(x) = 0 ⇒ sin(x) = 0 ⇒ x = nπ

Step 3: Test x = 0: Left of 0 (x = -0.1): -sin(-0.1) > 0 (concave up)

Step 4: Right of 0 (x = 0.1): -sin(0.1) < 0 (concave down)

Step 5: Concavity changes, so x = 0 is an inflection point

Step 6: Similarly for x = π, 2π, etc.

Practice Problems

Problem 1

Find all inflection points of $f(x) = x^3 + x^2 - 2x + 1$.

Step 1: f'(x) = 3x² + 2x - 2, f''(x) = 6x + 2

Step 2: 6x + 2 = 0 ⇒ x = -1/3

Step 3: Test intervals: x < -1/3 and x > -1/3

Step 4: x < -1/3: f''(x) < 0 (concave down)

Step 5: x > -1/3: f''(x) > 0 (concave up)

Step 6: Inflection point at x = -1/3

Problem 2

Find inflection points of $f(x) = e^{-x^2}$.

Step 1: f'(x) = -2xe^{-x^2}, f''(x) = -2e^{-x^2} + 4x²e^{-x^2} = 2e^{-x^2}(2x² - 1)

Step 2: 2x² - 1 = 0 ⇒ x = ±1/√2

Step 3: Test intervals: (-∞, -1/√2), (-1/√2, 1/√2), (1/√2, ∞)

Step 4: |x| < 1/√2: f''(x) < 0 (concave down)

Step 5: |x| > 1/√2: f''(x) > 0 (concave up)

Step 6: Inflection points at x = ±1/√2

Problem 3

Determine if $f(x) = x^4$ has any inflection points.

Step 1: f'(x) = 4x³, f''(x) = 12x²

Step 2: 12x² = 0 ⇒ x = 0

Step 3: Test intervals: x < 0 and x > 0

Step 4: x < 0: f''(x) > 0 (concave up)

Step 5: x > 0: f''(x) > 0 (concave up)

Step 6: No change in concavity, so x = 0 is not an inflection point

Problem 4

Find inflection points of $f(x) = \ln(x)$ for x > 0.

Step 1: f'(x) = 1/x, f''(x) = -1/x²

Step 2: f''(x) is never zero (always negative)

Step 3: No points where f''(x) = 0

Step 4: No inflection points

Note: The function is always concave down

Key Takeaways

  • Definition: Point where concavity changes from ∪ to ∩ or vice versa
  • Necessary condition: f''(a) = 0 (if it exists)
  • Not sufficient: Must verify concavity actually changes
  • Finding method: Solve f''(x) = 0, test intervals
  • Geometric meaning: Curve changes bending direction
  • Applications: Curve sketching, design, physics
  • Multiple points: Functions can have multiple inflection points
  • Horizontal inflection: Rare case where tangent is horizontal
