Understanding Rational Functions

A rational function is a function that can be expressed as the ratio of two polynomials: $f(x) = \dfrac{P(x)}{Q(x)}$ where P(x) and Q(x) are polynomials and Q(x) ≠ 0.

Basic Form: $y = \dfrac{k}{x - p} + q$

This is a simple rational function that can be graphed by transforming the basic hyperbola $y = \dfrac{k}{x}$ through horizontal and vertical shifts.

Transformations of Rational Functions

Horizontal Shift

The graph of $y = \dfrac{k}{x - p} + q$ is the graph of $y = \dfrac{k}{x} + q$ shifted p units to the right if p > 0, or |p| units to the left if p < 0.

Vertical Shift

The graph of $y = \dfrac{k}{x - p} + q$ is the graph of $y = \dfrac{k}{x - p}$ shifted q units up if q > 0, or |q| units down if q < 0.

Scaling Factor k

The parameter k affects the "steepness" of the branches:

  • If |k| > 1, the branches are steeper than the basic hyperbola
  • If |k| < 1, the branches are flatter than the basic hyperbola
  • If k < 0, the branches open downward instead of upward

Asymptotes, Domain, and Range

Vertical Asymptote

$x = p$ is a vertical asymptote. The function approaches ±∞ as x approaches p from either side.

Horizontal Asymptote

$y = q$ is a horizontal asymptote. The function approaches q as x approaches ±∞.

Domain

All real numbers except x = p: $x \in \mathbb{R}, x \neq p$

Range

All real numbers except y = q: $y \in \mathbb{R}, y \neq q$

Examples

Example 1: Basic Transformation

Sketch the graph of $y = \dfrac{1}{x - 2} + 3$
  • Start with $y = \dfrac{1}{x}$
  • Horizontal shift: 2 units right → $y = \dfrac{1}{x - 2}$
  • Vertical shift: 3 units up → $y = \dfrac{1}{x - 2} + 3$
  • Vertical asymptote: x = 2
  • Horizontal asymptote: y = 3

Example 2: Negative k Value

Sketch the graph of $y = \dfrac{-2}{x + 1} - 1$
  • Start with $y = \dfrac{1}{x}$
  • Scale by k = -2: steeper and opens downward → $y = \dfrac{-2}{x}$
  • Horizontal shift: 1 unit left → $y = \dfrac{-2}{x + 1}$
  • Vertical shift: 1 unit down → $y = \dfrac{-2}{x + 1} - 1$
  • Vertical asymptote: x = -1
  • Horizontal asymptote: y = -1

Example 3: Finding Equation from Graph

Find the equation of a rational function with vertical asymptote x = 3, horizontal asymptote y = -2, and passes through (4, 0).

The general form is $y = \dfrac{k}{x - 3} - 2$

Substitute point (4, 0): $0 = \dfrac{k}{4 - 3} - 2$

$0 = \dfrac{k}{1} - 2$ → $k = 2$

Equation: $y = \dfrac{2}{x - 3} - 2$

Example 4: Behavior Near Asymptotes

Describe the behavior of $y = \dfrac{3}{x - 1} + 2$ as x approaches 1 from the left and right.
  • As x → 1⁺ (from the right): numerator is positive, denominator approaches 0⁺, so y → +∞
  • As x → 1⁻ (from the left): numerator is positive, denominator approaches 0⁻, so y → -∞
  • As x → ±∞: y → 2 (horizontal asymptote)

Step-by-Step Graphing Strategy

Step 1: Identify Parameters

For $y = \dfrac{k}{x - p} + q$, identify k (scaling), p (horizontal shift), and q (vertical shift).

Step 2: Find Asymptotes

Vertical asymptote: x = p
Horizontal asymptote: y = q

Step 3: Determine Behavior

Check if the branches approach the asymptotes from above or below, and the steepness based on |k|.

Step 4: Plot Key Points

Plot points on either side of the vertical asymptote to determine the shape.

Step 5: Sketch the Graph

Draw smooth curves approaching the asymptotes but never touching them.

Applications

Physics: Inverse Square Law

Gravitational force F = GMm/r² and electric force F = kq₁q₂/r² follow inverse square relationships, which are rational functions.

Economics: Cost Analysis

Average cost functions often have rational forms, with vertical asymptotes representing minimum production levels.

Chemistry: pH and Concentration

pH = -log₁₀[H⁺] creates logarithmic relationships, and concentration vs. time in chemical reactions often follows rational functions.

Biology: Population Dynamics

Logistic growth models and predator-prey relationships can be expressed using rational functions.

Practice Problems

Problem 1

Sketch the graph of $y = \dfrac{1}{x + 3} - 2$ and identify the asymptotes.

Show Solution

Vertical asymptote: x = -3
Horizontal asymptote: y = -2
The graph is shifted 3 units left and 2 units down from y = 1/x.

Problem 2

Find the equation of a rational function with vertical asymptote x = 1, horizontal asymptote y = 4, and passes through (2, 6).

Show Solution

General form: $y = \dfrac{k}{x - 1} + 4$
Substitute (2, 6): $6 = \dfrac{k}{2 - 1} + 4$
$6 = k + 4$ → $k = 2$
Equation: $y = \dfrac{2}{x - 1} + 4$

Problem 3

Describe the transformations needed to get from $y = \dfrac{1}{x}$ to $y = \dfrac{-3}{x - 2} + 1$.

Show Solution

1. Scale by k = -3 (steeper, opens downward)
2. Shift 2 units right
3. Shift 1 unit up

Problem 4

What is the domain and range of $y = \dfrac{2}{x + 5} - 3$?

Show Solution

Domain: all real numbers except x = -5
Range: all real numbers except y = -3

Key Takeaways

Transformation Order

Always perform horizontal shifts first, then vertical shifts when transforming rational functions.

Asymptote Rules

Vertical asymptote at x = p, horizontal asymptote at y = q. The graph approaches but never touches these lines.

Behavior Near Asymptotes

The sign of k determines which side of the vertical asymptote the branches approach the horizontal asymptote.

Domain and Range

Domain excludes the vertical asymptote; range excludes the horizontal asymptote.

Real-World Connections

Rational functions model many physical phenomena, especially those involving inverse relationships.
