How to Find the Inverse Function of $f(x)$

Basic Concepts

The inverse of a function $f(x)$ is denoted as $f^{-1}(x)$ and represents the function that "undoes" what $f(x)$ does. If $f(a) = b$, then $f^{-1}(b) = a$.

Key Properties:
  • The domain of $f^{-1}(x)$ equals the range of $f(x)$
  • The range of $f^{-1}(x)$ equals the domain of $f(x)$
  • $f^{-1}(f(x)) = x$ for all $x$ in the domain of $f$
  • $f(f^{-1}(x)) = x$ for all $x$ in the domain of $f^{-1}$

Step-by-Step Procedure

  1. Replace $f(x)$ with $y$: Write $y = f(x)$
  2. Solve for $x$: Solve the equation $y = f(x)$ for $x$ in terms of $y$
  3. Swap variables: Interchange $x$ and $y$ to get $y = g(x)$
  4. Verify the inverse: Check that $f(f^{-1}(x)) = x$ and $f^{-1}(f(x)) = x$
  5. State domain and range: Determine the appropriate domain and range

Visual Guide

The graph of $f^{-1}(x)$ is the reflection of $f(x)$ over the line $y = x$.

Examples

Example 1: Linear Function

Find the inverse of $f(x) = 2x + 3$

1. Write: $y = 2x + 3$

2. Solve for $x$: $x = \frac{y - 3}{2}$

3. Swap: $y = \frac{x - 3}{2}$

4. Thus: $f^{-1}(x) = \frac{x - 3}{2}$

Example 2: Quadratic Function (One-to-One)

Find the inverse of $f(x) = x^2 + 4$ for $x \geq 0$

1. Write: $y = x^2 + 4$

2. Solve for $x$: $x = \sqrt{y - 4}$ (taking positive root)

3. Swap: $y = \sqrt{x - 4}$

4. Thus: $f^{-1}(x) = \sqrt{x - 4}$

Domain: $x \geq 4$, Range: $y \geq 0$

Example 3: Rational Function

Find the inverse of $f(x) = \frac{x + 1}{x - 2}$

1. Write: $y = \frac{x + 1}{x - 2}$

2. Solve for $x$: $y(x - 2) = x + 1 \Rightarrow yx - 2y = x + 1 \Rightarrow yx - x = 2y + 1 \Rightarrow x(y - 1) = 2y + 1 \Rightarrow x = \frac{2y + 1}{y - 1}$

3. Swap: $y = \frac{2x + 1}{x - 1}$

4. Thus: $f^{-1}(x) = \frac{2x + 1}{x - 1}$

Domain: $x \neq 1$, Range: $y \neq 2$

Domain and Range Considerations

When finding inverse functions:

  • Restrict domains: For functions that are not one-to-one, restrict the domain to make them invertible
  • Check vertical line test: Original function must pass vertical line test
  • Check horizontal line test: Original function must pass horizontal line test for invertibility
  • Exclude asymptotes: Be careful with vertical and horizontal asymptotes

Applications

  • Physics: Converting between different units (Celsius to Fahrenheit)
  • Finance: Compound interest calculations
  • Computer Science: Encoding/decoding functions
  • Engineering: Signal processing and transformations
  • Real-world: Temperature conversions, currency exchange rates

Practice Problems

1. Find the inverse of $f(x) = 3x - 7$

2. Find the inverse of $f(x) = \frac{1}{x + 2}$

3. Find the inverse of $f(x) = e^x$ (exponential function)

4. Find the inverse of $f(x) = \ln(x)$ for $x > 0$ (logarithmic function)

5. Determine if $f(x) = x^2$ has an inverse. If not, how can you make it invertible?

Key Takeaways

  • The inverse function "undoes" the original function
  • Graphs of inverses are reflections over the line $y = x$
  • Not all functions have inverses; they must be one-to-one
  • Domain and range swap between a function and its inverse
  • Always verify your inverse by checking composition
  • Restrict domains when necessary to ensure invertibility
