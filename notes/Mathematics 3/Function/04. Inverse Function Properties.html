Inverse Function Properties

Basic Properties

Property 1: Inverse Relationship

If $f(a) = b$, then $f^{-1}(b) = a$

This means the inverse function "undoes" what the original function does.

Property 2: Symmetry

The graphs of $f(x)$ and $f^{-1}(x)$ are symmetric with respect to the line $y = x$.

The inverse function is the reflection of the original function over the diagonal line.

Composition Properties

The most important properties involve function composition:

Property 3: Identity with Original Function

$f^{-1}(f(x)) = x$ for all $x$ in the domain of $f$

$f(f^{-1}(x)) = x$ for all $x$ in the domain of $f^{-1}$

These compositions result in the identity function.

Property 4: Double Inverse

$(f^{-1})^{-1} = f$

The inverse of the inverse function is the original function.

Domain and Range Properties

Property 5: Domain and Range Swap
  • Domain of $f^{-1} =$ Range of $f$
  • Range of $f^{-1} =$ Domain of $f$
Property 6: One-to-One Requirement

A function has an inverse if and only if it is one-to-one (injective).

This means no two different inputs produce the same output.

Graphical Properties

The graph of an inverse function can be obtained by:

  • Reflection: Reflecting the graph of $f(x)$ over the line $y = x$
  • Switching coordinates: Interchanging $x$ and $y$ coordinates of points
  • Horizontal line test: A function has an inverse if it passes the horizontal line test

Algebraic Properties

Property 7: Inverse of Linear Functions

If $f(x) = ax + b$ where $a \neq 0$, then $f^{-1}(x) = \frac{x - b}{a}$

The slope becomes $1/a$ and the intercept changes sign and divides by $a$.

Property 8: Inverse of Exponential and Logarithmic Functions
  • $(\ln x)' = \frac{1}{x}$ and $(\log_a x)' = \frac{1}{x \ln a}$
  • $(e^x)' = e^x$ and $(a^x)' = a^x \ln a$
  • $\ln(e^x) = x$ and $e^{\ln x} = x$

Examples

Example 1: Verifying Inverse Properties

Given $f(x) = 2x + 1$ and $f^{-1}(x) = \frac{x - 1}{2}$, verify the composition properties:

$f^{-1}(f(x)) = f^{-1}(2x + 1) = \frac{(2x + 1) - 1}{2} = \frac{2x}{2} = x$ ✓

$f(f^{-1}(x)) = f(\frac{x - 1}{2}) = 2(\frac{x - 1}{2}) + 1 = x - 1 + 1 = x$ ✓

Example 2: Domain and Range

For $f(x) = e^x$:

  • Domain of $f$: $(-\infty, \infty)$
  • Range of $f$: $(0, \infty)$
  • Domain of $f^{-1} = \ln x$: $(0, \infty)$
  • Range of $f^{-1} = \ln x$: $(-\infty, \infty)$

Applications

  • Cryptography: Encryption/decryption functions are inverses
  • Physics: Converting between different units (temperature scales)
  • Finance: Compound interest and present value calculations
  • Computer Science: Hash functions and their inverses
  • Engineering: Signal processing and filter design
  • Medicine: Dose-response relationships and their inverses

Practice Problems

1. Verify that $f(x) = 3x - 2$ and $g(x) = \frac{x + 2}{3}$ are inverses.

2. Find the domain and range of $f(x) = \sqrt{x - 1}$ and its inverse.

3. Show that $f(x) = x^2$ (for $x \geq 0$) has an inverse, but $g(x) = x^2$ (for all real $x$) does not.

4. Verify the composition properties for $f(x) = e^x$ and $f^{-1}(x) = \ln x$.

5. Graph $f(x) = |x|$ and explain why it doesn't have an inverse.

Key Takeaways

  • Inverse functions "undo" each other: $f^{-1}(f(x)) = x$
  • Graphs are symmetric over the line $y = x$
  • Domain and range swap between function and inverse
  • Only one-to-one functions have inverses
  • The inverse of an inverse is the original function
  • Composition with inverse yields the identity function
  • Linear functions always have inverses (if slope ≠ 0)
