Understanding Irrational Functions

An irrational function contains radicals (square roots, cube roots, etc.) in its definition. The most common type involves square roots of linear expressions: $y = \sqrt{a(x - p)} + q$.

Basic Form: $y = \sqrt{a(x - p)} + q$

This function can be graphed by transforming the basic square root function $y = \sqrt{x}$ through horizontal/vertical shifts and scaling/stretching.

Transformations of Square Root Functions

Horizontal Shift

The graph of $y = \sqrt{a(x - p)} + q$ is the graph of $y = \sqrt{a x} + q$ shifted p units to the right if p > 0, or |p| units to the left if p < 0.

Vertical Shift

The graph of $y = \sqrt{a(x - p)} + q$ is the graph of $y = \sqrt{a(x - p)}$ shifted q units up if q > 0, or |q| units down if q < 0.

Scaling Factor a

The parameter a affects the "width" of the parabola:

  • If |a| > 1, the graph is narrower than the basic square root
  • If |a| < 1, the graph is wider than the basic square root
  • If a < 0, the graph opens to the left instead of the right

Domain and Range

Domain

The domain is determined by the expression inside the square root: $a(x - p) \geq 0$

  • If a > 0: x ≥ p (starts at p and goes right)
  • If a < 0: x ≤ p (starts at p and goes left)

Range

The range depends on the vertical shift q:

  • If q ≥ 0: y ≥ q (starts at q and goes up)
  • If q < 0: y ≥ q (starts at q and goes up)

Square root functions always have y ≥ q, where q is the vertical shift.

Examples

Example 1: Basic Transformation

Sketch the graph of $y = \sqrt{x - 2} + 3$
  • Start with $y = \sqrt{x}$
  • Horizontal shift: 2 units right → $y = \sqrt{x - 2}$
  • Vertical shift: 3 units up → $y = \sqrt{x - 2} + 3$
  • Domain: x ≥ 2
  • Range: y ≥ 3

Example 2: Negative Scaling Factor

Sketch the graph of $y = \sqrt{-2(x + 1)} - 1$
  • Start with $y = \sqrt{x}$
  • Scale by a = -2: narrower and opens leftward → $y = \sqrt{-2x}$
  • Horizontal shift: 1 unit left → $y = \sqrt{-2(x + 1)}$
  • Vertical shift: 1 unit down → $y = \sqrt{-2(x + 1)} - 1$
  • Domain: -2(x + 1) ≥ 0 → x + 1 ≤ 0 → x ≤ -1
  • Range: y ≥ -1

Example 3: Finding the Equation

Find the equation of a square root function that passes through (3, 5) and (7, 7), and has its vertex at (3, 5).

The vertex is at (3, 5), so the equation is $y = a\sqrt{x - 3} + 5$

Substitute point (7, 7): $7 = a\sqrt{7 - 3} + 5$

$7 = a\sqrt{4} + 5$ → $7 = 2a + 5$ → $2a = 2$ → $a = 1$

Equation: $y = \sqrt{x - 3} + 5$

Example 4: Domain and Range Analysis

Find the domain and range of $y = \sqrt{3(x - 4)} + 2$
  • Domain: 3(x - 4) ≥ 0 → x - 4 ≥ 0 → x ≥ 4
  • Range: y ≥ 2 (since the square root is ≥ 0, plus the vertical shift of 2)

Example 5: Cube Root Function

Sketch the graph of $y = \sqrt[3]{x - 1} + 2$

Cube root functions have domain all real numbers and range all real numbers.

The graph is shifted 1 unit right and 2 units up from $y = \sqrt[3]{x}$.

Unlike square roots, cube roots can accept negative inputs and produce negative outputs.

Step-by-Step Graphing Strategy

Step 1: Identify Parameters

For $y = \sqrt{a(x - p)} + q$, identify a (scaling), p (horizontal shift), and q (vertical shift).

Step 2: Determine Domain

Set a(x - p) ≥ 0 and solve for x to find the domain.

Step 3: Find Range

The range is y ≥ q, where q is the vertical shift.

Step 4: Plot Key Points

Plot the vertex (p, q) and several points to determine the shape and direction.

Step 5: Sketch the Graph

Draw a smooth curve starting from the vertex in the appropriate direction based on the sign of a.

Applications

Physics: Projectile Motion with Air Resistance

The path of a projectile with air resistance often follows a square root relationship near the peak of its trajectory.

Biology: Population Growth Models

Some population models use square root functions to describe growth rates that depend on population density.

Engineering: Signal Processing

Square root functions appear in signal amplitude calculations and filter design.

Finance: Option Pricing Models

The Black-Scholes model for option pricing involves square root terms in volatility calculations.

Geometry: Distance and Area Problems

Many geometric optimization problems involve square root functions, such as minimizing distance or maximizing area.

Key Takeaways

Domain Restriction

The expression inside the square root must be non-negative: a(x - p) ≥ 0. This determines whether the graph opens right or left.

Range Always Positive

Square root functions always produce non-negative values, so the range is always y ≥ q where q is the vertical shift.

Vertex is Key Point

The vertex (p, q) is the starting point of the graph and helps determine the direction and shape.

Scaling Effects

The parameter a controls both the width (steeper = narrower) and the direction (negative = leftward) of the graph.

Cube Roots vs Square Roots

Cube root functions have no domain restrictions and can produce negative outputs, unlike square roots.
