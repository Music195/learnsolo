Quadratic Graph and Intersection with x-axis

Mathematical Foundations

The intersection points between a quadratic graph and the x-axis correspond to the real solutions of the quadratic equation. The x-axis represents y = 0, so finding intersections means solving the equation ax² + bx + c = 0. The number and nature of these intersections depend on the discriminant and the direction of the parabola.

One Intersection

Worked Examples

Example 1: Two Intersection Points

Graph: $y = x^2 - 5x + 6$

Find x-intercepts by solving: $x^2 - 5x + 6 = 0$

$(x-2)(x-3) = 0$ → x = 2 or x = 3

The parabola crosses the x-axis at (2, 0) and (3, 0).

Example 2: One Intersection Point (Tangent)

Graph: $y = x^2 - 4x + 4$

Find x-intercepts by solving: $x^2 - 4x + 4 = 0$

$(x-2)^2 = 0$ → x = 2 (repeated)

The parabola touches the x-axis at (2, 0).

Example 3: No Intersection Points

Graph: $y = x^2 + 2x + 3$

Find x-intercepts by solving: $x^2 + 2x + 3 = 0$

Discriminant: 4 - 12 = -8 < 0

The parabola never crosses the x-axis.

Practice Problems

Problem 1

How many times does $y = x^2 - 7x + 12$ intersect the x-axis?

$D = 49 - 48 = 1 > 0$ → Two intersection points

Problem 2

How many times does $y = 2x^2 - 4x + 2$ intersect the x-axis?

$D = 16 - 16 = 0$ → One intersection point (tangent)

Problem 3

How many times does $y = x^2 + x + 1$ intersect the x-axis?

$D = 1 - 4 = -3 < 0$ → No intersection points

Key Takeaways

  • The x-intercepts of a quadratic graph are the solutions to the equation ax² + bx + c = 0
  • The number of x-intercepts equals the number of real solutions
  • Two x-intercepts when D > 0 (parabola crosses x-axis twice)
  • One x-intercept when D = 0 (parabola touches x-axis)
  • No x-intercepts when D < 0 (parabola never touches x-axis)
  • The sign of 'a' doesn't affect the number of intersections, only the direction
  • X-intercepts occur where y = 0, representing the roots of the quadratic equation
