Mathematical Foundations

When solving quadratic inequalities of the form $ax^2 + bx + c > 0$ where $a > 0$, the parabola opens upward. The inequality $ax^2 + bx + c > 0$ represents regions where the graph is above the x-axis, while $ax^2 + bx + c < 0$ represents regions where the graph is below the x-axis.

Worked Examples

Example 1: $x^2 - 5x + 6 > 0$

First, solve $x^2 - 5x + 6 = 0$: $(x-2)(x-3) = 0$ → x = 2, 3

Test point x = 0: 0 - 0 + 6 = 6 > 0 ✓

Test point x = 2.5: (2.5)² - 5(2.5) + 6 = 6.25 - 12.5 + 6 = -0.25 < 0 ✗

Test point x = 4: 16 - 20 + 6 = 2 > 0 ✓

Solution: $x < 2$ or $x > 3$

Example 2: $x^2 - 4x + 4 ≤ 0$

Solve $x^2 - 4x + 4 = 0$: $(x-2)^2 = 0$ → x = 2 (repeated)

Test point x = 0: 0 - 0 + 4 = 4 > 0

Test point x = 2: 4 - 8 + 4 = 0 ≤ 0 ✓

Solution: $x = 2$

Example 3: $x^2 + x + 1 < 0$

Discriminant: 1 - 4 = -3 < 0, no real roots

Test point x = 0: 0 + 0 + 1 = 1 > 0

Graph never goes below x-axis

Solution: No solution

Practice Problems

Problem 1

Solve: $x^2 - 7x + 12 > 0$

Roots: x = 3, 4

Test x = 0: 12 > 0 ✓

Test x = 3.5: 12.25 - 24.5 + 12 = -0.25 < 0 ✗

Solution: $x < 3$ or $x > 4$

Problem 2

Solve: $2x^2 - 4x + 2 ≤ 0$

Roots: x = 1 (repeated)

Test x = 0: 2 > 0

Test x = 1: 2 - 4 + 2 = 0 ≤ 0 ✓

Solution: $x = 1$

Problem 3

Solve: $x^2 + x + 1 ≥ 0$

Discriminant: -3 < 0, no real roots

Test x = 0: 1 > 0

Graph always above x-axis

Solution: All real numbers

Key Takeaways

  • When $a > 0$, the parabola opens upward with a minimum point
  • $ax^2 + bx + c > 0$ is satisfied outside the roots (when D > 0) or everywhere (when D ≤ 0)
  • $ax^2 + bx + c < 0$ is satisfied between the roots (when D > 0) or never (when D ≤ 0)
  • $ax^2 + bx + c ≥ 0$ includes the x-axis and regions above it
  • $ax^2 + bx + c ≤ 0$ includes the x-axis and regions below it
  • Use test points to determine which regions satisfy the inequality
  • When D < 0, the quadratic is always positive (a > 0), so < 0 has no solution
  • When D = 0, the inequality equals zero only at the repeated root
