Quadratic Formula

Mathematical Foundations

The quadratic formula is one of the most important tools in algebra. It provides the solutions to any quadratic equation of the form $ax^2 + bx + c = 0$. The formula was derived by completing the square and works for all real coefficients.

Derivation by Completing the Square:

Start with: $ax^2 + bx + c = 0$

Divide by a: $x^2 + \dfrac{b}{a}x + \dfrac{c}{a} = 0$

Move constant: $x^2 + \dfrac{b}{a}x = -\dfrac{c}{a}$

Complete square: $x^2 + \dfrac{b}{a}x + \left(\dfrac{b}{2a}\right)^2 = -\dfrac{c}{a} + \left(\dfrac{b}{2a}\right)^2$

Factor: $\left(x + \dfrac{b}{2a}\right)^2 = \dfrac{b^2 - 4ac}{4a^2}$

Take square root: $x + \dfrac{b}{2a} = \pm \dfrac{\sqrt{b^2 - 4ac}}{2a}$

Solution: $x = \dfrac{-b \pm \sqrt{b^2 - 4ac}}{2a}$

The Discriminant ($b^2 - 4ac$):

  • If $D > 0$: Two distinct real solutions
  • If $D = 0$: One repeated real solution
  • If $D < 0$: No real solutions (complex solutions)

Interactive Quadratic Solver

Use the quadratic formula to solve equations of the form $ax^2 + bx + c = 0$.

Equation: $x^2 + 2x - 3 = 0$

Discriminant: $4 - 4(1)(-3) = 16$

Solutions: $x = \dfrac{-2 \pm \sqrt{16}}{2} = \dfrac{-2 \pm 4}{2}$

$x = 1$ or $x = -3$

Worked Examples

Example 1: Two Real Solutions

Solve: $2x^2 - 7x + 3 = 0$

Using quadratic formula: $x = \dfrac{7 \pm \sqrt{49 - 24}}{4} = \dfrac{7 \pm \sqrt{25}}{4} = \dfrac{7 \pm 5}{4}$

Solutions: $x = \dfrac{12}{4} = 3$ or $x = \dfrac{2}{4} = 0.5$

Example 2: One Repeated Solution

Solve: $x^2 - 6x + 9 = 0$

Discriminant: $36 - 36 = 0$

Solution: $x = \dfrac{6 \pm \sqrt{0}}{2} = \dfrac{6}{2} = 3$ (repeated)

Example 3: Complex Solutions

Solve: $x^2 + 2x + 5 = 0$

Discriminant: $4 - 20 = -16$

Solutions: $x = \dfrac{-2 \pm \sqrt{-16}}{2} = \dfrac{-2 \pm 4i}{2} = -1 \pm 2i$

Practice Problems

Problem 1

Solve: $x^2 - 5x + 6 = 0$

$x = \dfrac{5 \pm \sqrt{25 - 24}}{2} = \dfrac{5 \pm \sqrt{1}}{2} = \dfrac{5 \pm 1}{2}$

$x = 3$ or $x = 2$

Problem 2

Solve: $2x^2 + 3x - 2 = 0$

$x = \dfrac{-3 \pm \sqrt{9 + 16}}{4} = \dfrac{-3 \pm \sqrt{25}}{4} = \dfrac{-3 \pm 5}{4}$

$x = \dfrac{2}{4} = 0.5$ or $x = \dfrac{-8}{4} = -2$

Problem 3

Solve: $x^2 - 4x + 4 = 0$

$x = \dfrac{4 \pm \sqrt{16 - 16}}{2} = \dfrac{4 \pm \sqrt{0}}{2} = \dfrac{4}{2} = 2$

Repeated solution: $x = 2$

Key Takeaways

  • The quadratic formula solves any equation of the form $ax^2 + bx + c = 0$
  • The discriminant $D = b^2 - 4ac$ determines the number and type of solutions
  • $D > 0$: Two distinct real solutions
  • $D = 0$: One repeated real solution
  • $D < 0$: Two complex solutions
  • The formula works for all real coefficients $a$, $b$, and $c$ (where $a ≠ 0$)
  • The quadratic formula is derived by completing the square
For the quadratic equation $ax^2 + bx + c = 0$:
When $b^2 - 4ac \geq 0$
$x = \dfrac{-b \pm \sqrt{b^2 - 4ac}}{2a}$
↓
For the quadratic equation $ax^2 + 2b'x + c = 0$:
When $(b')^2 - ac \geq 0$
$x = \dfrac{-b' \pm \sqrt{(b')^2 - ac}}{a}$
