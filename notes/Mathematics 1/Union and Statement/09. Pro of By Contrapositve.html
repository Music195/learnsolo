Proof by Contrapositive

Mathematical Foundations

Proof by contrapositive is a powerful indirect proof technique. Since a conditional statement and its contrapositive are logically equivalent, proving one automatically proves the other. This method is particularly useful when the contrapositive is easier to prove.

Direct Proof

$p \Rightarrow q$

Assume p is true, prove q follows

Straightforward but sometimes difficult

Example: Prove properties directly

Proof by Contrapositive

$\neg q \Rightarrow \neg p$

Assume ¬q is true, prove ¬p follows

Equivalent to direct proof

Often easier to apply

Why It Works

$p \Rightarrow q \equiv \neg q \Rightarrow \neg p$

Logically equivalent statements

Same truth value in all cases

Proving either proves both

When to Use

When contrapositive is easier to prove

When dealing with "if not A, then not B"

For irrationality proofs

For number theory theorems

To prove the statement $p \Rightarrow q$,
It is also sufficient to prove its contrapositive:
$\overline{q}$ $\Rightarrow$ $\overline{p}$

Steps for Proof by Contrapositive

Step 1: Identify the original statement: "If P, then Q" or P → Q
Step 2: Write the contrapositive: "If not Q, then not P" or ¬Q → ¬P
Step 3: Assume ¬Q is true (assume the conclusion of contrapositive is false)
Step 4: Prove that ¬P must follow (prove the hypothesis of contrapositive is false)
Step 5: Conclude that the original statement P → Q is true

Worked Examples

Example 1: Divisibility

Statement to prove: If n is divisible by 6, then n is divisible by 2

Contrapositive: If n is not divisible by 2, then n is not divisible by 6

Proof: Assume n is not divisible by 2. Since 6 = 2 × 3, any multiple of 6 must be a multiple of 2. Therefore, if n is not divisible by 2, n cannot be divisible by 6.

Conclusion: The original statement is true ✓

Example 2: Irrational Numbers

Statement to prove: If √2 is rational, then it can be written as a fraction a/b in lowest terms

Contrapositive: If √2 cannot be written as a fraction a/b in lowest terms, then √2 is irrational

Proof: This is the famous proof that √2 is irrational. Assume √2 = a/b in lowest terms. Then a² = 2b², so a² is even, so a is even. Let a = 2c, then 4c² = 2b², so 2c² = b², so b² is even, so b is even. But this contradicts a/b being in lowest terms.

Conclusion: √2 is irrational ✓

Example 3: Even and Odd

Statement to prove: If n² is even, then n is even

Contrapositive: If n is odd, then n² is odd

Proof: Assume n is odd. Then n = 2k + 1 for some integer k. Then n² = (2k + 1)² = 4k² + 4k + 1 = 2(2k² + 2k) + 1, which is odd.

Conclusion: The original statement is true ✓

Example 4: Prime Numbers

Statement to prove: If 2ⁿ - 1 is prime, then n is prime

Contrapositive: If n is composite, then 2ⁿ - 1 is composite

Proof: Assume n is composite, say n = ab where 1 < a,b < n. Then 2ⁿ - 1 = 2^(ab) - 1 = (2^a - 1)(2^(a(b-1)) + 2^(a(b-2)) + ... + 2^a + 1), so it has factors other than 1 and itself.

Conclusion: The original statement is true ✓

Practice Problems

Problem 1

Use proof by contrapositive to prove: If x² is irrational, then x is irrational

Contrapositive: If x is rational, then x² is rational

Proof: If x is rational, x = a/b where a,b are integers with no common factors. Then x² = a²/b², which is rational.

Conclusion: The original statement is true.

Problem 2

Prove by contrapositive: If n² is divisible by 3, then n is divisible by 3

Contrapositive: If n is not divisible by 3, then n² is not divisible by 3

Proof: If n is not divisible by 3, n = 3k + 1 or n = 3k + 2. In both cases, n² ≡ 1 mod 3, so not divisible by 3.

Conclusion: The original statement is true.

Problem 3

Use proof by contrapositive to show: If a and b are odd integers, then a + b is even

Contrapositive: If a + b is odd, then a and b are not both odd

Proof: If a + b is odd, then one of a or b must be even (since even + odd = odd).

Conclusion: The original statement is true.

Problem 4

Prove: If x ≤ 3, then x < 5

Contrapositive: If x ≥ 5, then x > 3

Proof: If x ≥ 5, then certainly x > 3.

Conclusion: The original statement is true.

Key Takeaways

  • Logical equivalence: p → q ≡ ¬q → ¬p
  • Proof method: Prove contrapositive instead of original statement
  • When to use: When contrapositive is easier to prove or more intuitive
  • Common applications: Divisibility proofs, irrationality proofs, number theory
  • Steps: 1) Write contrapositive, 2) Assume ¬q, 3) Prove ¬p
  • Advantage: Sometimes direct proof is difficult, but contrapositive is straightforward
  • Validity: Proves the original statement with 100% certainty
  • Indirect proof: One of the main indirect proof techniques in mathematics
