Converse, Contrapositive, and Inverse of Statements

Mathematical Foundations

Related statements are transformations of conditional statements (implications). Understanding these relationships is crucial for logical reasoning and mathematical proofs. Only the contrapositive is logically equivalent to the original statement.

Original Statement

$p \Rightarrow q$

"If p, then q"

If it rains, then the ground is wet

Foundation of all related statements

Converse

$q \Rightarrow p$

"If q, then p"

If the ground is wet, then it rained

Swaps hypothesis and conclusion

Contrapositive

$\neg q \Rightarrow \neg p$

"If not q, then not p"

If the ground is not wet, then it did not rain

Logically equivalent to original

Inverse

$\neg p \Rightarrow \neg q$

"If not p, then not q"

If it does not rain, then the ground is not wet

Negates both parts of original

Worked Examples

Example 1: Triangle Properties

Original: "If a triangle is equilateral, then it is equiangular"

Converse: "If a triangle is equiangular, then it is equilateral" ✓

Contrapositive: "If a triangle is not equiangular, then it is not equilateral" ✓

Inverse: "If a triangle is not equilateral, then it is not equiangular" ✗

Only converse and contrapositive are true for equilateral triangles

Example 2: Numbers

Original: "If x = 4, then x² = 16"

Converse: "If x² = 16, then x = 4" ✗ (x could be -4)

Contrapositive: "If x² ≠ 16, then x ≠ 4" ✓

Inverse: "If x ≠ 4, then x² ≠ 16" ✗ (x could be -4)

Only contrapositive is equivalent to original

Example 3: Real World

Original: "If you study hard, then you will pass the exam"

Converse: "If you pass the exam, then you studied hard" ✗

Contrapositive: "If you don't pass the exam, then you didn't study hard" ✗

Inverse: "If you don't study hard, then you won't pass the exam" ✗

None of the related statements are necessarily true

Example 4: Geometry

Original: "If two lines are parallel, then they never intersect"

Converse: "If two lines never intersect, then they are parallel" ✓

Contrapositive: "If two lines do intersect, then they are not parallel" ✓

Inverse: "If two lines are not parallel, then they do intersect" ✗

Converse and contrapositive are true, inverse is false

Practice Problems

Problem 1

Given: "If x > 5, then x > 3"

Which of the following is equivalent to the original statement?

Converse: "If x > 3, then x > 5" ✗

Contrapositive: "If x ≤ 3, then x ≤ 5" ✓

Inverse: "If x ≤ 5, then x ≤ 3" ✗

The contrapositive is equivalent

Problem 2

Write the converse of: "If a number is divisible by 6, then it is divisible by 2"

Converse: "If a number is divisible by 2, then it is divisible by 6"

This statement is false (counterexample: 2 is divisible by 2 but not by 6)

Problem 3

Which statement is equivalent to: "If it is raining, then the streets are wet"?

Contrapositive: "If the streets are not wet, then it is not raining"

This is the only equivalent statement

Problem 4

Determine which related statements are true for: "If a quadrilateral is a square, then it is a rectangle"

Converse: "If a quadrilateral is a rectangle, then it is a square" ✗

Contrapositive: "If a quadrilateral is not a rectangle, then it is not a square" ✓

Inverse: "If a quadrilateral is not a square, then it is not a rectangle" ✗

Only the contrapositive is true

Key Takeaways

  • Contrapositive only: Only the contrapositive (¬q → ¬p) is logically equivalent to the original (p → q)
  • Converse: q → p may or may not be true (swaps hypothesis and conclusion)
  • Inverse: ¬p → ¬q may or may not be true (negates both parts)
  • Proof technique: To prove p → q, prove the contrapositive ¬q → ¬p instead
  • Truth tables: Original and contrapositive have identical truth tables
  • Common mistake: Thinking converse or inverse are equivalent to original
  • Real world: Related statements often fail in practical scenarios
  • Mathematics: Many theorems have true converses (equilateral ↔ equiangular)
