Intersection and Union of Sets

Mathematical Foundations

Set operations combine sets in different ways. The intersection finds common elements, while the union combines all elements from both sets.

Intersection

A ∩ B

Elements in both A and B

Common to both sets

Can be empty if no common elements

Union

A ∪ B

Elements in A or B or both

All elements from both sets

Removes duplicates

Properties

Commutative: A ∩ B = B ∩ A

Associative: (A ∩ B) ∩ C = A ∩ (B ∩ C)

Distributive: A ∩ (B ∪ C) = (A ∩ B) ∪ (A ∩ C)

Identity: A ∩ U = A, A ∪ ∅ = A

Worked Examples

Example 1: Basic Operations

Let A = {1, 2, 3, 4} and B = {3, 4, 5, 6}

Intersection: A ∩ B = {3, 4} (elements in both sets)

Union: A ∪ B = {1, 2, 3, 4, 5, 6} (all elements from both sets)

Example 2: Disjoint Sets

Let C = {1, 2} and D = {3, 4}

Intersection: C ∩ D = ∅ (no common elements)

Union: C ∪ D = {1, 2, 3, 4} (all elements combined)

Example 3: One Set Subset of Another

Let E = {2, 4} and F = {1, 2, 3, 4, 5}

Intersection: E ∩ F = {2, 4} (same as E since E ⊂ F)

Union: E ∪ F = {1, 2, 3, 4, 5} (same as F since E ⊂ F)

Example 4: Same Sets

Let G = {a, b, c} and H = {a, b, c}

Intersection: G ∩ H = {a, b, c} (same as both sets)

Union: G ∪ H = {a, b, c} (same as both sets)

Key Takeaways

  • A ∩ B contains elements that are in both A and B
  • A ∪ B contains elements that are in A or B or both
  • Intersection and union are commutative: A ∩ B = B ∩ A, A ∪ B = B ∪ A
  • Intersection and union are associative: (A ∩ B) ∩ C = A ∩ (B ∩ C)
  • Distributive property: A ∩ (B ∪ C) = (A ∩ B) ∪ (A ∩ C)
  • Identity elements: A ∩ U = A, A ∪ ∅ = A where U is universal set
  • If A and B are disjoint (A ∩ B = ∅), then |A ∪ B| = |A| + |B|
  • Inclusion-exclusion: |A ∪ B| = |A| + |B| - |A ∩ B|
  • Intersection can be empty, union is never empty if at least one set is non-empty
The set of all elements that belong to both sets $A$ and $B$ is called the intersection of $A$ and $B$, written as $A \cap B$ .
The set of all elements that belong to at least one of $A$ or $B$ is called the union of $A$ and $B$, written as $A \cup B$ .
