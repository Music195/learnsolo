Sets and Elements

Mathematical Foundations

A set is a collection of distinct objects, called elements or members. Sets can be defined in two main ways: by listing their elements explicitly, or by describing properties that characterize the elements.

Element Notation

$x$ is an element of set $A$: $x \in A$

$y$ is not an element of set $A$: $y \notin A$

The symbol $\in$ means "belongs to" or "is a member of"

The symbol $\notin$ means "does not belong to"

Set Definition Methods

Listing method: $\{1, 2, 3, 4\}$

Set-builder notation: $\{x \mid x$ is a positive integer less than $5\}$

Both define the same set of elements

Worked Examples

Example 1: Listing Method

The set of vowels in the English alphabet:

$V = \{a, e, i, o, u\}$

Here, $a \in V$, $b \notin V$, $e \in V$

Example 2: Set-Builder Notation

The set of even positive integers less than 10:

$E = \{x \mid x$ is an even positive integer and $x < 10\}$

This is equivalent to $E = \{2, 4, 6, 8\}$

Check: $2 \in E$, $3 \notin E$, $6 \in E$

Example 3: Positive Divisors

The set of all positive divisors of 15:

$B = \{1, 3, 5, 15\}$

Or using set-builder notation:

$B = \{x \mid x$ is a positive divisor of $15\}$

Verification: $1 \in B$, $2 \notin B$, $3 \in B$, $5 \in B$, $15 \in B$

Key Takeaways

  • A set is a collection of distinct objects called elements
  • $x \in A$ means "x is an element of set A"
  • $y \notin A$ means "y is not an element of set A"
  • Sets can be defined by listing elements: {1, 2, 3}
  • Sets can be defined by properties: {x | x satisfies property P}
  • Both methods define the same mathematical object
  • Elements in a set are unique - no duplicates
  • The order of elements in a set doesn't matter
  • Sets can contain numbers, letters, objects, or other sets
