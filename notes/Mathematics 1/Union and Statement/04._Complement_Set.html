Complement of a Set

Mathematical Foundations

The complement of a set contains all elements in the universal set that are not in the original set. It's a fundamental concept in set theory with important properties.

Complement Definition

For set A ⊂ U:

Ā = {x ∈ U | x ∉ A}

Elements in U but not in A

Also written as A' or Aᶜ

Key Properties

A ∩ Ā = ∅ (disjoint)

A ∪ Ā = U (covers universal set)

(Ā)̄ = A (double complement)

∅̄ = U, Ū = ∅

De Morgan's Laws

(A ∪ B)̄ = Ā ∩ B̄

(A ∩ B)̄ = Ā ∪ B̄

Complement of union/intersection

Fundamental set identities

Worked Examples

Example 1: Basic Complement

Let U = {1, 2, 3, 4, 5} and A = {2, 4}

Complement: Ā = {1, 3, 5} (elements in U but not in A)

Verification: A ∪ Ā = {1, 2, 3, 4, 5} = U ✓

Example 2: Empty Set Complement

Let U = {a, b, c} and A = ∅

Complement: Ā = {a, b, c} = U

The complement of the empty set is the universal set

Example 3: Universal Set Complement

Let U = {x, y, z} and A = {x, y, z} = U

Complement: Ā = ∅

The complement of the universal set is the empty set

Example 4: Double Complement

Let U = {1, 2, 3, 4} and A = {1, 3}

Complement: Ā = {2, 4}

Double complement: (Ā)̄ = {1, 3} = A

The complement of the complement is the original set

Key Takeaways

  • The complement Ā contains all elements in U that are not in A
  • A ∩ Ā = ∅ (complement is disjoint from original set)
  • A ∪ Ā = U (complement covers the universal set with original)
  • (Ā)̄ = A (double complement returns original set)
  • ∅̄ = U and Ū = ∅ (special cases for empty and universal sets)
  • Complement operation is involutive (applying twice returns original)
  • De Morgan's laws relate complement to union and intersection
  • Complement depends on the universal set being considered
  • The complement of a subset is also a subset of the universal set
For a subset $A$ of the universal set $U$, the set of all elements in $U$ that do not belong to $A$ is called the complement of $A$ with respect to $U$, and is written as $\overline{A}$ .
