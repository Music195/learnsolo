De Morgan's Laws
De Morgan's Laws

Mathematical Foundations

De Morgan's laws describe how complement operations interact with union and intersection. These fundamental laws allow us to rewrite complex set expressions and are essential for logical reasoning in set theory.

First Law: Complement of Union

$\overline{A \cup B} = \overline{A} \cap \overline{B}$

The complement of the union equals the intersection of the complements

Elements not in A ∪ B are elements not in A and not in B

Second Law: Complement of Intersection

$\overline{A \cap B} = \overline{A} \cup \overline{B}$

The complement of the intersection equals the union of the complements

Elements not in A ∩ B are elements not in A or not in B

Logical Equivalence

¬(P ∧ Q) ≡ ¬P ∨ ¬Q

¬(P ∨ Q) ≡ ¬P ∧ ¬Q

Same laws apply to logic with NOT, AND, OR

Worked Examples

Example 1: First Law - Complement of Union

Let U = {1, 2, 3, 4, 5}, A = {1, 2, 3}, B = {3, 4, 5}

A ∪ B = {1, 2, 3, 4, 5}

$\overline{A \cup B} = \overline{\{1, 2, 3, 4, 5\}} = \emptyset$

$\overline{A} = \{4, 5\}$, $\overline{B} = \{1, 2\}$

$\overline{A} \cap \overline{B} = \{4, 5\} \cap \{1, 2\} = \emptyset$

Both sides equal ∅ ✓

Example 2: Second Law - Complement of Intersection

Let U = {1, 2, 3, 4, 5}, A = {1, 2, 3}, B = {2, 3, 4}

A ∩ B = {2, 3}

$\overline{A \cap B} = \overline{\{2, 3\}} = \{1, 4, 5\}$

$\overline{A} = \{4, 5\}$, $\overline{B} = \{1, 5\}$

$\overline{A} \cup \overline{B} = \{4, 5\} \cup \{1, 5\} = \{1, 4, 5\}$

Both sides equal {1, 4, 5} ✓

Example 3: Logical Application

Negate: "x > 5 and x < 10"

Using De Morgan: ¬(P ∧ Q) ≡ ¬P ∨ ¬Q

¬(x > 5 ∧ x < 10) ≡ ¬(x > 5) ∨ ¬(x < 10)

≡ x ≤ 5 ∨ x ≥ 10

The solution is x ≤ 5 or x ≥ 10

Practice Problems

Problem 1

Given U = {1, 2, 3, 4, 5, 6}, A = {1, 2, 3}, B = {3, 4, 5}:

Verify: $\overline{A \cup B} = \overline{A} \cap \overline{B}$

A ∪ B = {1, 2, 3, 4, 5}

$\overline{A \cup B} = \{6\}$

$\overline{A} = \{4, 5, 6\}$, $\overline{B} = \{1, 2, 6\}$

$\overline{A} \cap \overline{B} = \{6\}$

Both sides equal {6} ✓

Problem 2

Given U = {a, b, c, d, e}, P = {a, b}, Q = {b, c}:

Verify: $\overline{P \cap Q} = \overline{P} \cup \overline{Q}$

P ∩ Q = {b}

$\overline{P \cap Q} = \{a, c, d, e\}$

$\overline{P} = \{c, d, e\}$, $\overline{Q} = \{a, d, e\}$

$\overline{P} \cup \overline{Q} = \{a, c, d, e\}$

Both sides equal {a, c, d, e} ✓

Problem 3

Use De Morgan's law to negate: "x ≥ 2 and x ≤ 8"

¬(x ≥ 2 ∧ x ≤ 8) ≡ ¬(x ≥ 2) ∨ ¬(x ≤ 8)

≡ x < 2 ∨ x > 8

The solution is x < 2 or x > 8

Key Takeaways

  • $\overline{A \cup B} = \overline{A} \cap \overline{B}$: Complement of union equals intersection of complements
  • $\overline{A \cap B} = \overline{A} \cup \overline{B}$: Complement of intersection equals union of complements
  • These laws apply to both set theory and logic (with ¬, ∧, ∨)
  • Essential for simplifying complex set expressions and logical statements
  • Foundation of digital logic, computer science, and mathematical reasoning
  • Can be extended to more than two sets: $\overline{\bigcup A_i} = \bigcap \overline{A_i}$
  • Help convert between union/intersection and their complements
  • Critical for solving inequalities and set theory problems
