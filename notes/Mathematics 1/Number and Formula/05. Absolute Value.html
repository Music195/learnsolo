Absolute Value

Mathematical Foundation

The absolute value of a real number represents its distance from zero on the number line, regardless of direction. This concept is fundamental in mathematics, providing a way to measure magnitude without considering sign. Absolute values appear in distance formulas, inequalities, complex numbers, and many real-world applications where only the magnitude matters.

Definition

|x| = x     if x ≥ 0
For non-negative numbers, absolute value equals the number itself.
|x| = -x    if x < 0
For negative numbers, absolute value equals the additive inverse (positive version).
|x| ≥ 0
Absolute value is always non-negative.

Geometric Interpretation

The absolute value |x| represents the distance from x to 0 on the number line:

  • If x ≥ 0, distance is x units to the right of zero
  • If x < 0, distance is |x| units to the left of zero (same as -x units to the right)
  • The absolute value function creates a "V" shape when graphed
  • It's like folding the negative x-axis up to meet the positive x-axis

Number Line Representation

Consider these examples:

  • |5| = 5 (distance from 0 to 5)
  • |-3| = 3 (distance from 0 to -3)
  • |0| = 0 (distance from 0 to 0)
  • |-7| = 7 (distance from 0 to -7)

The absolute value measures magnitude, not direction.

Properties of Absolute Value

|x| ≥ 0
Absolute value is always non-negative.
|-x| = |x|
Absolute value of negative is same as absolute value of positive.
|xy| = |x| × |y|
Absolute value of product equals product of absolute values.
|x/y| = |x| / |y|    (y ≠ 0)
Absolute value of quotient equals quotient of absolute values.
|x + y| ≤ |x| + |y|    (Triangle Inequality)
Absolute value of sum is less than or equal to sum of absolute values.
||x| - |y|| ≤ |x - y| ≤ |x| + |y|
Reverse triangle inequality relating differences and sums.

Solving Equations with Absolute Value

Equations of the form |ax + b| = c have two cases:

  • ax + b = c    or    ax + b = -c
  • Solve both equations and check solutions in original equation
  • If c < 0, no solution (absolute value can't be negative)

Practice Problems

Problem 1: Evaluate |-12| + |8| - |3|
Solution:
|-12| + |8| - |3| = 12 + 8 - 3 = 17
Problem 2: Solve |2x - 1| = 7
Solution:
2x - 1 = 7    or    2x - 1 = -7
2x = 8    or    2x = -6
x = 4    or    x = -3
Check: |2(4) - 1| = |8 - 1| = 7 ✓
|-2(-3) - 1| = |-6 - 1| = 7 ✓
Problem 3: Simplify |x² - 4| when x = -3
Solution:
x² - 4 = (-3)² - 4 = 9 - 4 = 5
|5| = 5
Problem 4: Verify the triangle inequality: |5 + (-8)| ≤ |5| + |-8|
Solution:
|-3| ≤ 5 + 8
3 ≤ 13 ✓

Key Takeaways

  • Definition: |x| = x if x ≥ 0; |x| = -x if x < 0
  • Distance: |x| represents distance from x to 0 on number line
  • Non-negative: |x| ≥ 0 for all real numbers x
  • Symmetry: |-x| = |x| - absolute value of negative equals absolute value of positive
  • Multiplication: |xy| = |x| × |y| - absolute value of product
  • Division: |x/y| = |x|/|y| (y ≠ 0) - absolute value of quotient
  • Triangle Inequality: |x + y| ≤ |x| + |y| - fundamental property
  • Equations: |ax + b| = c has two solutions: ax + b = c or ax + b = -c
  • No Solution: |expression| = negative number has no solution
  • Applications: Distance, magnitude, inequalities, complex numbers
