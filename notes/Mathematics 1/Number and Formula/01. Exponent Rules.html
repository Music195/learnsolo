Exponent Rules

Mathematical Foundation

Exponent rules, also known as laws of exponents, are fundamental principles that govern how to manipulate expressions involving powers. These rules allow us to simplify complex exponential expressions and perform algebraic operations efficiently. Understanding these rules is essential for working with polynomials, rational expressions, and many advanced mathematical concepts.

Basic Exponent Rules

xᵃ × xᵇ = x^(a+b)
When multiplying powers with the same base, add the exponents.
(xᵃ)^ᵇ = x^(a×b)
When raising a power to another power, multiply the exponents.
(x×y)^ᵇ = xᵇ × yᵇ
When raising a product to a power, distribute the exponent to each factor.
xᵃ / xᵇ = x^(a-b)
When dividing powers with the same base, subtract the exponents.
x⁰ = 1
Any nonzero number raised to the power of zero equals 1.
x^(-a) = 1/xᵃ
A negative exponent indicates the reciprocal of the positive exponent.
x^(1/a) = ᵃ√x
A fractional exponent indicates a root (rational exponents).

Worked Example 1: Multiplication Rule

Simplify: 2³ × 2⁵

Solution:

2³ × 2⁵ = 2^(3+5) = 2⁸ = 256

Worked Example 2: Power Rule

Simplify: (3²)⁴

Solution:

(3²)⁴ = 3^(2×4) = 3⁸ = 6561

Worked Example 3: Division Rule

Simplify: x⁷ / x³

Solution:

x⁷ / x³ = x^(7-3) = x⁴

Worked Example 4: Negative Exponents

Simplify: 2^(-3)

Solution:

2^(-3) = 1/2³ = 1/8 = 0.125

Key Takeaways

  • Multiplication: xᵃ × xᵇ = x^(a+b) - add exponents
  • Power Rule: (xᵃ)^ᵇ = x^(a×b) - multiply exponents
  • Product Rule: (x×y)^ᵇ = xᵇ × yᵇ - distribute exponent
  • Division: xᵃ / xᵇ = x^(a-b) - subtract exponents
  • Zero Power: x⁰ = 1 (for x ≠ 0)
  • Negative: x^(-a) = 1/xᵃ - reciprocal
  • Fractional: x^(1/n) = ᵃ√x - nth root
  • Base ≠ 0: Most rules require nonzero base
  • Order: Apply rules in correct sequence
  • Applications: Simplifying expressions, solving equations
