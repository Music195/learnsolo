Equations and Inequalities with Absolute Value

Mathematical Foundations

The absolute value of a number represents its distance from zero on the number line. For any real number $x$, the absolute value is denoted by $|x|$ and is defined as:

$$|x| = \begin{cases} x & \text{if } x \geq 0 \\ -x & \text{if } x < 0 \end{cases}$$
Key Concept: Absolute value is always non-negative. The equation $|x| = c$ has solutions when $c \geq 0$, and the inequalities $|x| < c$ and $|x| > c$ behave differently based on the value of $c$.

Worked Examples

Example 1: Solving |x| = 3

The absolute value equals 3 when x is 3 units from zero.

Solutions: $x = 3$ or $x = -3$

Example 2: Solving |x| < 4

The absolute value is less than 4 when x is within 4 units of zero.

Solution: $-4 < x < 4$

Example 3: Solving |x| > 2

The absolute value is greater than 2 when x is more than 2 units from zero.

Solution: $x < -2$ or $x > 2$

Example 4: Solving |2x - 1| = 3

First solve the equation: $2x - 1 = 3$ or $2x - 1 = -3$

$2x = 4$ → $x = 2$ or $2x = -2$ → $x = -1$

Solutions: $x = -1$ or $x = 2$

Practice Problems

Solutions: $x = 7$ or $x = -7$

Solution: $-6 < x < 6$

Problem 3

Solve: $|x| > 4$

Problem 4

Solve: $|3x + 2| = 5$

$3x + 2 = 5$ → $3x = 3$ → $x = 1$

$3x + 2 = -5$ → $3x = -7$ → $x = -\dfrac{7}{3}$

Solutions: $x = -\dfrac{7}{3}$ or $x = 1$

Key Takeaways

  • $|x| = c$ has two solutions when $c > 0$: $x = c$ and $x = -c$
  • $|x| = c$ has one solution when $c = 0$: $x = 0$
  • $|x| = c$ has no solutions when $c < 0$
  • $|x| < c$ means $-c < x < c$ (when $c > 0$)
  • $|x| > c$ means $x < -c$ or $x > c$ (when $c > 0$)
  • Absolute value equations and inequalities can be solved by considering the positive and negative cases separately
When $c$ is a positive constant:
Equation $|x| = c$ has the solution: $x = \pm c$
Inequality $|x| < c$ has the solution: $-c < x < c$
Inequality $|x| > c$ has the solution: $x < -c$, $x > c$
