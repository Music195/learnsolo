Expansion Formulas

Mathematical Foundation

Expansion formulas are algebraic identities that allow us to expand products of binomials and polynomials into sums. These formulas are essential tools in algebra, providing shortcuts for multiplying expressions without having to use the distributive law repeatedly. They are widely used in factoring, solving equations, and simplifying complex expressions. Mastering these formulas enables efficient manipulation of algebraic expressions.

Square Formulas

(a + b)² = a² + 2ab + b²
Square of a sum: expand by multiplying (a+b) by itself.
(a - b)² = a² - 2ab + b²
Square of a difference: expand by multiplying (a-b) by itself.

Difference of Squares

(a + b)(a - b) = a² - b²
Product of sum and difference equals difference of squares.

General Binomial Products

(x + a)(x + b) = x² + (a + b)x + ab
Product of two linear terms with common variable.
(ax + b)(cx + d) = acx² + (ad + bc)x + bd
General product of two binomials with coefficients.

Cube Formulas

(a + b)³ = a³ + 3a²b + 3ab² + b³
Cube of a sum: careful with coefficient pattern 1-3-3-1.
(a - b)³ = a³ - 3a²b + 3ab² - b³
Cube of a difference: alternating signs with same coefficients.

Sum and Difference of Cubes

(a + b)(a² - ab + b²) = a³ + b³
Sum of cubes: a³ + b³ factors as (a + b)(a² - ab + b²).
(a - b)(a² + ab + b²) = a³ - b³
Difference of cubes: a³ - b³ factors as (a - b)(a² + ab + b²).

Derivation: Square of Sum

Using the distributive law:

(a + b)² = (a + b)(a + b)

= a(a + b) + b(a + b)

= a² + ab + ba + b²

= a² + 2ab + b²

Note: ab = ba (commutative property)

Geometric Interpretation

The square formulas can be visualized geometrically:

  • (a + b)² represents the area of a large square with side (a + b)
  • This area equals the sum of: a² + b² + 2ab (four corner rectangles)
  • The difference of squares (a + b)(a - b) = a² - b² represents the area relationship
  • Cube formulas extend these geometric relationships to three dimensions

Worked Example 1: Square of Sum

Expand: (x + 3)²

Solution:

(x + 3)² = x² + 2(x)(3) + 3² = x² + 6x + 9

Worked Example 2: Difference of Squares

Expand: (2x + 1)(2x - 1)

Solution:

(2x + 1)(2x - 1) = (2x)² - 1² = 4x² - 1

Worked Example 3: Cube of Sum

Expand: (a + 2)³

Solution:

(a + 2)³ = a³ + 3(a²)(2) + 3(a)(2²) + 2³ = a³ + 6a² + 12a + 8

Practice Problems

Problem 1: Expand (x + 4)²
Solution:
(x + 4)² = x² + 2(x)(4) + 4² = x² + 8x + 16
Problem 2: Expand (3x - 2)²
Solution:
(3x - 2)² = (3x)² - 2(3x)(2) + 2² = 9x² - 12x + 4
Problem 3: Expand (x + 2)(x - 2)
Solution:
(x + 2)(x - 2) = x² - 2² = x² - 4
Problem 4: Expand (x + 3)(x + 5)
Solution:
(x + 3)(x + 5) = x² + (3+5)x + (3×5) = x² + 8x + 15

Key Takeaways

  • Squares: (a+b)² = a² + 2ab + b²; (a-b)² = a² - 2ab + b²
  • Difference: (a+b)(a-b) = a² - b² - difference of squares
  • Binomial: (x+a)(x+b) = x² + (a+b)x + ab - sum and product
  • Cubes: (a+b)³ = a³ + 3a²b + 3ab² + b³ - coefficient pattern 1-3-3-1
  • Sum of Cubes: a³ + b³ = (a+b)(a² - ab + b²)
  • Difference of Cubes: a³ - b³ = (a-b)(a² + ab + b²)
  • Signs: Be careful with signs in cube formulas
  • Applications: Factoring, expanding, equation solving
  • Patterns: Recognize coefficient patterns in expansions
  • Verification: Always check expansions by multiplying back
