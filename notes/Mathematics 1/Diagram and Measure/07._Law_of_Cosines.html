Law of Cosines

Mathematical Foundation

The Law of Cosines is a fundamental relationship that connects the lengths of the sides of a triangle with the cosine of one of its angles. It generalizes the Pythagorean theorem to all triangles (not just right-angled ones) and is particularly useful when you know two sides and the included angle, or when you know all three sides and want to find an angle.

Law of Cosines

$a^2 = b^2 + c^2 - 2bc \cos A$
$b^2 = c^2 + a^2 - 2ca \cos B$
$c^2 = a^2 + b^2 - 2ab \cos C$

Where:

  • a, b, c are the lengths of the sides
  • A, B, C are the measures of the opposite angles
  • cos is the cosine function

Connection to Pythagorean Theorem

When angle C = 90°, cos C = 0, so the Law of Cosines becomes:

$c^2 = a^2 + b^2$

This is exactly the Pythagorean theorem! The Law of Cosines is a generalization that works for all triangles.

Derivation: Coordinate Geometry Method

Place △ABC with vertex C at the origin (0,0), and side c along the x-axis:

  • Point A: (c, 0)
  • Point B: (b cos A, b sin A)
  • Point C: (0, 0)

Distance formula for side a (between B and C):

$a^2 = (b \cos A - 0)^2 + (b \sin A - 0)^2$

$a^2 = b^2 \cos^2 A + b^2 \sin^2 A$

$a^2 = b^2 (\cos^2 A + \sin^2 A) = b^2 (1)$

$a^2 = b^2$

This is incomplete! We need to consider the angle at C. Let me try again...

Correct approach: Place △ABC with vertex C at (0,0), side b along x-axis, side c at angle A:

  • Point A: (b, 0)
  • Point B: (c cos A, c sin A)
  • Point C: (0, 0)

Distance between A and B:

$a^2 = (c \cos A - b)^2 + (c \sin A - 0)^2$

$a^2 = c^2 \cos^2 A - 2bc \cos A + b^2 + c^2 \sin^2 A$

$a^2 = c^2 (\cos^2 A + \sin^2 A) + b^2 - 2bc \cos A$

$a^2 = c^2 (1) + b^2 - 2bc \cos A$

$a^2 = b^2 + c^2 - 2bc \cos A$ ✓

Applications of the Law of Cosines

  • SAS Triangles: Two sides and included angle → find third side
  • SSS Triangles: All three sides → find any angle
  • Navigation: Distance and bearing calculations
  • Physics: Vector analysis and force decomposition
  • Engineering: Structural analysis and surveying
  • Computer Graphics: 3D modeling and animation

Worked Example 1: SAS Triangle

In △ABC, sides b = 8 cm, c = 6 cm, and included angle A = 60°. Find side a.

Solution:

Using Law of Cosines: a² = b² + c² - 2bc cos A

a² = 8² + 6² - 2×8×6×cos 60°

a² = 64 + 36 - 96×0.5

a² = 100 - 48 = 52

a = √52 ≈ 7.21 cm

Worked Example 2: SSS Triangle

In △ABC, sides a = 7 cm, b = 5 cm, c = 9 cm. Find angle A.

Solution:

Using Law of Cosines: cos A = (b² + c² - a²)/(2bc)

cos A = (25 + 81 - 49)/(2×5×9) = (57 - 49)/90 = 8/90 = 4/45

A = cos⁻¹(4/45) ≈ 83.7°

Worked Example 3: Right Triangle Check

Verify if △ABC with sides 3, 4, 5 is right-angled.

Solution:

Check angle opposite side 5: cos C = (3² + 4² - 5²)/(2×3×4) = (9 + 16 - 25)/24 = 0/24 = 0

cos C = 0, so C = 90° ✓

This confirms the Pythagorean triple 3-4-5.

Practice Problems

Problem 1: In △ABC, sides b = 10 cm, c = 8 cm, angle A = 45°. Find side a.
Solution:
a² = 10² + 8² - 2×10×8×cos 45°
a² = 100 + 64 - 160×0.7071 ≈ 164 - 113.14 ≈ 50.86
a ≈ 7.13 cm
Problem 2: In △ABC, sides a = 12, b = 9, c = 15. Find angle C.
Solution:
cos C = (a² + b² - c²)/(2ab) = (144 + 81 - 225)/(2×12×9)
cos C = (0)/216 = 0
C = 90°
Problem 3: Find the length of the third side of a triangle with sides 7 and 11 and included angle 120°.
Solution:
a² = 7² + 11² - 2×7×11×cos 120°
a² = 49 + 121 - 154×(-0.5) = 170 + 77 = 247
a ≈ 15.72

Key Takeaways

  • Formula: a² = b² + c² - 2bc cos A (and cyclic permutations)
  • Pythagorean: When angle = 90°, reduces to a² = b² + c²
  • SAS Case: Two sides + included angle → third side
  • SSS Case: Three sides → any angle using cos A = (b²+c²-a²)/(2bc)
  • Angle Range: cos A between -1 and 1, so angles 0°-180°
  • Triangle Check: All sides positive, triangle inequalities
  • Acute/Obtuse: cos A > 0 (acute), cos A < 0 (obtuse)
  • Coordinate Proof: Distance formula with angle placement
  • Applications: Navigation, physics, engineering, computer graphics
  • Units: Angles in degrees, sides in consistent units
