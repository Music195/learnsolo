Definition of Trigonometric Ratios (Coordinate Plane)

Mathematical Foundation

The coordinate plane approach extends trigonometric ratios beyond right triangles to any angle. By placing the angle in standard position on a coordinate plane and using the unit circle (circle with radius 1), we can define trigonometric functions for all real numbers. This approach is fundamental to modern trigonometry and calculus.

Unit Circle Definition

Consider an angle θ in standard position with its terminal side intersecting the unit circle at point P(x, y):

Trigonometric Ratios on Unit Circle:
\( \sin\theta = y \)

\( \cos\theta = x \)

\( \tan\theta = \dfrac{y}{x} \) (when x ≠ 0)
Quadrant I (0° to 90°)
sin θ > 0, cos θ > 0, tan θ > 0
All trigonometric functions positive
Quadrant II (90° to 180°)
sin θ > 0, cos θ < 0, tan θ < 0
Only sine positive
Quadrant III (180° to 270°)
sin θ < 0, cos θ < 0, tan θ > 0
Only tangent positive
Quadrant IV (270° to 360°)
sin θ < 0, cos θ > 0, tan θ < 0
Only cosine positive

Angle Position and Reference Angles

The position of an angle determines the signs of trigonometric functions. Reference angles help find exact values for angles greater than 90°.

  • Reference Angle: The acute angle formed between the terminal side and the x-axis
  • Quadrant I: Reference angle = θ
  • Quadrant II: Reference angle = 180° - θ
  • Quadrant III: Reference angle = θ - 180°
  • Quadrant IV: Reference angle = 360° - θ

Reference Angle Examples

150°:
Reference = 180° - 150° = 30°
sin 150° = sin 30° = 1/2
210°:
Reference = 210° - 180° = 30°
sin 210° = -sin 30° = -1/2
330°:
Reference = 360° - 330° = 30°
cos 330° = cos 30° = √3/2

Worked Example 1: Unit Circle

Find sin 120°, cos 120°, tan 120° using the unit circle.

Solution:

120° is in Quadrant II. Reference angle = 180° - 120° = 60°

sin 120° = sin 60° = √3/2

cos 120° = -cos 60° = -1/2

tan 120° = -tan 60° = -√3

Worked Example 2: Coordinate Point

Point P(√2/2, -√2/2) lies on the unit circle. Find the angle θ.

Solution:

sin θ = y = -√2/2

cos θ = x = √2/2

θ = 315° or -45° (since sin is negative and cos is positive)

Practice Problems

Problem 1: Find sin 135°, cos 135°, tan 135°.
Solution:
135° is in Quadrant II. Reference angle = 45°
sin 135° = sin 45° = √2/2
cos 135° = -cos 45° = -√2/2
tan 135° = -tan 45° = -1
Problem 2: Point P(-1/2, √3/2) is on the unit circle. Find θ.
Solution:
cos θ = -1/2, sin θ = √3/2
θ = 120° (since cos is negative, sin is positive)
Problem 3: Find the quadrant and reference angle for 240°.
Solution:
Quadrant III, reference angle = 240° - 180° = 60°
Problem 4: If sin θ = -0.6 and cos θ = 0.8, find θ.
Solution:
sin negative, cos positive → Quadrant IV
θ = 360° - sin⁻¹(0.6) ≈ 360° - 36.87° = 323.13°
Problem 5: Find tan 300° using reference angle.
Solution:
Quadrant IV, reference angle = 360° - 300° = 60°
tan 300° = -tan 60° = -√3

Key Takeaways

  • Unit Circle: Circle with radius 1 centered at origin
  • Standard Position: Initial side on positive x-axis, vertex at origin
  • Coordinate Definition: sin θ = y-coordinate, cos θ = x-coordinate
  • Quadrant Signs: Determine signs of trigonometric functions
  • Reference Angle: Acute angle to x-axis, used for exact values
  • All Angles: Trigonometric functions defined for all real numbers
  • Periodicity: Functions repeat every 360° (2π radians)
  • Pythagorean Identity: sin²θ + cos²θ = 1 always holds
  • Applications: Essential for calculus, physics, engineering
  • Visualization: Unit circle provides geometric interpretation
